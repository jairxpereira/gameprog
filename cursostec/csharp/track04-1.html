04-1. Estruturas de decisão: if\switch
1. Instrução if


Veja o programa exemplo que ilustra como usar a instrução if:



 // Projeto: TestDecisao01
 // Testando a estrutura de decisão com o if
 using System;

 namespace TestDecisao
 {
    class Program
    {
        static void Main(string[] args)
        {
            int nvidas = 0;
            int nEnergia = 10;
            int npontos = 15000;
            bool lKitMedicoEncontrado = true;

            // Vamos configurar a janela de console
            Console.BackgroundColor = ConsoleColor.White;
            Console.ForegroundColor = ConsoleColor.Black;
            Console.Title = "Estruturas de decisão 01";
            Console.Clear();

            if (lKitMedicoEncontrado) nEnergia = nEnergia + 30;  
            Console.WriteLine("Nível de energia: " + nEnergia );
            
            if (nEnergia <= 40) Console.WriteLine("Sua energia está baixa!");

            if (npontos > 10000)
            {
                nvidas = nvidas + 1;
                Console.WriteLine("Vc ganhou uma vida extra!");
            } // fim do if

            Console.Read();

        } // main() fim
    } // fim da classe
 } // fim do nameSpace

	



2.1 Instrução switch


Veja o programa exemplo que ilustra como usar a instrução switch:

( O tema de fundo desse programa são aqueles jogos de videogame que mostram
sequências finais diferentes conforme a pontuação atingida ou certos objetivos
alcançados pelo jogador durante o jogo. )
	
	

 // Projeto: TestDecisao02
 // Testando a estrutura de decisão com o switch
 using System;

 namespace TestDecisao
 {
    class Program
    {
        static void Main(string[] args)
        {

            int nfinal;
            // Vamos configurar a janela de console
            Console.BackgroundColor = ConsoleColor.White;
            Console.ForegroundColor = ConsoleColor.Black;
            Console.Title = " Estrutura de decisão 02: switch";
            Console.Clear();


        for (nfinal = 0; nfinal < 5; nfinal++ )
        {

	switch (nfinal)
	{
	case 1:
		Console.WriteLine(nfinal + "::Mostrar video Final ruim");
		break;

	case 2:
		Console.WriteLine(nfinal + "::Mostrar video Final médio");
		break;

	case 3:
		Console.WriteLine(nfinal + "::Mostrar video Final ótimo");
		break;

	default:
		Console.WriteLine(nfinal + "::congratulations!");
		break;
                } // fim do switch
            } // fim do for

            Console.Read();

        } // main() fim
    } // fim da classe
 } // fim do nameSpace



2.2 Instrução break
Em C# é obrigatório o uso do break quando o case tiver sido acompanhado de 
código executável. Esclarecendo com um exemplo, no caso abaixo você não 
pode omitir o break para executar também o código do case 3:

case 2:
Console.WriteLine(nfinal + "::Mostrar video Final médio");
break; // Este break é obrigatório. 

Porém, tal como em C++ você pode fazer o seguinte:

case 2:
case 3:
Console.WriteLine(nfinal + "::Mostrar video Final médio");
break; // Este break é obrigatório.
