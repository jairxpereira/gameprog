05-3. Passagem de parâmetros
1. Visão geral
A passagem de parâmetros está ligado com a necessidade de troca 
de informações entre métodos. Geralmente há duas formas de troca
de dados: passagem por valor e passagem por referência.

Na passagem por valor, uma cópia do dado é feito na memória e depois
passado para a função sendo chamada. A variável original que foi
passada não sofre modificação do ponto de vista do método que
chamou a função.

Na passagem por referência, a variável tem o seu endereço passado.
Para grandes volumes de dados, isso geralmente economiza tempo e 
memória que são perdidos na passagem por  valor.  Nesse  caso  a 
variável  tem  o  conteúdo modificado pela função chamada e essa 
mudança também é vista no método que chamou a função. 

1. Passagem por valor

String caixa = "vazio";
String mochila = "faca"; 
permuta_string(ref caixa, ref mochila);

Na  passagem  por  valor não existe novidade. As variáveis são
definidas e são passadas para outra função sem nenhuma sintaxe
especial.

No código acima, a função permuta_string( )  não emite o resultado
da troca de strings  para o método que a chamou pois  não  propaga
a modificação de conteúdo das variáveis para o método que a chamou.

2.1 Passagem por referência - ref

String caixa = "vazio";
String mochila = "faca"; 
permuta_string(ref caixa, ref mochila);

Na passagem de parâmetros por referência, a palavra ref deve 
acompanhar  a  variável  na chamada da função e na definição
da  função  que  vai  aceitar  parâmetros  por  referências.

2.2 Passagem por referência - out


int item = 0;
String item_mensagem;
pegar_mensagem(item, out item_descricao);
mostrar_string(item_descricao);

(...)

Nesta forma de passagem por referência, a palavra out deve 
acompanhar  a  variável  na chamada da função e na definição
da  função  que  vai  aceitar  parâmetros  por  referências.

Não é necessário inicializar a variável que vai ser passada com
out. Veja  que  definimos a variável item_descrição mas ela não 
recebeu nenhum conteúdo inicialmente.

3. Function overloading
O sobregacarregamento de funções é feito tal como ocorre em c++.
Basta definir várias funções com assinaturas diferentes.
A assinatura  de  um método é definida pelo seu nome e sua lista
de parâmetros. Dois métodos diferem em sua  assinatura  se  eles
têm nomes ou lista de parâmetros diferentes. A assinatura difere
também  quando a lista de parâmetros tem quantidades ou tipos de
dados diferentes.

Em nosso código exemplo definimos duas vezes a função permuta_string();
uma versão para aceitar parâmetros por valor e outra versão para
aceitar parâmetros por referência.



// ***********************************************************
// Permuta o conteúdo de duas strings passadas por referência
    static void permuta_string(ref String txt01, ref String txt02)
    {
      String temp = "";
      temp = txt01;
      txt01 = txt02;
      txt02 = temp;
    } // permuta_string(ref, ref).fim
	

// Permuta o conteúdo de duas strings passadas por valor.
// *** Não emite o resultado para a função que a chamou.
    static void permuta_string(String txt01, String txt02)
    {
      String temp = "";
      temp = txt01;
      txt01 = txt02;
      txt02 = temp;
    } // permuta_string().fim

// Projeto prg_permuta - Arquivo: Program.cs
// Ilustra duas formas de passagem de argumentos
// Ilustra também function overloading
using System;

namespace prj_permuta
{
  class Program
  {
    static void Main(string[] args)
    {

      String caixa = "vazio";
      String mochila = "faca";
      String item_descricao;
      int item = 0;

      config_janela("projeto prj_permuta");
      mostrar_string("Conteúdo dos recipientes", "caixa", "mochila");
      mostrar_string("--------------------------------------------",
      caixa, mochila);
      
      // Passagem por valor
      permuta_string(caixa, mochila);
      mostrar_string("\n 1.Passagem por valor:", caixa, mochila);

      // Passagem por referência
      permuta_string(ref caixa, ref mochila);
      mostrar_string("2.Passagem por referência:", caixa, mochila);
      
      // Testando out
      mostrar_string("--------------------------------------------");
      for (item = 0; item < 3; item++)
      {
        pegar_mensagem(item, out item_descricao);
        mostrar_string("",item.ToString(),item_descricao);

      }

          
      Console.Read();
    } // Main().fim
    

    // ***********************************************************
    // Permuta o conteúdo de duas strings passadas por referência
    static void permuta_string(ref String txt01, ref String txt02)
    {
      String temp = "";
      temp = txt01;
      txt01 = txt02;
      txt02 = temp;
    } // permuta_string(ref, ref).fim

    // Permuta o conteúdo de duas strings passadas por valor.
    // *** Não emite o resultado para a função que a chamou.
    static void permuta_string(String txt01, String txt02)
    {
      String temp = "";
      temp = txt01;
      txt01 = txt02;
      txt02 = temp;
    } // permuta_string().fim

    // Mostra uma mensagem e duas strins
    static void mostrar_string(String msg, String txt01, String txt02)
    {
    Console.WriteLine ("\n {0}\n {1} {2}", msg, txt01, txt02 );

    } // mostrar_string().fim


    // Mostra uma mensagem 
      static void mostrar_string(String msg)
    {
      Console.WriteLine(" {0}\n", msg);

    } // mostrar_string().fim


    // Coloca uma mensagem variável recebida por referência
    static void pegar_mensagem (int item, out String msg)
    {
      switch (item)
        {
        
        case 1:
          msg = "Isto é uma faca!";
          break;
        
        case 2:
          msg = "Isto é uma mochila!";
          break;
         
        default:
          msg = "Nada de especial!";
          break;
        } // fim do switch

    } // pegar_mensagem().fim


    // Método para configurar a janela
    static void config_janela(String titulo)
    {
      // Vamos configurar a janela
      Console.ForegroundColor = ConsoleColor.Blue;
      Console.BackgroundColor = ConsoleColor.White;
      Console.Title = titulo;
      Console.SetBufferSize(80, 30);
      Console.SetWindowSize(80, 30);
      Console.Clear();
    
    } // config_janela() fim
  } // fim da classe
} // fim do namespace
