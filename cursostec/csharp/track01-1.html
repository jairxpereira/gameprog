1.1 Estrutura básica do programa na linguagem C#
1. Estrutura Geral


A imagem acima é do nosso primeiro programa em C#, escrito com o mínimo de
código possível para demonstrar a estrutura básica de qualquer programa
escrito nesta linguagem.

Veja agora o código do programa que produziu essa saída: 
Projeto Intro01:



 
 using System;

 namespace Gameprog
 {
    class Program
    {
        static void Main()
        {                        
        
        Console.WriteLine("Programar é legal!");   
        Console.Read();                                 

        } // fim: main()

    } // fim: classe Program

 } // fim: namespace Gameprog   



using System
A palavra-chave using adiciona as bibliotecas de classes e funções que serão
usadas dentro da aplicação. Neste caso estaremos utilizando a biblioteca System.





namespace Gameprog
{
} // fim: namespace Gameprog

O namespace, espaço de nomes, identifica uma coleção de classes. Esse conceito
foi criado para resolver conflito de classes de nomes iguais que ocorriam entre
os diversos desenvolvedores de software. 



 
class Program
{
} // fim: classe Program

A classe é um recipiente de métodos e propriedades que dá origem aos objetos
que herdam essas características. A classe Program e seu método main() é o
ponto de início de execução da aplicação. 

Com a finalidade de facilitar a leitura e o desenvolvimento dos programas, 
em nosso estilo de programação, após a chave (}) que fecha o bloco de código
identificamos o bloco sendo fechado com um comentário pertinente. Então,
será visto em nosso código fim da classe tal, fim da função tal e 
assim por diante.
	



static void Main()
{                        

Console.WriteLine("Programar é legal!");   
Console.Read();                                 

} // fim: main() 

Eis aqui então o método main() que identifica o início de execução da aplicação.
static indica que o método é único e particular, não sendo replicado a cada objeto.
Console.WriteLine() exibe os dados na tela.
Console.Read() faz uma captura de dados do teclado. Foi usada aqui para produzir
uma pausa para visualizar a tela de saída do programa.

2. Janela de console


Durante este curso as aplicações serão desenvolvidas no modo console que 
é suficiente para assimilar toda a conceituação central da linguagem C#.

Essa janela de console é produzida e controlada pela classe Console da
biblioteca System.

Veja abaixo outros métodos e propriedades da classe Console que vamos utilizar
para melhorar a estética dessa tela de saída durante nosso curso:

Console.Clear()
Esse método limpa a tela.

Console.Title
Essa propriedade configura ou recupera o título da janela.
	
BackgroundColor
ForegroundColor
Essas propriedades configuram, respectivamente, a cor de fundo
e a cor de frente da tela.

Exemplo: 	
Console.Title = "Intro01";
Console.BackgroundColor = ConsoleColor.White;
Console.ForegroundColor = ConsoleColor.Black;
Console.Clear();
