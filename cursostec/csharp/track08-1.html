08-1 Utilizando a classe String
1. Visão geral
O tipo string em C# é implementado pela classe String.   Então  há uma série
de métodos que facilitam o trabalho de manipulação de strings que  vamos ver 
nesse capítulo.

A classe String apresenta alguns métodos estáticos que podem ser usados para
manipular strings sem a necessidade da instanciação de objetos strings.
 
2. Declarando strings 
string empresa = "Gameprog";
String game = "Castlevania IV";

O objeto string pode ser declarado com string ou String; string é um apelido
que remete à classe String.

2.1 String literais com arroba (@)
Utilizando-se o arroba (@) é possível utilizar caracteres especiais na string
dispensando o uso da barra de escape (\). Isso é interessante para encurtar
a declaração de caminhos do sistema.

Exemplo: 
string scaminho = @"c:\windows"; 

Aonde for necessário um caminho em algum método pode ser utilizando uma string
desta forma.

Ainda, o simbolo @ permite repartir a declaração de uma grande string ou um
texto em múltiplas linhas:


string texto_grande = @"
--------------------------------------
Produzido por Gameprog
By Jair Pereira
Brasil - 2013";

Console.WriteLine("\n nome: {0} ", scaminho);
Console.WriteLine(texto_grande);      



3. Lenght

int ntam = texto_grande.Length;

A propriedade Lenght retorna o tamanho da string.

4.1.1 Acesso aos caracteres de uma string
Os colchetes, [], típicas do uso de array dão acesso aos caracteres individuais
de uma string.  Essa  forma  de  estruturação  de acesso aos dados de um objeto
através dos colchetes é chamada de indexer pelo C#. 



string empresa = "Gameprog";
ntam = empresa.Length;

for (int ncx = 0; ncx < ntam; ncx++)
 Console.WriteLine(" empresa[{0}] = {1}", ncx, empresa[ncx]);



4.1.2 CompareTo()
Essa  função  compara  esta string com outra. A função retorna zero (0) em
caso de igualdade das strings.  A  diferenciação  dos  caracteres  é  case 
sensitive, isto é, a questão caixa baixa\alta faz diferença na comparação.
O método estático string.Compare() tem funcionalidade equivalente.



string s1 = "Gameprog";
string s2 = "gameprog";

int sts = s1.CompareTo(s2);

if (sts == 0) show("As string são iguais (s1, s2)");
else
show("As strings são diferentes");
// Resulta em 'As strings são diferentes'



 4.1.3 operator+=()
 O operador += concatena o conteúdo de duas strings. Equivalente ao
 método estático string.Concat().
 
 

s1 = "Produzido por";
s2 = " Studio Gameprog";
// Ou s1 = string.Concat(s1, s2); 
s1 += s2;



 4.1.4 string.Copy( string str)
 Essa função retorna uma cópia da string especificada. O operador =() produz
 o mesmo resultado.
 

s1 = "Gameprog";
show(s2);
s2 = string.Copy(s1);  
// Ou s2 = s1 



 4.1.5 CopyTo()
 Essa função copia a quantidade de caracteres especificados da string fonte

// Vamos testar copyTo()

show("  s1.CopyTo(4, snome, 0, 4);");

 4.1.7 EndsWith( string str) \ StartsWith( string str)
 EndsWith() retorna verdadeiro se a string termina com a string especificada;
 StartsWith() retorna  verdadeiro  se a string especificada está no início da

show(@"Vamos testar EndsWith()\StartsWith()");
bool bSiteBrasileiro = false;
bool bWeb = false;
string site = "www.gameprog.com.br";

bWeb = site.StartsWith("www");

if (bWeb == true) show(" O protocolo de comunição é www");
 

 
 4.1.8 toUpper() \ toLower()
 toUpper() converte toda a string para caixa alta; toLower() retorna uma versão
 em caixa baixa da string.
 

show(@"Vamos testar toUpper()\ toLower()");
s1 = "Gameprog";
s1 = s1.ToUpper();
show(s1);
s1 = s1.ToLower();
show(s1);
pause();



4.1.9 Trim() \ TrimStart() \ TrimEnd()
Esta família de funções remove os espaços em branco de uma string.
TrimStart() 	- remove os espaço iniciais
TrimEnd()	- remove os espaços finais
Trim()		- remove os espaços iniciais e finais ao mesmo tempo

Ao invés de remover espaços, estas funções podem se configuradas para remover os 
caracteres especificados por uma array de caracteres.

 

show(@"Vamos testar TrimStart() \ TrimEnd() \ Trim()");
s1 = "     Gameprog  ";
Console.WriteLine("{0} {1} string original",s1, s1.Length);
show(s1);

s1 = s1.TrimStart();
Console.WriteLine("{0} {1} TrimStart()", s1, s1.Length);
show(s1);

s1 = s1.TrimEnd();
Console.WriteLine("{0} {1} TrimEnd()", s1, s1.Length);
show(s1);

s1 = "     Gameprog ";
Console.WriteLine("{0} {1} string original", s1, s1.Length);
char[] letras = new char[] {' ', 'g', 'G' };
s1 = s1.Trim( letras);      
Console.WriteLine("{0} {1} Trim()", s1, s1.Length);
show(s1);
pause();



4.1.10 Substring( nInicio, nQtd)

4.1.11 Split() \ string.Join()
Split() explode a string fonte, retornando uma  array  de  strings  que  estavam
delimitadas pelos separadores especificados.

string.Join() faz o caminho inverso, recebendo uma array de strings e retornando
uma única string  com  os  elementos  da  array  separados  pelos  delimitadores
especificados.



show("Vamos testar Split()");
s1 = "user=robocop;pwd=1234;banco=teste.sql;";
char[] separadores = new char[] { '=', ';' };
string[] palavras = s1.Split(separadores);
for (int ncx = 0; ncx < palavras.Length; ncx++)
show(palavras[ncx]);
// Produz uma array com seis strings: user,robocop, pwd, 1234, banco, teste.sql

// Vamos testar Join()
show("Vamos testar Join()");
s1 = string.Join(";", palavras);
show(s1);
// Produz uma string: user;robocop;pwd;1234;banco;teste.sql;



4.1.12 string.Format()
Esta função produz uma string formatada nos moldes da função WriteLine().



// Vamos testar format()
string nome = "Bill Gates";
int idade = 65;
decimal salario = 2100900.85M;

show("Testando string.Format()");
s1 = string.Format ("Nome: {0}, Idade: {1}, Salário: {2:C2}",
nome, idade, salario); 
show (s1);
// Retorna a string: Nome: Bill Gates Idade:65 Salario: R$ 2.100.900,85

4.1.14 Contais() \ IndexOf()
A função Contains() verifica se uma substring está contida na string fonte;
se estiver contida, utilize a função IndexOf() para obter o posicionamento 
da primeira ocorrência dela na string fonte.



// Vamos testar Contains() \ IndexOf();
show("Vamos testar Contains() \\ IndexOf()");
s1 = "Gameprog";
s2 = "prog";
show(s1); show(s2); 
if (s1.Contains(s2)) show("A string s2 está contida em s1");
int npos = s1.IndexOf(s2, 0); 
Console.WriteLine("na posição {0}", npos);
pause();



4.1.15 Insert() \ Replace()
A função Insert() insere uma string dentro da string fonte na posição 
especificada. A função Replace() substitui  uma  substring  que  está
contida na string fonte pela nova substring especificada.



// Vamos testar Insert() \ Replace()
s1 = "Gameprog".ToLower();
show(s1);
s2 = s1.Insert(0, "www.");
show(s2);
s2 = s2.Insert(s2.Length, ".com.br");
show(s2);
s2 = s2.Replace("gameprog", "nucleoararat"); 
show(s2);



4.1.16 PadLeft() \ PadRight()
Estas funções redefinem o tamanho da string, completando com espaços
ou outro carácter definido a diferença  do tamanho inicial da string.
PadLeft()  realiza  seu  trabalho  pela  esquerda  e PadRight() pela
direita.



// Vamos testar  PadLeft() \ PadRight()
show(" Testando PadLeft() \\ PadRight()");
s1 = "Gameprog"; 
s1 = s1.PadLeft(20, '-');
show(s1);
s1 = s1.PadRight(40, '-'); 
show(s1);
// Aproxidamente, resulta em
// ------------Gameprog
// ------------Gameprog-------------------



5. Strings são imutáveis
As strings são objetos imutáveis uma vez que foram criados. Apesar do aparente
trabalho  de  modificação  da  string original pelos vários métodos que vimos, 
o que  ocorre  é que eles retornam uma referência de uma nova string trabalhada
em  outro espaço alocado. Para trabalhos intensos com strings, é recomendado o
uso da classe StringBuilder.


Veja agora a listagem do programa exemplo:

// Método para configurar a janel
private static void config_janela(string titulo)
{
  // Vamos configurar a janela
  Console.ForegroundColor = ConsoleColor.Blue;
  Console.BackgroundColor = ConsoleColor.White;
  Console.Title = titulo;
  Console.SetBufferSize(80, 30);
  Console.SetWindowSize(80, 30);
  Console.Clear();
  Console.Write("\n");
} // config_janela() fim    

// Apresenta uma pausa para leitura da tela
private static void pause()
{
  Console.WriteLine("\n pause: pressione alguma tecla para continuar...");
  Console.ReadKey();
} // pause()
