05-7 Funções genéricas
1. Visão geral


Programação genérica é um recurso que permite construir classes ou funções que
se adaptam a qualquer tipo de dado.  Por  exemplo,  na linguagem C++ você pode
construir  uma  função  genérica  para  retornar  o dobro de um número, esteja 
ele  em  qualquer tipo de dado. Porém, em C# este recurso está limitado e essa
função  de  exemplo  não  pode  ser desenvolvida pois em C# os tipos genéricos
não  podem  sofrer  operações  aritméticas. Mesmo com essa e outras limitações
propositais,  a  programação  genérica  é  usada utilmente em C# para construir 
containers  ou  recipientes  de  dados  que  operam  com qualquer tipo de dado.
Posteriormente  vamos  ver a classe genérica List que permite construir listas
com qualquer tipo de dado incluindo objetos de qualquer classe. 

O foco  em  nosso  curso vai ser dado na utilização das classes genéricas que a
infraestrutura  que  cerca  a linguagem C# oferece. Por hora vamos aprender como
construir funções genéricas para suavizar posteriormente a curva de aprendizado 
das classes genéricas. 
 
2. Declarando uma função genérica


// Permuta o conteúdo de duas variáveis
private static void permutar<Generico>(ref Generico v1, ref Generico v2)
{
  Generico temp;
  temp = v1;
  v1 = v2;
  v2 = temp;
} // permutar<Generico>().fim



A marca de reconhecimento de uma função ou classe genérica é a presença de um 
nome simbólico entre os sinais de maior\menor: <Generico>

Utilizamos  a  palavra Generico mas poderia ser qualquer outra e frequentemente
o  tipo  genérico  é  nomeado  como   <T>   dentro  das  classes nativas do C#.

Quando o compilador encontra uma chamada  de função  genérica  chamada  com  um
tipo particular ele cria uma versão da função substituindo todas as referências
genéricas pelo tipo particular encontrado.

3. Chamando uma função genérica


// Variáveis inteiras
int x = 320;
int y  = 240;

// Variáveis de ponto flutuante
float fx = 3.1415f;
float fy = 6.2829f;

// Objetos da classe string
string sx = "Programar";
string sy = "É Legal";

// Vamos permutar os conteúdo das variáveis
permutar<int>(ref x, ref y);
permutar<float>(ref fx, ref fy);
permutar<String>(ref sx, ref sy);


Veja que na chamada, a função genérica recebe a informação do tipo particular 
que ela deve tratar.

4. Código fonte do programa exemplo




// Projeto prj_funcoesGenericas - Arquivo: Program.cs
// Este programa mostra o uso de funções genéricas
using System;

namespace prj_funcoesGenericas
{
  class Program
  {
    static void Main(string[] args)
    {
           
      config_janela("prj_funcoesGenericas");
      
      // Variáveis inteiras
      int x = 320;
      int y  = 240;

      // Variáveis de ponto flutuante
      float fx = 3.1415f;
      float fy = 6.2829f;

      // Objetos da classe string
      string sx = "Programar";
      string sy = "É Legal";

      // Valores originais
      Console.WriteLine("\n");
      Console.WriteLine(" Elementos originais:");
      Console.WriteLine(" Primeiro \t Segundo");
      Console.WriteLine(" -----------------------------------------------------");
      Console.WriteLine(" {0} \t\t {1} \t\t tipo inteiro", x, y);
      Console.WriteLine(" {0} \t {1} \t tipo float", fx, fy);
      Console.WriteLine(" {0} \t {1} \t tipo objeto string", sx, sy);
      
      // Vamos permutar os conteúdo das variáveis
      permutar<int>(ref x, ref y);
      permutar<float>(ref fx, ref fy);
      permutar<String>(ref sx, ref sy);

      // Os conteúdos foram permutados!
      Console.WriteLine("\n\n");
      Console.WriteLine(" Elementos originais tiveram seus valores permutados:");
      Console.WriteLine(" Primeiro \t Segundo");
      Console.WriteLine(" -----------------------------------------------------");
      Console.WriteLine(" {0} \t\t {1}", x, y);
      Console.WriteLine(" {0} \t {1}", fx, fy);
      Console.WriteLine(" {0} \t {1}", sx, sy);     
      Console.Read();

    } // Main() fim

    // Método para configurar a janela
    private static void config_janela( string titulo)
    {
      // Vamos configurar a janela
      Console.ForegroundColor = ConsoleColor.Blue;
      Console.BackgroundColor = ConsoleColor.White;
      Console.Title = titulo;
      Console.SetBufferSize(80, 30);
      Console.SetWindowSize(80, 30);
      Console.Clear();
    } // config_janela() fim    

    
    // Permuta o conteúdo de duas variáveis
    private static void permutar<Generico>(ref Generico v1, ref Generico v2)
    {
      Generico temp;
      temp = v1;
      v1 = v2;
      v2 = temp;
    } // permutar<Generico>().fim

  } // fim da classe Program
} // fim do namespace prj_funcoesGenericas
