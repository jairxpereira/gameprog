 15. Matrizes


        a) Matrizes simples
        A matriz é um conjunto de elementos acessados com um só nome,
        cada elemento é acessado individualmente através de um número
        denominado índice.
        
        DIM nome$(5)
        DIM NomeDaVariavel(nQtd%)

        Você usa o comando DIM para criar matrizes. Com esta sintaxe você cria
        matrizes simples ou unidimensionais.
        O argumento nQtd% determina o número máximo de elementos que a matriz
        contém. Neste exemplo,DIM nome$(5), nós criamos uma matriz string para
        guardar cinco nomes.

        Cada elemento da matriz pode ser gravado e acessado seguinte maneira:
        nome$(1) = "Jair"
        nome$(2) = "Maria"
        .
        .
        .
        nome$(5) = "Renata"

        Para acessar os elementos é só fornecer o índice. Ex.:
        FOR indice% = 1 to 5
        MsgBox indice%,,nome$(indice%)
        NEXT indice%

        Você também pode declarar matrizes desta maneira:
        DIM NomeDaVariável(nQtd%) AS TipoDeDado
        DIM nome(5) AS STRING
        DIM idade(5) AS INTEGER

        b)Matrizes bidimensionais

        Você pode imaginar uma matriz bidimensional como sendo uma
        tabela de linhas e colunas. Uma necessidade de uso de matriz
        bidimensional é quando você vai estocar coordenadas de tela
        que você necessita guardar valores para as coordenadas de
        linha e coluna.

        Você declara uma matriz bidimensional deste jeito:
        DIM NomeDaVariável (nQtd01%, nQtd02%)
        DIM TextoPos(3,2) AS INTEGER
        DIM TextoPos%(3,2)
        Aqui neste exemplo acima, nós criamos uma matriz bidimensional
        com três linhas e duas colunas. Para referir-se aos elementos
        é só fornecer o índice da linha e da coluna:

        TextoPos%(1,1)=9
        TextoPos%(1,2)=10
        TextoPos%(2,1)=11
        TextoPos%(2,2)=15
        TextoPos%(3,1)=13
        TextoPos%(3,2)=20



Visualização da Matrix TextoPos%(3,2)

Coluna_1  Coluna_2

  Linha 01   9   10  


  Linha 02   11   15   


  Linha 03   13   20   





 Vamos usar os valores dos elementos desta matriz para posicinar texto
 na tela com LOCATE. Observe no diagrama da visualização da matriz TextoPos%(3,2)
 que os valores para o parâmetro linha% de LOCATE estão na coluna_1, e os valores
 para o parâmetro coluna% de LOCATE estão na coluna_2 desta  matriz.
 Você pode usar assim os dados desta matriz bidimensional:

        FOR y% = 1  to 3
        x$ = str$(TextoPos%(y%,1)) ,, str$(TextoPos%(y%,2))
        MsgBox "Testando matrizes."
        NEXT y%

        c)Matrizes tridimensionais e multimensionais

        Para criar matrizes tridimensionais e multidimensionais bastar acrescentar
        o número de elementos da dimensão seguinte.

        DIM NomeDaVariável (nQtd01%, nQtd02%,nQtd03%,...)
        DIM Ponto3dPos(10,10,10) AS INTEGER

        d) Cuidado com matrizes

        O seu programa pára de funcionar quando você faz referência a um
        elemento com um índice maior que o número máximo de elementos.
        Se você declara DIM nome$(5) e aplica um nome$(6)="...", seu
        programa pára e acusa estouro de índice.
