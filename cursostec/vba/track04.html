 4. Rótulo,desvio incondicional e sinal dois pontos(:)

        a) Rótulo

        O rótulo é um nome seguido de dois pontos que serve para
        identificar trechos do programa.

        O programa é executado sequêncialmente mas através do
        rótulo essa linearidade pode ser quebrada.

        A função do rótulo é servir de referência para saltos condicionais
        ou incondicionais para outros trechos do programas devidamente
        identificados.

        Exemplos de rótulos:

        inicio:
                MsgBox "Produzido por Gameprog", , "Curso de VBA"

        final:
                MsgBox "FIM", , "Gameprog" 


        b) Saltos incondicionais e quebra de execução do programa

        GOTO

        O comando GOTO desvia a execução do programa para o rótulo
        especificado.

        Ex.:
		inicio:

		Dim nome As String
		nome = InputBox("Digite seu nome", "Coleta de informação")

		MostraNome:
		MsgBox nome,, "Nome"

		IF nome <> "parar" then GOTO inicio

	Este programa é executado indefinidamente até que o usuário digite
	parar na caixa de texto.

        c) O uso dois pontos (:)

        Você pode usar dois ou mais comandos do VBA numa mesma linha,
        bastando para isso separá-los por dois pontos.

	Ex.:
	nome = InputBox("Digite seu nome", "Coleta de informação") : MsgBox nome,, "Nome" 

        O comando REM, exclusivamente em sua forma abreviada,
        dispensa o uso dos dois pontos (:). Ex.:

        y = (x ^ 2) + 2 'Esta é a fórmula da parábola
