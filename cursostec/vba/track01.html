01. Conceitos básicos

Programa é um conjunto de declarações, comandos, funções e instruções
organizadas de maneira lógica, que o computador executa para produzir
um resultado.  Este  resultado  pode  ser  a  solução  de um problema 
matemático,  pode ser um editor de texto,  pode  ser um jogo ou outro 
software que tenhamos decidido construir.

Ao longo do curso você vai aprender gradualmente o que é e como usar
declarações,  comandos,  funções  e instruções que o VBA oferece. 

O  termo  algoritmo  é muito similar ao significado de programa;  
algoritmo  é  uma  sequência de passos lógicos  para  atingir um 
objetivo.  Geralmente   programa   apresenta  vários  algoritmos 
pequenos para se produzir o resultado final.

Comando é uma palavra que representa uma ordem que o computador
executa.  Por exemplo,  STOP  é um comando que faz parar imediatamente
a execução do programa VBA.

Função é um procedimento  que  retorna  um  valor.  Por exemplo, 
x = sqr(16) é uma função que retorna a raiz quadrada do número
entre parêntesis.

Declaração  é  uma palavra que estabelece uma informação, por 
exemplo, DIM x AS INTEGER é uma declaração que estabelece que
x é um número inteiro.

Instrução é uma palavra que orienta a lógica e o fluxo de dados
num programa. Por exemplo, IF idade > 18 THEN GOTO MaiorDeIdade
é uma instrução que desvia a execução do programa para o módulo 
MaiorDeIdade caso idade seja maior que o valor 18.

Sintaxe  é  a  maneira correta de se escrever o comando, a declaração, 
função  ou  instrução para que o processo seja realizado corretamente. 
Em resumo, são as regras de boa ortografia das palavras chaves usadas
em programação.

Não se preocupe em decorar termos e definições, no decorrer do curso,
a própria prática e o uso do VBA vai fazer os conceitos se fixarem
normalmente.
