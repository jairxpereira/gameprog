3.Primeiros comandos e operações matemáticas

        a) Os comandos END,STOP e REM
        END
        Este comando finaliza normalmente o programa VBA. Ex. END

        REM
        O comando REM permite que um comentário seja adicionado no
        corpo do programa sem causar efeito em sua execução. Ex.:
        REM Isto aqui é um teste de programação.
        O comando REM pode ser substituído por sua forma abreviada,
        o sinal de apóstrofe simples (').

        Use o REM para informar o que faz as partes do seu programa
        ou para desabilitar certos trechos por questões de testes.


        STOP
        O comando STOP é usado para quebrar imediamente a execução
        programa e retorna ao editor de código do VBA.
		
		Exemplo:
		STOP        

        b) Operações matemáticas

        Em VBA as operações matemáticas são definidas por estes
        simbolos:

                        + Adição
                        - Subtração
                        * Multiplicação
                        / Divisão inteira
                        \ Divisão exata
                        ^ Potênciação
                        MOD Retorna o resto de uma divisão

        Você pode criar expressões numéricas usando variáveis.
        Você pode usar parentêsis para expressões mais complexas. Exemplos:

        y = x ^ 2 + 4
        x = (y ^ 2) - 2
        msgbox str$(10 mod 2)
