03. Exibindo os tipos de dados na tela

Abaixo, em nosso programa printf.c,  nós usamos  esta  linha  mostrar
uma string na tela:

printf("%s\n",sNome);

Em programação sintaxe significa a maneira  correta  de  se  escrever
uma função de modo  que  ela  funcione  perfeitamente.   Quando  você
soletra mal o nome de uma função ou confunde a  ordem  de  argumentos
de entrada ou saída, o compilador lhe informará que você  cometeu  um
erro de sintaxe.

A sintaxe da função printf() é printtf(sControle, Variáveis, Vars...);
sControle representa a string de controle na  qual  vem  mensagens  e
códigos de formatação que indicam o tipo de dado que  será  impresso.
Depois da string de controle os próximos argumentos  são as variáveis
que terão o seu conteúdo impresso na tela.

Para imprimir uma string o código de controle é %s, o \n indica que o
cursor deve ir para o início da linha seguinte.  O código de controle
para mostrar um inteiro é %i.

Teste o programa abaixo:



/* Programa: printf.c - track03 */
#include "stdio.h"
#include "stdlib.h"

int main(void)  {
char sNome[]="Megaman";
int nIdade = 26;

/* Configura a janela */
system("title printf.c");
system("color F1");

printf("\n Nome: %s - %i anos (2013) \n\n",sNome, nIdade);

system("pause");

return 1;
}









Veja abaixo a tabela que traz os códigos de formatação  para  mostrar
os respectivos tipos de dados na tela:




Tipo de dadoCódigo de formatação


string%s


char%c ou %d p/ mostrá-lo como número


int%i ou %d


short int%i ou %d


long%li


float%f


double%lf


unsigned int%u


unsigned long%lu


unsigned short int%hu


Veja abaixo o programa que testa os tipos de dados básicos:


/* Programa: datatype.c  - track03 */
#include "stdio.h"
#include "stdlib.h"

int main(void) {

char            sString[]   = "Testando tipo de dados";
char            cRank       = 'A';
unsigned int    nInteiro    = 50000;
long            nLongo      = -568600;
short int       nShort      = 15800;
unsigned char   nEnergia    = 98;
float           pi          = 3.141516;

/* Configura a janela */
system("title datatype.c");
system("color F1");


printf("\n\n");
printf(" Char:             %c\n", cRank);
printf(" String:           %s\n", sString);
printf(" Unsigned Inteiro: %u\n", nInteiro);
printf(" Longo:            %li\n", nLongo);
printf(" Unsigned char:    %d \n", nEnergia);
printf(" Short int:        %hi \n", nShort);
printf(" float:            %f \n", pi);
printf(" Hexadecimal:      %x  %X \n", nShort, nShort);
printf("\n\n");

system("pause");
return 1; }
