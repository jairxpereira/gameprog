07. Repetição condicional

Há ocasiões em que temos a necessidade de se repetir a execução de um
processo um determinado número de vezes  ou  repetí-lo  conforme  uma
situação seja verdadeira ou não.

A trajetória da repetição é circular, considerando que  há  o  início
da execução de certo  código,   há  na  sequência  a  verificação  da
variável que controla o número de repetições efetuadas, há  novamente
o retorno para o início desde que a condição para  isto  se  mantenha
verdadeira. Em programação, essa repetição  controlada  é  denominada
looping ou laço.

Na linguagem C os loopings são realizados pelas instruções for, while
e o conjunto do...while.

Preste atenção no programa exemplo  e  no  seu  resultado  de  saída:


/* Programa: loop01.c - track07 */
/* Este programa testa a instrução for */
#include "stdio.h"
#include "stdlib.h"

int main(void)  {

int ncx;

/* Configura a janela */
system("title loop01.c");
system("color F1");
printf("\n\n");

for (ncx = 1; ncx <= 3; ncx = ncx + 1) printf (" %i \n",ncx);

printf("\n");
for (ncx = 0; ncx < 10; ncx = ncx + 2)
printf (" %i \n",ncx);

printf("\n\n");
system("pause");

return 1;
}




 



Na sintaxe mais básica  do  for, você deve definir o valor inicial da
variável que controla o laço,  a situação lógica com esta  variável e
o passo positivo  ou  negativo  dela.  O  passo  positivo  é  chamado
incremento e o passo negativo  é  chamado  decremento  ou  incremento 
negativo.

Isto é feito da seguinte maneira:
for (valor Inicial; expressão lógica; incremento) código;
for (ncx = 1; ncx <= 3; ncx = ncx + 1) printf ("%i \n",ncx);

Aqui nesta linha de código, determinamos que  a função  printf()  vai
ser executada 3 vezes.

ncx = 1; 
Esta instrução define o valor inicial do laço.

ncx <= 3;
Esta instrução define que o laço vai ser repetido  enquanto  ncx  for
menor ou igual ao valor 3.

ncx = ncx + 1
define que o incremento de ncx será de um em um.

A expressão acima, ncx = ncx + 1, você pode encurtar para ncx++
que é uma expressão equivalente em c.

Depois do for  o código pode estar na mesma linha,  imediatamente  na
linha seguinte, ou poder ser um  bloco  de  código  delimitado  pelas
chaves.

for (valor Inicial; expressão lógica; incremento)
código;

for (valor Inicial; expressão lógica; incremento) {
bloco de  código;
(...);
}

7.1 Laço decrescente

O laço pode ser feito usando uma configuração decrescente,  iniciando
a  variável,   estabelecendo   a   expressão  lógica  corretamente  e
decrescendo o valor da variável.

Ex.:
for (ncx = 10; ncx > 0; ncx = ncx - 1) printf("Teste ");

Aqui  vai ocorrer uma contagem decrescente de 10 até 1.  A  expressão
ncx = ncx - 1  pode  ser  abreviada  para  a  expressão  ncx--
Posteriormente você vai  obter  mais  detalhes  sobre  os  operadores
unários representados por ++ e --

7.2 for - sintaxes alternativas

A linguagem C  possui uma sintaxe muito versátil que facilita muito a
construção dos nossos programas. A instrução for possui sintaxes muito
práticas.   Você  pode  suprimir  tudo  ou  qualquer  um  dos  pontos
integrantes  da  sintaxe  básica  tratada  no  tópico  acima.

for (;;) código;

Ex.:
int ncx = 0;
for (;;) {
printf ("%i \n",ncx);
ncx++;
if (ncx == 17) break;
}

Aqui nesta sintaxe o código ou  um  bloco  de  código  será  repetido
infinitamente  ou indefinidamente até que você use um break para sair
do laço.   Nesta sintaxe você deve controlar todos os fatores do laço
dentro do código ou do bloco de código que você pode usar:  você deve
inicializar   uma   variável,   incrementar   esta   variável   e  um
if(...) break; para finalizar este laço.

for (valor inicial;;incremento) código;
for (valor inicial;expressão;) código;
for (;;incremento) código;

O que você deve ter em mente é  que você deve tratar dentro do código
ou do bloco de código o ponto que você suprimir dentro dos parêntesis.

7.3 for - com função() na expressão lógica

Você pode usar uma função em qualquer ponto do for,   mas quando esta
função ocorrer dentro da expressão  lógica,  o  laço  será  executado
enquanto a função não retornar 0  que é o valor que representa  falso.

Ex.: for (ncx = 0; teste(), ncx++) printf("Teste ");

Veja o programa exemplo:


/* Programa: loop02.c - track07 */
/* Este programa testa a instrução for */
#include "stdio.h"
#include "stdlib.h"

/* Variável global mcx */
int mcx;

/* Função teste() */
int teste(void)
{
mcx = mcx - 1;
return mcx;
}

/* Função main()  */
int main(void)  {

int ncx = 0;
mcx = 6;

/* Configura a janela */
system("title loop02.c");
system("color F1");
printf("\n\n");

printf(" ncx - Var. de controle \n ----------------------- \n");

for (ncx=0; teste(); ncx++)
 {
   printf (" ncx:%i  mcx:%i\n", ncx, mcx);
 }

printf("\n\n");
system("pause");
return 1;
}




 




7.4 while

A instrução while repete o código ou  um  bloco  de  código  enquanto
uma  situação  permanece  verdadeira.

while (expressão) código;

while (expressão) {
bloco de código; }

Ex.:
int ncx = 0;
while (ncx < 10) {
ncx++;
printf("%i \n", ncx);
}

Você pode usar uma função no lugar da expressão lógica. O  código  ou
o bloco de código será executado  até  a  função  retornar  0,  falso.

Ex. while (teste());

Veja o programa exemplo e sua tela  de  saída,  ele  exemplifica  uma
diferença  importante  entre  for  e  while:


/* Programa: while.c - track07 */
/* Este programa testa a instrução while */
#include "stdio.h"
#include "stdlib.h"

/* Variável global */
int mcx = 6;

int teste(void) {
mcx--;
printf(" mcx:%i  - teste( )\n",mcx);
return mcx;
}

int main(void)  {

/* Configura a janela */
system("title while.c");
system("color F1");
printf("\n\n");

while (teste());

printf("\n");
system("pause");

/* Este looping não roda - condição inicial falsa */
mcx = 5;
while (mcx > 10)
{
 mcx++;
 printf("mcx: %i - main( )\n",mcx);
}

printf("\n\n");
system("pause");
return 1;
}




 


A expressão é avaliada no início do código, então a  instrução  while
não executa  o  código  quando  se  depara  inicialmente  com  a  sua
expressão retornado falso. Perceba que no programa exemplo,  o  bloco
de código do segundo  while  não  é  processado,  pois  a  situação é 
 mcx > 10  e  o  valor  de  mcx  é  5,   acarretando  uma   falsidade.

7.5 do...while

O conjunto  do...while  repete condicionalmente um bloco de código da
mesma maneira que a instrução while com  a  importante  diferença  de
que a checagem da condição é feita  no  final  do  bloco  de  código.
Essa diferença acarreta  que o bloco é executado uma vez mesmo quando
a condição é falsa.

Esta é a sintaxe de uso:

do código; while(condição);
do ncx++; while(ncx < 10);

do {
bloco de código;
...;
} while (condição);

do {
ncx++;
printf("%i \n",ncx);
} while (ncx < 10);

Veja o programa exemplo e preste atenção  na  saída,  perceba  que  o
bloco de código controlado por do...while rodou uma vez mais porque a
checagem da condição só ocorre no final do bloco.


/* Programa: dotst.c - track07 */
/* Este programa testa a instrução while */
#include "stdio.h"
#include "stdlib.h"

int main(void)  {
int ncx=0;

/* Configura a janela */
system("title dotst.c");
system("color F1");
printf("\n\n");

do {
ncx++;
printf(" %i \n", ncx);
} while (ncx < 3);

/* Avança ncx até 30 */
do ncx++; while(ncx < 30);

do {
ncx++;
printf(" %i ==> Executado sob condicao falsa \n", ncx);
} while (ncx < 2);

printf("\n\n");
system("pause");
return 1;
}
