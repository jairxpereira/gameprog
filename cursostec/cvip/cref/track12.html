12.01 assert
        Esta função testa uma condição, abortando caso resulte em zero o valor
        dessa condição testada. Em caso de condição igual a zero, o programa é
        abortado; antes disso assert() exibe o nome da variável que resultou em
        zero, o nome do programa em C e o número da linha.
        
        void assert(int teste);
        assert(nTeste);

        Ex.:    int a,b=3,c=3;
                c = a - b;
                assert(c);
                /* O programa é abortado porque c resulta em zero.*/

        Veja este programa exemplo:
        #include "stdio.h"
        #include "conio.h"
        #include "assert.h"
