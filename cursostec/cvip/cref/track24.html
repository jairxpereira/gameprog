24.02 _control87
        Esta função configura a palavra de controle das operações com ponto
        flutuante.
        
        unsigned int _control87(unsigned int new, unsigned int mask);
        nSts = _control87(nNewSts, nBitMask);

        nNewSts...Nova palavra de configuração para os processos de ponto flutuante
        nBitMask..Mascara do bit que será ligado na palavra de configuração
        nSts......A função retorna a nova configuração da palavra de controle.

        Para configurar a palavra de controle dos processos de ponto flutuante,
        para que o bit de controle dessa palavra seja ligado é necessário haver
        uma correspondência de bit entre nNewSts e nBitMask, assim a mesma posição
        binária de nNewSts e nBitMask deve ter o mesmo valor.

        Para nNewSts e nBitMask você tem estas constantes definidas em float.h:
        ------------------------------------------------------------------------

        SW_INVALID........0x0001....Invalid operation
        SW_DENORMAL.......0x0002....Denormalized operand
        SW_ZERODIVIDE.....0x0004....Zero divide
        SW_OVERFLOW.......0x0008....Overflow
        SW_UNDERFLOW......0x0010....Underflow
        SW_INEXACT........0x0020....Precision (Inexact result)

        MCW_EM............0x003f....interrupt Exception Masks
        EM_INVALID........0x0001....invalid
        EM_DENORMAL.......0x0002....denormal
        EM_ZERODIVIDE.....0x0004....zero divide
        EM_OVERFLOW.......0x0008....overflow
        EM_UNDERFLOW......0x0010....underflow
        EM_INEXACT........0x0020....inexact (precision)

        MCW_IC............0x1000....Infinity Control
        IC_AFFINE.........0x1000....affine
        IC_PROJECTIVE.....0x0000....projective

        MCW_RC............0x0c00....Rounding Control
        RC_CHOP...........0x0c00....chop
        RC_UP.............0x0800....up
        RC_DOWN...........0x0400....down
        RC_NEAR...........0x0000....near

        MCW_PC............0x0300....Precision Control
        PC_24.............0x0000....24 bits
        PC_53.............0x0200....53 bits
        PC_64.............0x0300....64 bits

        CW_DEFAULT........RC_NEAR+PC_64+IC_AFFINE+EM_UNDERFLOW+EM_INEXACT

        Ex.:    _control87(MCW_EM,MCW_EM);

24.03 _fpreset
        Esta função reiniciliza o pacote de rotinas de ponto flutuante.
        Esta função é usada em conjunto com a série de funções exec...()
        e spawn...() de process.h, desde que processos filhos podem alterar
        o flag de status das operações com ponto flutuante.
        
        void _fpreset(void);
        _fpreset();

        Ex.: _fpreset();

24.04 _status87
        Esta função retorna o flag de status das operações com ponto flutuante.
        
        unsigned int _status87(void);
        nSts = _status87();

        Ex.:    unsigned int nSts;
                nSts = _status87();
                printf("%d\n",nSts);
