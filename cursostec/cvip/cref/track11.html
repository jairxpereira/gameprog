 11.01 isaalnum
        Esta função retorna true se o byte testado for uma letra ou um número.
      
        isalnum(c);
        nSts = isalnum(byte);

        Ex.:    unsigned char byte='A';
                int nSts;
                nSts = isalnum(byte);
                printf("%d\n",nSts);
                /* Resulta em 4 */

        Detalhes:       Em nossos testes, a função retornou 4 para letras maiúsculas,
                        8 para letras minúsculas e 2 para números.

 11.10 isspace
        Esta função retorna (1) true se o byte testado for um espaço,chr(32).
        
        isspace(c);
        nSts = isspace(byte);

        Ex.:    char sMsg[]="abc abc";
                unsigned char byte;
                int cx,nSts;

                for (cx=0;sMsg[cx] != NULL; cx++) {
                byte = sMsg[cx];
                nSts = isspace(byte);
                printf("%c.%d\t",byte,nSts);}

                /* Resulta em
                a.0     b.0     c.0      .1     a.0     b.0     c.0 */

 11.13 toascii
        Esta função converte um byte maior que 127 num código ascii padrão.
        Esta função opera esta conversão por zerar o sétimo bit do byte,
        que é o bit da extrema esquerda do byte.
      
        int toascii(int ch);
        nStdAsc = toascii(byte);

        Ex.:    unsigned char byte=170;
                int nSts;
                nSts = toascii(byte);
                printf("%d\n",nSts);
                /* Resulta em 42 */
.......................................................................................
 11.14 _tolower
        Esta função converte uma letra maiúscula para um letra minúscula.
      
        int _tolower(int ch);
        cLower = _tolower(cUpper);

        Ex.:    unsigned char cLower,cUpper;
                cUpper = 'A';
                cLower = _tolower(cUpper);
                printf("%c\n",cLower);
                /* Resulta em a */

        Detalhes:       Caso cUpper não for um letra maiúscula, o valor de retorno da
                        função é imprevisível.
.......................................................................................
 11.15 tolower
