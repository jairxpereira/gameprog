12. Estrutura de dados e Tipo Definido pelo usuário 2/2

12.1 typedef 

A instrução typedef  é  usada  para  criar  novos  tipos  através  da
redefinição dos tipos básicos sob outros nomes.  Por exemplo, sabemos
que  um byte não-sinalizado é um valor cuja faixa está entre 0 e 255.
Podemos  criar  um  tipo  "byte"  desta  forma:

typedef TipoBasico sNovoTipo;
typedef unsigned char byte;
byte cLetra = 'x';
byte nIdade = 250;

Veja acima que  depois podemos usar o novo tipo da mesma  forma  como
usamos  os  tipos  básicos.

12.2 typedef com struct

O uso de  typedef  com  struct  torna  mais  confortável  o  uso  das
estruturas, pois ela redefine uma estrutura como  sendo  um  tipo  de
dado e assim encurtando  a  sintaxe  de  declaração  das  instâncias.

Você  deve  usar  typedef struct  para  declarar  as  estruturas.
Sua  sintaxe  geral  é  esta:

typedef struct StructNome {
tipo vVar;
tipo vVar;
...
tipo vVar;
} NomeTipo...;

StructNome.......Nome da estrutura
tipo.............Tipo da variável int, float, char...
vVar.............Nome da variável
NomeTipo.........NomeTipo  agora  representa  a  instrução  para  se
declarar estruturas do tipo StructNome,  bastando usar NomeTipo xVar;
para  criar  uma  instância  desta  estrutura.

Exemplo:


/* Declara a estrutura xFuncionario como um tipo definido */
typedef struct Funcionario
{
 char sNome[40];
 byte nIdade;
 float nSalario;
} xFuncionario;
	(...)

/* Declara programador diretamente como tipo xFuncionario */
xFuncionario programador;
/* Preenche programador */
programador.nSalario = 3870.99;
programador.nIdade = 85;
strcpy(programador.sNome,"Jair Pereira");



12.3 enum: enumerações

Há um tipo de dado  estrutural  denominado  enumeração  que  consiste
numa lista de valores  constantes  que  facilita  na  memorização  de
valores  e  na  leitura  do  código  do  programa.

O modo de declaração de uma enumeração é semelhante ao de uma struct:


enum ListaNome {  NOME01 = nValor,
		 NOME02,
		 (...),
		 NOME0N};


/* Declara uma enumeração */
enum codigos
{
 GRAVAR,
 APAGAR,
 MODIFICAR,
 RETROCEDER
};


O que aparece é uma lista de valores fixos, convenientemente nomeados
por  NOME01,  NOME02,  GRAVAR,  APAGAR.

Opcionalmente você pode definir o valor de cada item  da  enumeração;
você pode apenas definir o valor do primeiro item  e  neste  caso  os
itens  seguintes  tem  o  valor  do  item  anterior  +  1.

Na ausência da definição do valor do primeiro item  é  assumido valor
zero. No nosso exemplo, supomos que uma função executará processos de
gravação, apagamento, modificação de dados, e que esta  mesma  função
receberá como argumento um valor inteiro que identifica as operações;
o uso da enumeração  dispensa-nos  de  decorar  os  valores  de  cada
operação.

Posteriormente no programa basta usar o nome do item  da  enumeração,
durante o processo de compilação  o  nome  do  item  é  trocado  pelo
valor  do  item.  Ex.: Executa (MODIFICAR);

12.4 union: uniões

Outro tipo de dado estrutural é a union que  também  é  declarada  de
de maneira semelhante a uma struct.  Nela você declara vários  tipos,
porém o grande destaque da union é que é reservado apenas espaço para
suportar o maior dos tipos que você declarou.  Veja  a  sintaxe desta
estrutura:

union UnionNome {
tipo vVar01;
tipo vVar02;
} xVar, lista de variáveis;

Os elementos vVar01, vVar02  compartilham o mesmo espaço  de  memória
implicando que você somente pode usar um ou outro, nunca os  dois  ao
mesmo tempo. O fato de você inicializar o valor do maior  elemento, o
tipo cuja faixa de valor é maior,  sobrescreve  o  conteúdo  do  tipo
menor.

Exemplo:


/* Declara um dado do tipo union */
union Byte_ou_Int {
unsigned char mybyte;
unsigned int myint;
} ByteOuInt;

	(...)

/* ---------------- Mostra a union ByteOuInt */
printf(" ByteOutInt.mybyte:     %X %d \n", ByteOuInt.mybyte,ByteOuInt.mybyte);
printf(" ByteOuInt.myint:       %X %d \n", ByteOuInt.myint,ByteOuInt.myint);


Resulta em:
ByteOutInt.mybyte: A0 160
ByteOutInt.myint:  41A0 16800

Perceba que  o  valor  do  primeiro  elemento  foi  sobrescrito  pelo
conteúdo do segundo  elemento  que  foi  inicializado  na  sequência.

12.5 Programa exemplo: thetypes.c

Veja  abaixo  o  programa  exemplo  que  ilustra  o  uso  de  typedef,
typedef struct, union:


 




/* Programa: thetypes.c - track 12*/
/* Ilustra tipos definidos pelo usuário*/
#include "stdio.h"
#include "stdlib.h"
#include "string.h"

/* Protótipo de initSys() */
void initSys( char *title);

/* Declara um dado do tipo union */
union Byte_ou_Int {
unsigned char mybyte;
unsigned int myint;
} ByteOuInt;

/* Declara uma enumeração */
enum codigos
{
 GRAVAR,
 APAGAR,
 MODIFICAR,
 RETROCEDER
};

/* Declara byte como um tipo definido por usuário */
typedef unsigned char byte;

/* Declara a estrutura xFuncionario como um tipo definido */
typedef struct Funcionario
{
 char sNome[40];
 byte nIdade;
 float nSalario;
} xFuncionario;

/* -------------------------- main() ----------------------------- */
int main(void)  {

/* Declara programador diretamente como tipo xFuncionario */
xFuncionario programador;

initSys("title thetypes.c");

/* Preenche programador */
programador.nSalario = 3870.99;
programador.nIdade = 85;
strcpy(programador.sNome,"Jair Pereira");

/* Inicializa a estrutura union ByteOuInt */
ByteOuInt.mybyte = 170;
ByteOuInt.myint = 16800;

/* ---------------- Mostra programador */
printf(" Programador: %s   \n",programador.sNome);
printf(" Idade:       %d   \n", programador.nIdade);
printf(" Salario:     %.2f \n\n",programador.nSalario);

/* ---------------- Mostra a enumeração codigos*/
printf (" Codigos: \n");
printf (" gravar:    %d \t apagar:     %d \n",GRAVAR,APAGAR);
printf (" modificar: %d \t retroceder: %d \n\n",MODIFICAR,RETROCEDER);

/* ---------------- Mostra a union ByteOuInt */
printf(" ByteOutInt.mybyte:     %X %d \n",ByteOuInt.mybyte,ByteOuInt.mybyte);
printf(" ByteOuInt.myint:       %X %d \n",ByteOuInt.myint,ByteOuInt.myint);

getchar();

return 1;

}

/*************** Função initSys()  */
void initSys( char *title)
{
/* Configura a janela */
system(title);
system("color F1");
printf("\n\n");
}
