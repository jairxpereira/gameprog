11.6 Nível de detalhe dinâmico
1.1 Visão geral


A classe ProgressiveMesh produz um  modelo  3d  no  qual  o  nível  de
detalhe  pode  ser reduzido ou ampliado  dinamicamente.   Apesar desse
objeto realizar seu trabalho numa velocidade muito boa é mais prudente
produzir vários  meshes progressivos com níveis de detalhes diferentes
para utilizar ao longo da aplicação. O processo inicial de produção do
mesh  progressivo  também  necessita  de  limpeza  prévia no modelo 3d 
original.


1.2 Estrutura principal da aplicação




Esta aplicação mostra um modelo 3d e uma versão  dinâmica  do  mesmo.
Pressione W para ligar ou desligar o modo de  renderização  wireframe.
A base inicial dessa aplicação é o prj_Modelo3d do tópico 4.4 que foi
adaptado para este tópico corrente. A função gerarModeloProgressivo()
gera o modelo progressivo no qual o nível de detalhe é alterado pelas
teclas  (+) e (-)  respectivamente aumentando ou reduzindo o nível de 
detalhe do modelo progressivo.

O mesh progressivo é renderizável  com  seu  método  DrawSubset(), no
entanto  para aproveitamento da função desenharObjeto() esse método é
renderizado  como  BaseMesh  nesta  função  que  teve  sua assinatura
alterada para aceitar esse objeto base.  Em algumas partes do código,
o mesh progressivo recebeu um cast para BaseMesh para a obtenção dos
resultados adequados.

progressivo  e  vai  ser  alterada  pelo pressionamento das teclas de

um grande afastamento deve produzir um mesh mais simples e uma grande
proximidade deve gerar um mesh mais bonito e detalhado.

Essa linha de código produz o mesh progressivo a partir do mesh limpo.
Este mesh inicialmente nasce o mais simplificado possível na direção
do limite sugerido (1).

Este bloco de código reconfigura o mesh para mostrar o máximo possível
de detalhes.  Então  inicialmente o modelo pogressivo é apresentado em
sua melhor aparência.

Depois com o pressionamento das teclas (+) ou (-)  o nível de detalhe
na renderização percorre os pontos intermediários  destes dois níveis
extremos de detalhamento do modelo 3d.

2.5 Controlando o nível de detalhe

private void Tela_KeyDown(object sender, KeyEventArgs e) 
{
 // Controle de modo wireframe
 if (e.KeyCode == Keys.W)  bWireframe = !bWireframe; 
 if (bWireframe) device.RenderState.FillMode = FillMode.WireFrame;

conseguindo  renderizar o Mesh e o ProgressiveMesh sem alterações no

O  casting  ocorrido  aqui é necessário conforme a razão explicada na
função Tela_KeyDown().

2.7 Desenhando o BaseMesh
Repare na assinatura desta função que foi alterada para receber o Mesh 
e o ProgressiveMesh para renderização sem alteração no bloco de código
interno.


private void desenharObjeto(BaseMesh obj, Propriedades3D props) 
{

 // Ajusta rotação do objeto 3d
 Matrix obj_rot = Matrix.RotationX(props.rotation.X + angulo) * 
 Matrix.RotationY(props.rotation.Y)  * 
 Matrix.RotationZ(props.rotation.Z);  

 // Ajusta posição do objeto 3d
 Matrix obj_pos = Matrix.Translation(props.position);  

 // Tranfere posição e rotação para o mundo
 device.Transform.World = obj_rot * obj_pos; 

 // Prepara e aplica  material\textura no objeto
 for (int ncx = 0; ncx < g_meshMtl.Length; ncx++) 
 {
 // Informa ao dispositivo o material a ser utilizado
 // na renderização
 device.Material = g_meshMtl[ncx];  
 device.SetTexture(0, g_meshTex[ncx]);  

 // Renderiza o mesh
 obj.DrawSubset(ncx);  
 } // endfor

} // desenharObjeto().fim

    private void AtualizarCamera() 
    {
      // Dados para a configuração da matriz de projeção
      int largura = this.Width; // largura da janela
      int altura = this.Height;  // altura da janela
      float aspecto = largura / altura; // aspecto dos gráficos
      float campo_visao = (float)Math.PI / 4; // Campo de visão
      float corte_perto = 1.0f;

      // Essa variável foi modificada para acomodar o tamanho
      // do modelo

      // Atualiza ângulo para dar movimento ao objeto 3d
      angulo += 0.02f;

      // Mostra a parte interna do polígono

      // Configura a matriz de projeção
      device.Transform.Projection = Matrix.PerspectiveFovLH(campo_visao, 
          aspecto, corte_perto, corte_longe);  

      // Dados para a configuração da matriz de visualização

      Vector3 cam_orientacao = new Vector3(0, 1.0f, 0);

    private void desenharObjeto(BaseMesh obj, Propriedades3D props) 
    {

      // Ajusta rotação do objeto 3d
      Matrix obj_rot = Matrix.RotationX(props.rotation.X + angulo) * 
        Matrix.RotationY(props.rotation.Y)  * 
        Matrix.RotationZ(props.rotation.Z);  

      // Ajusta posição do objeto 3d
      Matrix obj_pos = Matrix.Translation(props.position);  

      // Tranfere posição e rotação para o mundo
      device.Transform.World = obj_rot * obj_pos; 

      // Prepara e aplica material\textura no objeto
      for (int ncx = 0; ncx < g_meshMtl.Length; ncx++) 
      {
        // Informa ao dispositivo o material a ser utilizado
        // na renderização
        device.Material = g_meshMtl[ncx];  
        device.SetTexture(0, g_meshTex[ncx]);  

        // Renderiza o mesh
        obj.DrawSubset(ncx);  
      } // endfor

    } // desenharObjeto().fim
