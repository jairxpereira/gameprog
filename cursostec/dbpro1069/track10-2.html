10-2. Bitmaps

 VISÃO GERAL
 

 
 
 
 Bitmap  é  um  tipo  de  arquivo  de  imagem que pode ser usado para as 
 seguintes finalidades: ilustração, textura ou  servir  como  célula  de 
 desenho animado. O nome deste arquivo geralmente termina com a extensão
 .bmp.

 A tela visível é tratada como um arquivo de bitmap, o bitmap zero. Você
 pode copiar conteúdo de um bitmap para  outro, fazendo  assim  montagem 
 de  cenários  e  animação.  Para   você  exibir  um  bitmap  você  deve 
 carregá-lo ou copiá-lo para o bitmap zero que vimos que é  a  sua  tela
 visível.

 A cada bitmap carregado deve ser dado um número de identificação através
 do qual esse bitmap vai ser manipulado e sofrer os vários efeitos que  o
 Darkbasic oferece. Esse número  de  identificação  geralmente é  chamado 
 handle ou alça de manipulação. O bitmap que é  identificado  com  zero é 
 mostrado imediatamente na tela.
 
 
1. Funções de manipulação de bitmap 
 -------------------------------------------------------------------------------
 load bitmap bmp$, nHandle
 Esta função carrega o arquivo  bitmap bmp$ e identifica esse bitmap com
 o número dado em nHandle. Você pode carregar até 32 bitmaps, sendo este
 então  o  limite  para  nHandle.  Se nHandle do bitmap for omitido, ele 
 recebe o valor zero, sendo por isso imediamente exibido na tela.

 Exemplo: 
 ' Carrega e mostra o bitmap textura.bmp 
 load bitmap "c:\media\textura.bmp" 

 copy bitmap nbmpFonte, nbmpDestino
 Esta função tem duas sintaxes, nesta primeira sintaxe o bitmap fonte
 é copiado para o bitmap de destino.

 Ex.: 
 load bitmap "c:\media\textura.txt", 1
 copy bitmap 1, 0 

 copy bitmap nbmpFonte, nTop, nLeft, nBottom, nRight, nbmpDestino,
 nTop, nLeft, nBottom, nRight
 Nesta sintaxe, a parte especificada do bitmap fonte é copiado para
 a parte especificada do bitmap de destino.

 nbmpFonte      - Handle do bitmap fonte
 nbmpDestino    - Handle do bitmap destino

 nTop, nLeft    - Coordenada que demarca o inicio do quadrado
 nBottom, nRight - Coordenada que demarca o final do quadrado.



rem Testando Copy bitmap()

hide mouse

load bitmap "c:\media\textura.bmp",1

copy bitmap 1,0,0,128,128,0,10,10,138,138

rem Copiamos (0,0)-(128,128) de textura.bmp
rem para (10,10)-(138,138) do bitmap zero

suspend for key 


 O programa pára se o bitmap fonte transferido for maior do que a área total
 do bitmap de destino.
 A parte do bitmap fonte transferida torna-se parte do  bitmap  de  destino.
 A parte fonte é reescalada quando a área de destino é dada numa área maior.
 Esta função de copiar bitmap é muito útil para animação e efeitos gráficos
 uma vez que podemos manipular os bitmaps na memória  antes  de  mostrá-los.
 

 -------------------------------------------------------------------------------
 fade bitmap nBmp, nFade
 Este comando aplica um efeito fade no bitmap especificado.

 nBmp           é o handle identificador do bitmap

 nFade          é a taxa de aplicação do efeito. A faixa é de 0 a 100.

 Ex.:   fade bitmap 1,50 

 * Este comando é bom para simular um anoitecer, ou tirar personagens
 de foco quando eles morrem.


 -------------------------------------------------------------------------------
  blur bitmap nBmp, nBlur
 Aplica um efeito blur ou borrão no bitmap, a faixa de valores para nBlur pode
 variar de 0 a 8.

 ex.:   blur bitmap 0,6 

 * Este efeito é bom para simular a visão de uma personagem recuperando
 da inconsciência, ou visão através de vidro embaçado.

 
 -------------------------------------------------------------------------------
  mirror bitmap nBmp
 Este comando aplica um efeito espelho no bitmap nBmp.
 O espelhamento é horizontal. O que estava na direita aparece
 na esquerda invertido e vice-versa.

 Ex.:   mirror bitmap 1 

 * O primeiro mirror inverte e o segundo faz o bitmap voltar a
 sua forma natural.

 
 -------------------------------------------------------------------------------
  flip bitmap nBmp
 Este comando aplica um espelhamento vertical no bitmap. O que estava
 em baixo aparece em cima invertido e vice-versa.

 * Primeiro flip inverte o bitmap, o segundo flip faz o bitmap
 voltar a sua forma original.

 Para espelhamentos com mais acurácia use mirror seguido de um flip.

 Ex.:   flip bitmap 1 

 -------------------------------------------------------------------------------
  delete bitmap nBmp
 Deleta o bitmap especificado. A deleção de bitmaps ociosos aumenta a
 performance do sistema, aumenta a quantidade de memória disponível.

 Ex.:   delete bitmap 1 


 -------------------------------------------------------------------------------
  create bitmap nBmp, nLargura, nAltura
 Cria um bitmap com o handle nBmp, com a largura e altura especificada.
 Este bitmap criado torna-se o bitmap corrente. Use set current bitmap 0
 para deixar tudo como estava.

 nBmp           é número do bitmap criado, é o seu handle.

 nLargura       é a largura do bitmap (xcol final)

 nAltura        é a altura do bitmap (ylin final)

 Veja nosso programinha exemplo:



rem Testando CREATE BITMAP nBmp, xLargura,yAltura

load bitmap "c:\media\textura.bmp",0

create bitmap 2,256,256

set current bitmap 0

copy bitmap 0,0,0,128,128,2,0,0,255,255

copy bitmap 2,0

print "Ok" : suspend for key 

 O primeiro ponto deste bitmap criado é o ponto (0,0) e o último ponto é
 o (255,255), veja que o ponto final é um a menos em largura e altura
 especificados na criação. Você deve tomar cuidado ao referir-se ao último
 ponto do bitmap para não ultrapassar a área do bitmap e causar erro
 em outras funções.

  set current bitmap nBmp
 Este comando estabelece um bitmap como sendo o corrente, isto é,
 é o bitmap que vai sofrer todas as operações de desenhos, extração
 e colagem.

 Ex.:   set current bitmap 2 

 * Este comando é útil para estabelecer o bitmap vítima,
 o qual vamos submeter a algum processo, e depois mostrá-lo em
 sua nova aparência.
 
 
 2. Obtendo configurações de bitmap 
-------------------------------------------------------------------------------
 nLargura	= bitmap width(nBmp)
 nAltura	= bitmap height(ntBmp)
 nProfundidade 	= bitmap depth(nBmp)
 Estas funções retornam números inteiros contendo,  a  largura  do  bitmap, 
 a altura e a profundidade do bitmap especificado. Profundidade é o número
 de bits que é usado para compor as cores do bitmap. A  profundidade  mais 
 comum atualmente é a de 24 bits, sendo 8 bits para cada componente da cor,
 vermelho (R), verde (G) e azul (B).

 Usadas sem o argumento nBmp, estas funções retornam a informação do bitmap
 corrente.

 Ex.: 

 print "Largura(Width):",bitmap width(1)

 print "Altura(Height): ",bitmap height(1)

 print "Profundidade(Depth): ",bitmap depth(1)

 print bitmap width() 
 

 -------------------------------------------------------------------------------
 nBool = bitmap exist(nBmp)
 nBool = bitmap mirrored(nBmp)
 nBool = bitmap flipped(nBmp)

 Estas funções retornam informações com relação ao estado do bitmap
 nBmp especificado. nBool vai ser um quando o estado verificado for
 verdadeiro, zero quando o estado verificado estiver desligado.

 bitmap exists() retorna 1 caso o bitmap exista, nBmp nesta função é
 obrigatório.
 bitmap mirrored() retorna 1 caso o bitmap esteja espelhado.
 bitmap flipped() retorna 1 caso o bitmap esteja flipado.

 O retorno falso é representado por zero. Caso nBmp não seja
 provido, a verificação é feita no bitmap corrente.

 
 -------------------------------------------------------------------------------
  nBmp = current bitmap()
 Esta função retorna o número do bitmap que está corrente.
 
 
 Veja o programa exemplo desta seção:
 

 
 

 

 
Rem Project: bitmaps
Rem Created: 18/08/2013 14:33:05

inicio:
global branco as integer = 0xffffff
global azul   as integer = 0x0000ff

tecla as string = ""

initsys("bitmaps.dba - Pressione 'q' para sair")

' Mostra menu e bitmap
menu()
carregar_mostrar_bmp()


' Faz leitura do teclado
teclado:
tecla = inkey$()

if tecla = "q" then goto final

' Manipulação do bitmap
if tecla = "b" then borrar_bmp()
if tecla = "c" then tst_copiar_bmp(0,0,127,127)
if tecla = "d" then sumir_bmp()
if tecla = "f" then virar_bmp()
if tecla = "l" then carregar_mostrar_bmp()
if tecla = "m" then espelhar_bmp()

' Mostra informações sobre o bitmap 1
info_bmp()
goto teclado

final:
delete bitmap 1
print
print " Pressione qualquer tecla para encerrar"
suspend for key
end

' ----- initsys() - Configuração básica da aplicação -----
function initsys(txt as string)
' configurando a janela
set window on
set window title txt
ink azul, branco
cls branco
set text opaque
endfunction

' ------ menu() - Apresenta menu da aplicação
function menu()
Set Current Bitmap 0
Text  10, 300, "l- Load Bitmap()":    Text 210, 300, "d- Fade Bitmap()"
Text  10, 320, "m- Mirror Bitmap()":  Text 210, 320, "b- Blur Bitmap()"
Text  10, 340, "f- Flip Bitmap()":    Text 210, 340, "c- Copy Bitmap()"
endfunction

' ------ carregar_mostrar_bmp() - Carrega e mostra um bitmap
function  carregar_mostrar_bmp()
Load Bitmap "c:\gdkmedia\bitmap\textura.bmp",1
Copy Bitmap 1,0
endfunction

' ------ espelhar_bmp() - Testa Mirror Bitmap
function espelhar_bmp()
Mirror Bitmap 1
Copy Bitmap 1,0
endfunction

' ------ virar_bmp() - Testa Flip Bitmap
function virar_bmp()
Flip Bitmap 1
Copy Bitmap 1,0
endfunction

' ------ sumir_bmp() - Testa Fade Bitmap
function sumir_bmp()
Fade Bitmap 1,80
Copy Bitmap 1,0
endfunction

' ------ borrar_bmp() - Testa Blur Bitmap
function borrar_bmp()
Blur Bitmap 1,5
Copy Bitmap 1,0
endfunction


' ------ tst_copiar_bmp() - Testa Copy Bitmap
function tst_copiar_bmp(nEsq as integer, nTop as integer, nDir as integer,
								 nFun as integer)

' Font = bmp fonte  -  Dest = bmp destino
Font   as integer = 1
Dest   as integer = 0

' esqAlvo dirAlvo = ponto esquerdo e direito do destino
' topAlvo funAlvo = ponto superior e inferior do destino
esqAlvo     as integer = 300
topAlvo     as integer = 0
dirAlvo     as integer = 428
funAlvo     as integer = 120
Copy Bitmap Font, nEsq, nTop, nDir,nFun, Dest,esqAlvo, topAlvo, dirAlvo, funAlvo
endfunction


' ------ info_bmp() - Mostra configurações do bitmap
function info_bmp()
txt as string

' Dados de configuração do bitmap
nlargura       as integer
nAltura        as integer
nprofundidade  as integer
nvirado        as integer
nEspelhado     as integer

' Obtendo os dados do bitmap 1
nlargura      = Bitmap Width(1)
nAltura       = Bitmap Height(1)
nprofundidade = Bitmap Depth(1)
nvirado       = Bitmap Flipped(1)
nEspelhado    = Bitmap Mirrored(1)

' Mostrando os dados
Set Current Bitmap 0
txt = " Largura: " + str$(nLargura)
txt = txt + " Altura: " + str$(nAltura)
txt = txt + " Profundidade: " + str$(nprofundidade)
Text  10,260, txt

txt =  " Virado: " + str$(nvirado)
txt = txt + " Espelhado: " + str$(nEspelhado)
Text 10,280, txt
endfunction
