03. Execução condicional 1/2

 if (condição) then (reação)
 A instrução if...then possibilita a execução condicional de uma instrução
 ou de um bloco de instruções  conforme  a  condição  seja  avaliada  como 
 verdadeira. Ex.: 

 input "Digite a sua idade: ", idade

 if idade = 18 then print "Vc tem 18 anos"
 if idade < 18 then print "Vc é menor de idade"
 if idade > 18 then print "Vc é maior de idade" 

 A mensagem que vai ser exibida está condicionada ao resultado da avaliação
 da variável idade.

 operadores relacionais
 Os operadores relacionais são os símbolos de comparação ou igualdade que 
 são usados para estabelecer a condição que deve ser avaliada:
 
 

 
Expressão Comparação 

 
 
idade  = 18                     idade é igual a 18?


 
idade  < 18                   idade é menor que 18?


 
idade  > 18                   idade é maior que 18?


 
idade <> 18                idade é diferente de 18?

                  
 
idade <= 18                   idade é menor ou igual a 18?


 
idade >= 18                   idade é maior ou igual a 18?

 
 


 operadores lógicos
 Os operadores lógicos são símbolos que são usados para concatenar duas ou
 mais condições numa avaliação:

 and	concatena duas ou mais condições, executando o bloco de
	instruções caso uma e outra sejam verdadeiras.

 or	concatena duas ou mais condições, executando o bloco de
	instruções caso uma ou outra seja verdadeira. Ex.: 

 if idade = 18 and salario >= 1000 then print "Empréstimo concedido"

 if idade < 18 or salario < 1000 then print "Empréstimo negado" 

 if (condição) then (reação) else (reação alternativa) 
 Essa sintaxe de uso da instrução if executa o primeiro bloco de código caso
 a condição seja verdadeira, caso contrário (else) o segundo bloco de
 código, a reação alternativa é executada. Ex.: 

 if salario < 1000 then print "Empréstimo negado" else print "Ok" 

 if...else...endif - execução condicional de blocos
 Há uma sintaxe confortável da instrução if que permite a execução condicional
 de blocos. 

 if (condição)
        bloco de código
 else
        bloco alternativo
 endif


 if (condição)
        bloco de código
 endif   


 Veja abaixo um exemplo: 

 if salario > 1000
        print "Empréstimo Ok"
 else
        print "Empréstimo negado"
 endif 

 Perceba que nessa sintaxe de bloco, a instrução then é obrigatoriamente
 dispensada.

 operadores matemáticos
 Os operadores matemáticos são os símbolos que você usa para realizar as
 operações matemáticas básicas.

 +              adição          a = b + c

 -              subtração       a = b - c

 *              multiplicação   a = b * c

 /              divisão         a = b / c

' Configurando a janela
branco as integer = 0xFFFFFF
azul   as integer = 0x0000FF
set window on
set window title "ifteste.dba"
ink azul, branco
cls branco

' Itens do inventário do jogador
espada_qtd     as integer = 0
escudo_qtd     as integer = 0
flecha_qtd     as integer = 0

' Preço dos itens na lojinha do game
espada_custo   as integer = 120
escudo_custo   as integer = 60
flecha_custo   as integer = 8

' Conta (R$) do jogador e monitoramento da escolha
conta_jogador  as integer = 300
escolha        as integer = 0

do
cls branco
print

set cursor 20,10
print " Dinheiro: ", conta_jogador, " Espadas: ", espada_qtd;
print " Escudos: ", escudo_qtd, " Flechas: ", flecha_qtd

set cursor 20,40: print "[1] - Espada - R$ 120,00 "
set cursor 20,60: print "[2] - Escudo - R$ 60,00  "
set cursor 20,80: print "[3] - Flecha - R$ 8,00   "
set cursor 20,100: print "[4] - Sair"

set cursor 20,140
input "Digite sua escolha: ", escolha

if escolha = 1
conta_jogador = conta_jogador - espada_custo
espada_qtd = espada_qtd + 1
endif

if escolha = 2
conta_jogador = conta_jogador - escudo_custo
escudo_qtd = escudo_qtd + 1
endif

if escolha = 3
conta_jogador = conta_jogador - flecha_custo
flecha_qtd = flecha_qtd + 1
endif

if escolha = 4 then end

if escolha < 1 or escolha > 3
print
print " Este item nao existe!"
endif

print
print " Dinheiro: ", conta_jogador

print " Pressione qualquer tecla para continuar "
suspend for key
loop
end
