11. Imagens
                                                                   
 No DarkBasic, uma imagem é um recorte de um bitmap. Esse recorte pode ser                                              
 usado para texturizar um objeto ou terreno, e pode ser usado para dar                                                  
 origem a uma célula de animação.                                                                                       
                                                                                                                        
 GET IMAGE nImg, nTop, nLeft, nBottom, nRight                                                                    
 Este comando captura um recorte do bitmap corrente.                                                                    
                                                                                                                        
 nImg...........representa o identificador da imagem pelo qual ela vai ser                                              
                manipulada. A faixa de valores para nImg varia de 0 a 65535                                             
                                                                                                                        
 nTop, nLeft     coordenada superior esquerda do início do recorte                                                      
 nBotton, nRight coordenada inferior direita do final do recorte.                                                       
                                                                                                                        
 Antes de usar GET IMAGE você precisa carregar o bitmap fonte e definí-lo como                                   
 corrente com SET CURRENT BITMAP nBmp.                                                                           
                                                                                                                        
 PASTE IMAGE nImg, x, y                                                                                          
 Esta função cola a imagem nImg no bitmap corrente. Você precisa definir o                                              
 bitmap corrente com SET CURRENT BITMAP nBmp.                                                                    
                                                                                                                        
 x,y    Representa a posição aonde a imagem vai ser colada.                                                             
                                                                                                                        
Veja nosso programa exemplo:
                           
cls                                                                                                                     
hide mouse                                                                                                              
                                                                                                                        
load bitmap "\media\textura.bmp",1                                                                                      
                                                                                                                        
set current bitmap 1                                                                                                    
get image 1,20,20,110,110                                                                                               
                                                                                                                        
set current bitmap 0                                                                                                    
paste image 1,10,10                                                                                                     
                                                                                                                        
suspend for key 
                                                                                           
                                                                                                                        
 LOAD IMAGE sArquivo$, nImagem                                                                                   
 Este comando carrega um arquivo de imagem e assinala o handle nImagem                                                  
 a ela. Posteriormente essa imagem pode ser usada para texturizar,                                                      
 ilustrar ou servir de base para animação.                                                                              
 A imagem pode ser do tipo JPG, BMP ou TGA.                                                                             
                                                                                                                        
 Ex.:                                                                                           
                                                                                                                        
 LOAD IMAGE "c:\media\textura.jpg",1
