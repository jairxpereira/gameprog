01.1 Definindo variáveis


1. Declaração de variáveis no darkbasic clássico 

 No darkbasic clássico  os tipos de variáveis são basicamente três:
 string, inteiro e real. Segue abaixo exemplos de como você declara
 esses  três tipos de variáveis do jeito antigo  no  darkbasic pro.

 tipo inteiro
 idade =  37
 valor = -1000

 tipo real
 salario# = 816.99
 valor# = -100.85

 string
 nome$ = "Jair Pereira"
 empresa$ = "Studio Waterboys"

 identificadores das variáveis
 As variáveis strings são identificadas com o simbolo $ [nome], enquanto que o
 tipo real é identificado pelo símbolo # [salario#].

  Programa exemplo:



 




 rem programa: var01.dba
 rem Definição básica de variáveis

 set window title "var01.dba"
 ink 0x0000FF, 0xFF0000
 cls 0xFFFFFF

 idade = 36
 salario# = 816.99
 nome$ = "Jair Pereira"

 print
 print " Nome: ", nome$
 print " Idade: ",idade
 print " Salario: ",salario#

 suspend for key

 
 
 set window on
 Esse comando estabelece que o programa vai rodar em modo janela.
  
 set window title "var01.dba"
 Esse comando define o título da janela.
 
 ink 0x0000FF, 0xFF0000
 Esse comando define a cor de frente e a cor de fundo, definindo a cor
 azul (0x0000FF) para a cor de frente.
 
 cls 0xFFFFFF
 Esse comando limpa a tela deixando o fundo branco.
 
 print " Nome: ", nome$
 O comando print mostra uma lista de dados.
 
 2. Declaração de variáveis no darkbasic pro 
 
 Embora o Darkbasic profissional, versão 1.069, aceite o modo antigo de 
 declarar variáveis é mais conveniente usar a forma moderna que vamos
 apresentar abaixo:
 


idade   as integer
salario as float
nome    as string

mychar  as byte
myword  as word
mydword as dword

mylong  as double integer
mypi    as double float



Além dos três tipos básicos apresentados acima  há os seguintes
tipos que somente assumem valores positivos:  byte, word, dword.
Veja abaixo uma tabela que resume os tipos de dados e as faixas
de valores que eles podem assumir.




Tipo de dadoTamanho em bitsFaixa de valores


byte          80 até 255
 

word          160 até 65.535


dword         320 até 4.294.967.295


integer       32-2.147.483.648 até 2.147.483.647


double integer64-9.223.372.036.854.775.808 até 9.223.372.036.854.775.807


float         323.4 +-38(Precisão: 7 digitos depois da vírgula)


double float  641.7 +-308 (Precisão: 15 digitos)




As variáveis podem ter seus valores atribuidos também na declaração:

branco as integer = 0xFFFFFF
azul   as integer = 0x0000FF

Veja mais um programa exemplo:


 

 


 rem programa: var02.dba
 rem Definição básica de variáveis

 set window on
 set window title "var02.dba"
 ink 0x0000FF, 0xFF0000
 cls 0xFFFFFF

 idade   as integer
 salario as float
 nome    as string

 mychar  as byte
 myword  as word
 mydword as dword

 mylong  as double integer
 pi      as float
 mypi    as double float

 idade   = 36
 salario = 816.99
 nome    = "Jair Pereira"

 pi      = 3.14
 mypi    = 3.141516
 mychar  = 65
 myword  = 65535
 mydword = 876000
 mylong  = -876000

 print
 print " Nome:    "; nome
 print " Idade:   "; idade
 print " Salario: "; salario

 print
 print " pi:      "; pi
 print " mypi:    "; mypi
 print " myword:  "; myword
 print " mydword: "; mydword
 print " mylong:  "; mylong
 
 suspend for key
