31. Funções e comandos úteis
                                                  
 Utilização do clipboard ou área de transferência                                                                
 WRITE TO CLIPBOARD texto$ .... Coloca um texto na memória clipboard                                             
 texto$ = GET CLIPBOARD$() .... Recupera um texto da memória clipboard                                           
                                                                                                                        
 Tempo e data                                                                                                    
 horas$ = GET TIME$() ... Obtém a hora do sistema                                                                
 hoje$ = GET DATE$() .... Obtém a data do sistema                                                                
 nTempo = TIMER() ....... Obtém a hora do sistema em milessegundos                                               
                                                                                                                        
 Executando programas externos                                                                                   
 EXECUTE FILE Programa$, Cmd$, Dir$                                                                              
 EXECUTE FILE Programa$, Cmd$, Dir$, nEstado                                                                     
                                                                                                                        
 Este comando executa um programa, cmd$ é a string de comando do programa aonde                                         
 você pode digitar o nome do arquivo para o programa abrir ou um argumento de                                           
 configuração; Dir$ é o diretório de referência para o programa; nEstado é                                              
 opcional, pode ser um ou zero, e representa se o DarkBasic vai esperar o                                               
 fechamento do programa para retornar.                                                                                  
                                                                                                                        
 Ex.:                                                                                           
 execute file "snes.exe","cfg","c:\games"                                                                        
                                                                                                                        
 cmdline$ = CL$()                                                                                         
 Esta função retorna os argumentos da linha de comando. É utilizada quando                                              
 o usuário chama o programa a partir do subsistema MsDos ou quando ele                                                  
 determina a linha de comando alterando as propriedades do atalho ao                                                    
 executável.                                                                                                            
 Esta função é útil porque você pode deixar o seu programa mais flexível para                                           
 configurações, ou o seu programa pode aceitar um arquivo como argumento.                                               
 Há casos em que o texto dessa linha de comando vem delimitado por aspas ("),                                           
 sendo então necessário um tratamento para utilizar os argumentos da linha                                              
 de comando.                                                                                                            
                                                                                                                        
 WAIT nMilessegundos                                                                                      
 SLEEP nMilessegundos                                                                                     
 Estes comandos ocasionam uma espera no programa antes de executar o código                                             
 seguinte. Para um segundo de espera você deve configurar nMilessegundos para 1000.                                     
                                                                                                                        
 Verificando memória disponível                                                                           
 nFreeVideo = SYSTEM DMEM AVAILABLE()                                                                            
 Esta função retorna a quantidade de memória de vídeo disponível.                                                       
                                                                                                                        
 nFree = SYSTEM SMEM AVAILABLE()                                                                                 
 Esta função retorna a quantidade de memória disponível.                                                                
                                                                                                                        
 nTotal = SYSTEM TMEM AVAILABLE()                                                                                
 Esta função retorna o total de memória disponível.
