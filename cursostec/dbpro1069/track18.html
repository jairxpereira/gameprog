18. Objetos 3D básicos 2/2
                                                    
                                                                                                                        
 MAKE OBJECT CONE nObjeto, nRaio                                                                                 
 Este comando cria um cone tendo o tamanho da base dado por nRaio.                                                      
                                                                                                                        
 MAKE OBJECT PLAIN nObjeto,nLargura, nAltura                                                                     
 Este comando cria um plano com a largura e a altura dada por nLargura                                                  
 e nAltura.                                                                                                             
                                                                                                                        
 MAKE OBJECT CYLINDER nObjeto, nTamanho                                                                          
 Este comando cria um cilindro com o tamanho especificado por nTamanho.                                                 
                                                                                                                        
 MAKE OBJECT CUBE nObjeto, nTamanho                                                                              
 Este comando cria um cubo com o tamanho nTamanho.                                                                      
                                                                                                                        
 Animação de objeto                                                                                       
 Você pode desenvolver objetos com animações usando editores de objetos 3d como                                         
 o 3D Studio Max ou o Blender. Naturalmente o DarkBasic apresenta comandos                                              
 para tocar a animação de objetos toda ou partes selecionadas.                                                          
                                                                                                                        
 PLAY OBJECT nObjeto                                                                                             
 PLAY OBJECT nObjeto, nQuadroInicial                                                                             
 PLAY OBJECT nObjeto, nQuadroInicial, nQuadroFinal                                                               
 Este comando toca a animação pré-definida do objeto. Você pode determinar                                              
 o início da animação configurando nQadroInicial e ainda você pode determinar                                    
 até que quadro essa animação vai tocar configurando nQuadroFinal.                                               
                                                                                                                        
 LOOP OBJECT nObjeto                                                                                             
 LOOP OBJECT nObjeto, nQuadroInicial                                                                             
 LOOP OBJECT nObjeto, nQuadroInicial, nQuadroFinal                                                               
 Este comando repete a animação pré-definida do objeto. Você pode determinar                                            
 o início da repetição configurando nQadroInicial e ainda você pode determinar                                   
 até que quadro essa animação vai repetir-se configurando nQuadroFinal.                                          
                                                                                                                        
 STOP OBJECT nObjeto                                                                                             
 Este comando pára de tocar a animação do objeto especificado por nObjeto.                                              
                                                                                                                        
 *** Geralmente uma personagem de videogame vem com várias animações diferentes                                         
 pré-definidas no mesmo arquivo de objeto e conforme o desenvolvimento do                                               
 jogo e as situações enfrentadas toca-se selecionadamente cada animação                                          
 para criar o contexto correto.                                                                                         
                                                                                                                        
 SET OBJECT SPEED nObjeto, nVelocidade                                                                           
 Este comando determina a velocidade de apresentação da animação do objeto                                              
 selecionado.                                                                                                           
                                                                                                                        
 Veja nosso programa exemplo:                                                                                           

                                                       
cls: hide mouse                                                                                                         
                                                                                                                        
load object "c:\dbCurso\porta.3ds",1                                                                                    
scale object 1,200,200,200                                                                                              
color object 1,rgb(255,10,10)                                                                                           
color limb 1,1,rgb(0,255,10)                                                                                            
                                                                                                                        
set object speed 1,50                                                                                                   
play object 1,75,100                                                                                                    
loop object 1,50                                                                                                        
                                                                                                                        
suspend for key                                                                                                         
stop object 1                                                                                                           
suspend for key                                                                                                         
delete object 1
                                                                                            
                                                                                                                        
 Efeitos de objeto                                                                                        
 Há comandos interessantes que mudam o visual e o estado dos objetos. Por exemplo,                                      
 há o efeito fantasma (ghost) que deixa o objeto transparente que pode ser usado                                        
 para indicar a morte da personagem.                                                                                    
                                                                                                                        
 GHOST OBJECT ON nObjeto                                                                                         
 GHOST OBJECT OFF nObjeto                                                                                        
 Estes comandos ligam ou desligam o efeito fantasma do objeto especificado.                                             
                                                                                                                        
 FADE OBJECT nObjeto, nFade                                                                                      
 Este comando aplica um efeito fade ao objeto selecionado.                                                              
                                                                                                                        
 SET OBJECT 1,bWire,bTransp,bCull                                                                                
 Este comando configura as seguintes propriedades do objeto selecionado: modo                                           
 de renderização, wireframe ou texturizado configurado por bWire com valor                                       
 zero ou um; transparência dada por bTransp; e renderização das faces escondidas                                 
 dada por bCull. Os argumentos desse comando somente assumem valor zero ou um                                    
 que significam ligado ou desligado.                                                                                    
                                                                                                                        
 Ex.:                                                                                           
 bWire = 0                                                                                                              
 bTransp = 1                                                                                                            
 bCull = 1                                                                                                              
 set object 1,bWire,bTransp,bCull                                                                                
                                                                                                                        
 SET OBJECT 1,bWire,bTransp,bCull, bFilter, bLight, bFog, bAmbient                                               
 SET OBJECT configura outras propriedades do objeto como sensibilidade a luz,                                    
 neblina, luz ambiente e qualidade da texturização dada por bFilter (textura),                                          
 bLight (luz), bFog (neblina), bAmbient (luz ambiente).                                                                 
                                                                                                                        
 HIDE OBJECT nObjeto                                                                                             
 Este comando esconde o objeto selecionado. A fim de melhorar a velocidade                                              
 do seu jogo, esconda os objetos sempre que possível, por exemplo, enquanto                                             
 uma porta estiver fechada você pode esconder os objetos dentro do recinto                                              
 ainda não adentrado pelo jogador.                                                                                      
                                                                                                                        
 SHOW OBJECT nObjeto                                                                                             
 Este comando mostra o objeto selecionado, escondido previamente.                                                       
                                                                                                                        
 LOCK OBJECT ON nObjeto                                                                                          
 LOCK OBJECT OFF nObjeto                                                                                         
 Estes comandos trancam ou destrancam a visão do objeto na tela, o objeto                                               
 fica permanentemente visível acompanhando sempre a câmera. Lock Object On                                       
 apresenta o problema de reposicionar o objeto tendo que se procurar o                                                  
 objeto.
