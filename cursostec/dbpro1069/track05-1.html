05.1 Repetição controlada

 for...next
 A instrução for...next permite  repetir a execução de um bloco de código
 de maneira determinada, ou ainda permite passar  por  uma mesma variável
 uma faixa de valores. 

 For var = valor_inicial to valor_final step passo
 (...) código
 next var 

 Ex.: 

 for cx = 1 to 10
 print cx
 next cx 

 Outro exemplo, produzindo uma contagem regressiva: 

 for contagem = 10 to 1 step -1
 print contagem
 next contagem 

 A instrução step permite determinar o passo do reajuste da variável. 
 Por exemplo, um  step 2  faz a contagem ocorrer de dois em dois, um 
 step -2 (negativo) faz a contagem regredir.

 Você pode sair do bloco de código a qualquer momento usando a instrução
 exit.
 
-------------------------------------------------------------------------------
 do...loop
 A instrução do...loop executa indefinidamente um bloco de código. 
 A execução é interrompida pela instrução exit.

 Ex.: 

 cx = 1
 
 do
 print cx
 cx = cx + 1
 if cx >= 10 then exit
 loop 

-------------------------------------------------------------------------------
 while...endwhile
 A instrução while...endwhile executa um bloco de código enquanto a condição
 dada é verdadeira. A verificação da condição é feita antes do início da
 execução do código. 

 while condição
 (...) código
 endwhile

 Exemplo: 

 cx = 1
 
 while cx < 10
 print cx
 cx = cx + 1
 endwhile 

 Você pode sair do bloco de código a qualquer momento usando a instrução
 exit.
 
-------------------------------------------------------------------------------
 repeat...until
 Essa instrução permite a execução do bloco até uma condição tornar-se
 verdadeira. O bloco de código  é  executado ao menos uma vez porque a 
 verificação da condição é dada no final do bloco. O bloco de código é 
 executado enquanto a condição é falsa.

 repeat
 (...) código
 until condição 

 Exemplo: 
 
 cx = 1 
 repeat
 print cx
 cx = cx + 1
 until cx > 5 

 Você pode sair do bloco de código a qualquer momento usando a instrução
 exit.
 
 Veja o programa exemplo:
 

  
 

 

 
Rem Project: repetition
Rem Created: 02/08/2013 12:39:51

inicio:
' Configurando a janela
branco as integer = 0xFFFFFF
azul   as integer = 0x0000FF
set window on
set window title "repetition.dba"
ink azul, branco
cls branco

ncx      as integer = 0
escolha  as integer = 0

repeat
 cls branco
 gosub menu

 select escolha

   case 1
   gosub contagem_progressiva
   endcase

   case 2
   gosub contagem_regressiva
   endcase

   case 3
   gosub teste_do_loop
   endcase

   case 4
   gosub teste_while
   endcase

   case 5
   gosub teste_until
   endcase

   case 6
   end
   endcase

   case default
   print " Esta opcao nao existe: ", escolha
   endcase

 endselect

 print
 print " Pressione qualquer tecla para continuar"
 suspend for 
 key

until (escolha = 6)

' Fim do programa
suspend for key
end


menu:
print
print " 1- for...next progressivo"
print " 2- for...next regressivo"
print " 3- do...loop"
print " 4- while...endwhile"
print " 5- repeat...until"
print " 6- Sair"
print
input " Digite sua escolha: ", escolha
return

contagem_progressiva:
print " for ncx = 1 to 5"
for ncx = 1 to 5
print " Contagem: ", ncx
next ncx
return

contagem_regressiva:
print " for ncx = 5 to 1 step -1"
for ncx = 5 to 1 step -1
print " Contagem: ", ncx
next ncx
return

teste_do_loop:
print " if ncx = 10 then exit"
ncx = 0
do
ncx = ncx + 1
print " Contagem: ", ncx
if ncx = 10 then exit
loop
return

teste_while:
print " while ncx <= 7"
ncx = 0
while ncx <= 7
print " Contagem: ", ncx
ncx = ncx + 1
endwhile
return

teste_until:
print " until ncx = 10"
ncx = 0
repeat
print " Contagem: ", ncx
ncx = ncx + 1
until (ncx = 10)
return
