08.3 Pesquisando fontes instaladas no sistema

perform checklist for fonts
Este comando realiza uma pesquisa por fontes instaladas no sistema.

nqtd = checklist quantity()
Este comando retorna a quantidade de itens descobertos pela 
pesquisa, aqui no caso, a quantidade de fontes instaladas.

txt = checklist string$ (nItem)
Este comando retorna o nome do item pesquisado conforme o índice
passado, aqui no caso, a fonte correspondente ao índice.

(*) Dica: 

 Há uma série de funções no DarkBasic que realizam pesquisas (perform(...)):
 - pesquisa de fontes, pesquisa de modos de vídeo, pesquisa de conexões de
 rede etc. Todas estas funções utilizam a mesma interface  de  comunicação, 
 isto é, todas elas vão usar checklist quantity() para retornar a quantidade
 de  itens  encontrados  e  checklist string$() para  retornar  itens  quando 
 houver itens do tipo string.










Rem Project: fontes
Rem Created: 03/08/2013 22:18:06

inicio:
global branco as integer = 0xffffff
global azul   as integer = 0x0000ff

nqtd  as integer = 0
ncx   as integer = 0
txt   as string = ""

initsys("fontes.dba")

' Verificando as fontes instaladas no sistema 
perform checklist for fonts
nqtd = checklist quantity();

print
for ncx = 1 to 6
txt = checklist string$ (ncx)
print space$(4), ncx, ".", txt
next ncx

' Vamos omitir muitas fontes
print
print space$(4), "(...)"
print

' Última fonte instalada
print space$(4), nqtd, ".", checklist string$(nqtd)

print
print " Total de fontes instaladas no sistema: ", nqtd

final:
print
print " Pressione qualquer tecla para encerrar"
suspend for key
end

' ----- initsys() - Configuração básica da aplicação -----
function initsys(txt as string)
' configurando a janela
set window on
set window title txt
ink azul, branco
cls branco
endfunction
