22.01 _doserrno.........Variável inteira que identifica o erro ocorrido num serviço MsDos
22.02 errno.............Variável inteira que identifica tipo de erro ocorrido numa função

22.01 _doserrno
22.02 errno

        _doserrno é uma variável inteira que recebe o valor numérico que identifica
        o último erro ocorrido numa função de serviço do MsDos, em especial as
        funções que manipulam arquivos.

        errno é uma variável inteira que recebe o valor numérico que identifica
        o último erro ocorrido numa função, em especial as funções de entrada
        e saída.

        Significado dos valores numéricos em errno e _doserrno:
        -------------------------------------------------------------------------------------
        EZERO.........0....Error 0........................Nenhum erro ocorreu
        EINVFNC.......1....Invalid function number........Número de função inválida
        ENOFILE.......2....File not found.................Arquivo não encontrado
        ENOPATH.......3....Path not found.................Caminho não encontrado
        ECONTR........7....Memory blocks destroyed........Blocos de memória destruídos
        EINVMEM.......9....Invalid memory block address...Endereço inválido bloco de memória
        EINVENV......10....Invalid environment............Ambiente inválido
        EINVFMT......11....Invalid format.................Formato inválido
        EINVACC......12....Invalid access code............Código de acesso inválido
        EINVDAT......13....Invalid data...................Dado inválido
        EINVDRV......15....Invalid drive specified........Drive especificado inválido
        ECURDIR......16....Attempt to remove CurDir.......Tentativa deleção diretório corrente
        ENOTSAM......17....Not same device................Não é o mesmo dispositivo
        ENMFILE......18....No more files..................Não há mais arquivos

        ENOENT.......2....No such file or directory.......Não há tal arquivo ou diretório
        EMFILE.......4....Too many open files.............Há arquivos abertos demais
        EACCES.......5....Permission denied...............Permissão negada
        EBADF........6....Bad file number.................Número de arquivo inválido
        ENOMEM.......8....Not enough core.................Memória principal insuficiente
        ENODEV......15....No such device..................Não há tal dispositivo
        EINVAL......19....Invalid argument................Argumento inválido
        E2BIG.......20....Arg list too long...............Argumentos demais!
        ENOEXEC.....21....Exec format error...............Erro de formato em EXEC
        EXDEV.......22....Cross-device link...............Ligação cruzada em dispositivo
        EDOM........33....Math argument...................Argumento matemático
        ERANGE......34....Result too large................Resultado grande demais
        EEXIST......35....File already exists.............Arquivo já existe

        EFAULT......-1....Unknown error...................Erro desconhecido
