01. O primeiro programa
Vamos começar nosso curso analisando um programa simples  escrito  na
linguagem C. Observe atentamente o código pois ele  traz os primeiros
aspectos gerais da linguagem que você deve assimilar:


/* Programa: basico.c */
#include "stdio.h"

int main(void)  
{

printf("Programar e' legal!");

return 1;
}










/* Programa: basico.c */
Estes delimitadores /*  e   */  são usados para inserir comentários e
'documentar' o programa.  É muito importante  comentar  e  documentar 
para facilitar a leitura do programa.


#include "stdio.h"
A linguagem C é constituída por um conjunto de bibliotecas que vem em
arquivos separados chamados  arquivos de cabeçalho cuja extensão é .h
Estes arquivos contém a definição das funções que você pode  usar  no
seu programa.  Esta definição traz o nome da função e referência  aos
valores de entrada e de saída da função.

O arquivo stdio.h contém as funções genéricas de entrada e  saída; em
nosso programa basico.c,  a função printf() pertence  a  este arquivo.

Então  incluimos  via  #include  "arquivo.h"  o arquivo cabeçalho que
contém a declaração ou definição das funções que  vamos  utilizar  em 
nosso programa.   Esta  definição  da  função  é chamada tecnicamente
protótipo da função.  O  código  executável destas funções declaradas 
nos arquivos de cabeçalho geralmente está contido nos  arquivos  .lib
quando o compilador é da borland ou da microsoft ou nos  arquivos  .a
em compiladores alternativos.


int main(void)
A função main() é uma função obrigatória em linguagem C; ela assinala
o ponto inicial de execução do programa. No lado esquerdo do nome das
funções deve vir o tipo de dado retornado por esta  função, em  nosso
primeiro caso, a função main() retorna um int ao sistema operacional,
que é um valor entre  0  e  65535.   Dentro  dos  parêntesis  vêm  os
argumentos das funções, ou os tipos de dados  que  representam  estes
argumentos; em nosso caso a função main() não requer nenhum parâmetro
de entrada e isso é indicado através de void.

O código das funções é delimitado pelas chaves { código; } abrindo  e
fechando o bloco de instruções da função.  O final da linha de código
do programa é  obrigatoriamente  assinalado pelo ponto-e-vírgula (;).
A ausência dele provoca  uma mensagem de erro durante  a  compilação.
Esclarecemos que compilação é o processo de transformar  o  programa
fonte escrito em C ou em outra linguagem para a linguagem de máquina
que é a forma executável pelo computador.

Neste documento, o código fonte de exemplo está com o fundo amarelo e
a saída ou o resultado do programa está em azul. Estes códigos fontes 
sempre aparecem completos  de  maneira  que  basta  você  aplicar  os 
processos  selecionar-copiar-colar-salvar  e   compilar  para  vê-los 
executados.


01.01 Melhorando o primeiro programa
Tempo para ver a janela
Dentro do ambiente do  Windows  e  na  forma  como  o  programa  está
elaborado acima, você não consegue o tempo suficiente para visualizar
os resultados, pois o programa roda rapidamente e fecha  de  imediato
a  janela. Veja abaixo duas formas de  adicionar  uma  pausa  no  seu
programa.

getchar(); 
Esta função pára a execução do programa e espera o usuário pressionar
uma  tecla  para  continuar.   A  finalidade  exata  dela consiste em 
capturar uma tecla digitada e estocá-la em uma variável.

system("pause"); 
Esta função  do arquivo stdlib.h,  executa um comando de  console  do
Windows. Neste caso é executado o comando pause.
Para usar a função system() inclua esta linha no seu programa: 
#include "stdlib.h"

Colocando título na janela
system("title basico.c"); 
Foi desta forma  que  nosso  primeiro  programa  teve  a  sua  janela
renomeada.

Limpando a janela
system("cls"); 
CLS é o comando que limpa a janela de console

Colorindo a janela
system("color 1F"); 
O comando  color  muda a cor de frente e de fundo da janela.  São  16
cores   representadas   por   uma   letra   do   seguinte   conjunto:
0123456789ABCDEF.  Esse  conjunto  apresentado  é  conhecido  como  o
conjunto dos caracteres hexadecimais  e eles são  utilizados  em  uma
forma alternativa de representar os números em computação.
