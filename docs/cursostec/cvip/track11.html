11. Estrutura de dados e Tipo Definido pelo usuário 1/2

11.1 Visão geral

A linguagem C  apresenta instruções para criarmos estrutura de dados,
que pode ser imaginada como sendo um Registro ou uma Ficha Cadastral,
reunindo  vários  campos  ou  tipos  de  dados  básicos  formando um 
conjunto.

Por exemplo, uma ficha cadastral de funcionários é  uma  estrutura de
dados que traz campos como nome, endereço, idade e salário que seriam
representados dentro de um programa através  dos  tipos  básicos  int, 
char, strings e floats reunidos formando um só conjunto.


11.2 struct - Criando e usando a estrutura

A  instrução  para  criarmos  estruturas  de  dados  é  struct.
Sua  sintaxe  geral  é  esta:

struct StructNome {
tipo vVar;
tipo vVar;
...
tipo vVar;
} xVar...;

StructNome...Nome da estrutura
tipo.........Tipo da variável int, float, char...
vVar.........Nome da variável
xVar.........xVar é opcional, e consiste na primeira ou nas primeiras
		variáveis declaradas como sendo estruturas StructNome.

Exemplo:

struct Funcionario {
char sNome[40];
char sEndereco[40];
int nIdade;
float nSalario;
} xVendedor;

sNome, sEndereco,  nIdade  e  nSalario  são  elementos  da  estrutura
Funcionario. A declaração de variáveis dentro da estrutura ocorre  da
mesma  maneira  usada  nas  declarações  normais  noutras  partes  do 
programa.

xVendedor é a primeira variável da estrutura  Funcionario.  Caso  ela
não tenha sido criada na declaração da estrutura, você deve  proceder
assim  para  criar  instâncias  das  estruturas:

struct StructNome xVar;
struct Funcionario xVendedor;
struct Funcionario Programador;

Declaramos aqui xVendedor  como  sendo  uma  instância  da  estrutura
Funcionario.

O acesso aos elementos da estrutura,  é  feita  com  um  ponto  entre
o nome da variável instanciada  e  o  nome  do  elemento  componente:

xVar.vVar = nValor;
xVendedor.nSalario = 680.52;

Como você pode depreender  do  exemplo  acima,  a  inicialização  dos 
valores dos elementos é feito tal qual as variáveis comuns, tanto que
a  inicialização  dos  elementos-strings   dentro  da  declaração  da 
estrutura e no decorrer do programa se dá nos  mesmos  moldes  vistos
nos tópicos sobre strings.

strcpy(xVendedor.sNome,"Jair Pereira");

Veja  nosso  programa  exemplo  para  ilustrar  bem  este  tópico:


 




/* Programa: struct.c - track 11*/
/* Ilustra criação e uso de estrutura */
#include "stdio.h"
#include "stdlib.h"
#include "string.h"

void initSys( char *title);
void espera(void);

struct xFuncionario {
char nome[40];
char endereco[40];
int idade;
float salario;
} Vendedor;

int main(void)  {

struct xFuncionario programador;

initSys("title struct.c");

/* Preenche o registro Vendedor    */
strcpy(Vendedor.nome, "Jair Pereira");
strcpy(Vendedor.endereco, "Rua Geral, 386");
Vendedor.idade = 28;
Vendedor.salario = 700.75;

/* Preenche o registro programador */
strcpy(programador.nome, "Zeca Bytes");
strcpy(programador.endereco, "Rua Intel, 8088");
programador.idade = 55;
programador.salario = 1500.75;

/* Mostra o registro programador */
printf("\t Programador \n\n");
printf(" Nome:        %s    \n",programador.nome);
printf(" Endereco:    %s    \n",programador.endereco);
printf(" Idade:       %i    \n",programador.idade);
printf(" Salario:     %.2f  \n",programador.salario);

/* Mostra o registro vendedor */
printf("\n \t Vendedor \n\n");
printf(" Nome:        %s    \n",Vendedor.nome);
printf(" Endereco:    %s    \n",Vendedor.endereco);
printf(" Idade:       %i    \n",Vendedor.idade);
printf(" Salario:     %.2f  \n",Vendedor.salario);

getchar();

return 1;
}

/*************** Função initSys()  */
void initSys( char *title)
{
/* Configura a janela */
system(title);
system("color F1");
printf("\n\n");
}



11.3 Arrays de estruturas

Você pode criar arrays de estruturas como  você  faz  com  os  outros
tipos de dados. A sintaxe é esta,  supondo que a estrutura foi criada
anteriormente:

struct StrutNome[nElementos];
struct Funcionario xProgramadores[10];

O  acesso  aos  dados  será  feito  desta  maneira:

xVar[nIndíce].vVar = nValor;
xProgramadores[1].nSalario = 3500;

Naturalmente você pode criar arrays multidimensionais de estruturas.

11.4 Estruturas dentro de estruturas

Uma estrutura pode conter outra estrutura como parte:


struct StructNome {
tipo vVar;
struct OutraStructNome xExtra;
...
tipo vVar;
} xVar...;

Exemplo, estrutura interna:

struct Endereco {
char sRua[40];
char sBairro[40];
unsigned int nRua;
unsigned int nCep;
};

Exemplo, estrutura principal:

struct Funcionario {
char sNome[40];
struct Endereco xEndereco;
int nIdade;
float nSalario;

} xVendedor;

A estrutura mais interna deve ser declarada antes da  estrutura  mais
externa.

O acesso aos dados ficará assim:
xVar.xExtra.vVar = nValor;
xVendedor.nsalario = 3500;
xVendedor.xEndereco.nRua = 250;
xVendedor.xEndereco.nCep = 14500;
strcpy(xVendedor.xEndereco.sRua, "Rua Geral, 386");
strcpy(xVendedor.xEndereco.sBairro, "Vila do Silício");

A sintaxe fica bastante complexa  e  interessante quando você declara
uma  array  destas  estruturas  compostas:

struct Funcionario xVendedor[100];
xVendedor[2].xEndereco.nCep = 58245;

Você também pode acessar individualmente bytes  de  elementos-strings
das estruturas, com base no exemplo  acima  podemos  proceder  assim:
xVendedor[2].xEndereço.sRua[1] = 'x';

Veja o programa exemplo:


 




/* Programa: struc02.c - track 11*/
/* Ilustra criação e uso de array de estruturas */
#include "stdio.h"
#include "stdlib.h"
#include "string.h"

/* Protótipo initSys() */
void initSys( char *title);

/* Estrutura xEndereco */
struct xEndereco {
char srua[40];
char sbairro[40];
unsigned int nrua;
unsigned int ncep;
};

/* Estrutura xFuncionario */
struct xFuncionario {
char nome[40];
struct xEndereco endereco;
int idade;
float salario;
} Vendedor;

/* ********************** main() ***************** */
int main(void)  {

struct xFuncionario programadores[10];
int ncx = 0;

initSys("title struc02.c");

/* Preenche programadores[0] */
strcpy(programadores[0].nome, "Zeca Bytes");
strcpy(programadores[0].endereco.srua, "Rua Intel");
strcpy(programadores[0].endereco.sbairro, "Vila do Silicio");
programadores[0].endereco.nrua = 8088;
programadores[0].idade = 55;
programadores[0].salario = 1500.75;

/* Preenche programadores[1] */
strcpy(programadores[1].nome, "Zeca Bit");
strcpy(programadores[1].endereco.srua, "Rua Intel");
strcpy(programadores[1].endereco.sbairro, "Vale do Silicio");
programadores[1].endereco.nrua = 8086;
programadores[1].endereco.srua[0] = 'X';
programadores[1].idade = 16;
programadores[1].salario = 1500.75;

/* Mostra programadores[ncx] */
for (ncx = 0; ncx < 2; ncx++) {
printf("\n \t Programador:\n");
printf(" Nome:     %s \n",programadores[ncx].nome);
printf(" Endereco: %s, ",programadores[ncx].endereco.srua);
printf(" nro:      %u \n",programadores[ncx].endereco.nrua);
printf(" Idade:    %i \n",programadores[ncx].idade);
printf(" Salario:  %.2f \n",programadores[ncx].salario);}

getchar();
return 1;
}

/*************** Função initSys()  */
void initSys( char *title)
{
/* Configura a janela */
system(title);
system("color F1");
printf("\n\n");
}
