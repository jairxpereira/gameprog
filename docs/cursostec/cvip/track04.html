04. Funções básicas de entrada e saída

Vamos ver aqui a sintaxe de algumas funções de entrada  e  saída  que
vamos usar no decorrer do curso.

Funções de entrada
São denominadas funções de entrada as  funções  que  obtém  dados  do
teclado, de arquivos em discos e outros periféricos.

Funções de saída
As funções de saída são	aquelas que  exibem  dados  no  monitor,  que 
gravam em  arquivos  e  que  enviam  dados  para  outros  periféricos.

Você já conhece a função printf() que  exibe  dados  na  tela.  Vamos
mostrar inicialmente a sintaxe das funções tal como elas aparecem nos
arquivos de cabeçalho e na sequência vamos mostrar uma  sintaxe  mais
fácil de ser assimilada. Veja abaixo a  sintaxe  e  a  explicação  da
função printf():

4.1 printf()
Esta função exibe qualquer tipo de dado na tela dentro da  formatação
determinada e retorna o número inteiro que representa a quantidade de
caracteres impressos. 


int printf(const *format,...);
int printf("%Formatos",variáveis);

Ex.:
char myname[]="Jair Pereira";
int myage=33;
float pi=3.1415;
printf("%s %d %f \n",myname,myage,pi);

4.2 getchar()
Esta função captura um caracter do teclado.

int getchar(void)
cChar = getchar();

Ex.: mychar=getchar();

Detalhes:
getchar() aguarda o pressionamento da tecla enter.

4.3 scanf()
Esta função captura dados  formatados  do  teclado  e  estoca-os  nas
devidas variáveis.  Ela  retorna  o  número  de  campos  preenchidos,
isto é, o número de entradas solicitadas.

int scanf(const char *format,...);
int scanf("%formatos",variáveis);

Ex.:
int myage;
char myname[31];
scanf("%s",myname);
scanf("%d",&myage);
scanf("%s %d",myname,&myage);

Detalhes:
O espaço chr(32)  atua como delimitador entre  os dados. As variáveis
numéricas  devem  ser  precedidas  pelo  provedor  de  endereço  &.


Ressaltando um detalhe importante,  o  operador  &  indica  que  está
sendo passado o endereço de memória da variável  para  a  função.  No
caso de strings,  como no exemplo acima  myname,  este operador não é
necessário pois operações com strings sempre lidam diretamente com os
endereços das mesmas.

Um fato importante sobre strings é que quando você for  capturar  uma
string do teclado, por exemplo um nome, você deve reservar  o  espaço 
para guardar este nome. Cada letra de uma palavra ocupa um byte sendo
que você deve reservar este espaço necessário  com um byte a mais que
é usado pelo sistema para colocar o sinal de fim de string  que  é  o
byte de valor zero.

char sNome[31];

A linha acima reserva espaço para um nome de 30 letras, mais o espaço
de um byte que será usado pelo byte de final de string.

4.4 clrscr()

void clrscr(void);
clrscr();

Esta função limpa a tela.
Ex.: clrscr();



Esta função, que pertence ao  arquivo  conio.h,  somente  funciona  em
compiladores da Borland. No Dev-cpp utilize system("cls"); para limpar
a tela.



Veja agora o programa exemplo dessa seção:


/* Programa entrsaida.c - track04 */
/* Teste de funções de entrada e saida */
#include "stdio.h"
#include "stdlib.h"

int main() {

int myage = 26;
char myname[31] = "Megaman";
float pi = 3.141516f;

/* Configura a janela */
system("title entrsaida.c");
system("color F1");

printf("\n\n");
printf(" %s %i %f \n\n", myname, myage, pi);

printf("\n Digite seu primeiro nome: ");
scanf("%s", myname);

printf(" Digite sua idade: ");
scanf("%i", &myage);
printf( " %s %i anos\n\n", myname, myage);

printf("\n Digite seu primeiro nome e idade separados por um espaco:\n ");
scanf("%s %d",myname,&myage);
printf( " %s %i anos\n\n", myname, myage);

system("pause");

}
