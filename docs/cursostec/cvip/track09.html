09. Operadores matemáticos, unários e lógicos

9.1 Operadores matemáticos
Os operadores matemáticos são os sinais que  você  usa  para  indicar
uma operação matemática normal:




OperadorOperação

 +  adição


 -  subtração


 *  multiplicação


 /  divisão


 %  Modulus (Resto da Divisão)




O operador modulus % retorna o resto de  uma  divisão,  por  exemplo:

int x = 5 ,y = 2,z;
z = x % y;

A variável z recebe o  valor  1   que  é  o  resto  da  divisão  5/2.

if (nIdade % 2 == 0) printf ("Número par encontrado\n");
A expressão (nIdade % 2)  retorna o resto da divisão de nIdade por 2;
se cair um número par, por exemplo, 26, que dividindo  por  2  dá  13
e o resto é zero, então a avaliação se define em verdadeira mostrando
nossa mensagem.


9.2 Operadores unários
Os operadores unários são os sinais tal como ++ e -- que você viu nos
programas  exemplos.   Eles   são  usados  para  encurtar  expressões
matemáticas rotineiras.  Veja  a  tabela  abaixo  com  as  expressões
matemáticas  equivalentes.  Vamos  usar  int ncx;  como  exemplo:




OperadorExemploExpressão equivalente

 -  ncx = -ncx; Inverte sinal de ncx


 ++  ncx++; ncx = ncx + 1;


 --  ncx--; ncx = ncx - 1;


 +=  ncx += 2; ncx = ncx + 2;


 -=  ncx -= 2; ncx = ncx - 2;



 *=  ncx *= 2; ncx = ncx * 2;


 /=  ncx /= 2; ncx = ncx / 2;




9.3 Operadores lógicos
Os operadores lógicos são usados principalmente  em  conjunto  com  a
instrução if. Eles são usados para combinar condições  ou  expressões
lógicas.




OperadorOperação

 &&  AND ou E lógico


 ||  OR ou OU lógico


 !  NOT ou inversor lógico




Exemplos:
if (nIdade > 12 && nPagamento > 0) liberar();
Isto significa,  se nIdade > 12  E  nPagamento maior que zero,  então
execute   liberar().    Neste  caso  se  ambas  as  condições   forem
verdadeiras, o resultado geral é verdadeiro.

if (nIdade <= 12 || nPagamento == 0) negar();
Isto significa, se nIdade <= 12 OU nPagamento  igual  a  zero, então
execute negar(). Aqui se UMA OU OUTRA condição for verdadeira, então
o resultado é verdadeiro.


if (!(nIdade == 18)) printf ("Vc não tem 18 anos!\n");
O operador Not inverte o resultado da avaliação,  se nIdade for maior
ou menor que 18 que é um resultado falso da avaliação  mais  interna,
este falso é invertido  para verdadeiro  ocasionando  a  exibição  da
mensagem.


Veja o programa exemplo dos operadores:


 





/* Programa: operats.c - track09  */
/* Operadores lógicos, unários e matemáticos */
#include "stdio.h"
#include "stdlib.h"

void liberar(void);
void negado(void);
void initSys(void);
void espera(void);
void mostra_saldo(char* msg, int nValor);

int main(void)  {

int nIdade = 15, nCarteira = 10;
int sim = 1, nao = 0;

int pgto_efetuado = nao;
int nIngressoValor = 5;

initSys();

printf(" Filme: O senhor dos aneis - Censura: 12 anos\n");
printf(" Valor do ingresso: R$ 5,00 \n\n");

printf(" Sua idade: %i \n", nIdade);
printf(" Pagamento: %i\n", nCarteira);

if (nCarteira < nIngressoValor)
 printf(" Vc nao tem saldo para comprar o ingresso. \n");

if (nCarteira >= nIngressoValor && nIdade > 12)
{
nCarteira -= nIngressoValor;  /* nCarteira = nCarteira - nIngressoValor; */
pgto_efetuado = sim;
}


if (nIdade > 12 && pgto_efetuado == sim) liberar();
if (nIdade <= 12 || pgto_efetuado == nao) negado();

if (nCarteira > 0 && nIdade > 12)
 printf(" Seu troco: %i \n\n", nCarteira);

nCarteira += 3; mostra_saldo("+= 3", nCarteira);
nCarteira *= 2; mostra_saldo("*= 2", nCarteira);
nCarteira++; mostra_saldo("++  ", nCarteira);
nCarteira /= 3; mostra_saldo("/= 3", nCarteira);
nCarteira %= 2; mostra_saldo("%= 2", nCarteira);

espera();
return 1;
}

/* ******************** funções ************************** */
/*************** Função liberar()  */
void liberar(void) {
printf(" Sessao de cinema liberada!\n");
}

/*************** Função negado()  */
void negado(void) {
printf(" Sessao apenas para maiores de 12 anos \n");
}


/*************** Função initSys()  */
void initSys(void)
{
/* Configura a janela */
system("title operats.c");
system("color F1");
printf("\n\n");
}

/*************** Função espera()  */
void espera(void)
{
printf("\n\n");
system("pause");
}

void mostra_saldo(char *msg, int nvalor)
{
printf(" nCarteira %s \t Seu saldo:  R$ %i,00 \n", msg, nvalor);
}
