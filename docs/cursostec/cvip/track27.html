27. Palavras finais

Neste  curso  nós  cobrimos  toda  a  sintaxe    de   programação  da
linguagem    C    de    maneira      profunda,   fazendo  questão  de
mostrar    detalhes    que  são  omitidos  noutros  cursos por  serem
simples  demais  ou  por    serem  complexos  demais  como o caso dos
ponteiros  para  funções,  manipulação  de  arquivos    binários    e
outras temáticas.

Dentro  dos  temas  da  linguagem  C  o  curso está completo, porém a
linguagem  C  tem  muito  mais  a  oferecer do que foi  exibido aqui.

O  domínio  absoluto  desta  linguagem  e    a  capacidade  de  total
manipulação  do  hardware  do  computador  exige  que   o programador
aprenda  a  linguagem  Assembly    de    programação  e  que  adquira
um  conhecimento  muito  íntimo  do  processador principal e detalhes
técnicos  de  hardware.  É  um  nível  de programação muito impopular
e  mais  voltado  para  quem  desenvolve  drivers de  periféricos  ou
que  queira  a  máxima  performance    dos periféricos.  Como exemplo
destes  recursos  podemos  citar  a  manipulação  direta da placa  de
vídeo,  acesso  direto  à  memória  da  placa de  vídeo;  programação
da placa de som, etc.

Este  curso  encadeia-se  com  o  curso  C++  Total, que deve ser seu
próximo alvo. Siga adiante.
