23. Argumentos da função main()

23.1 int main(void) - Argumento de retorno

Como  toda  função,  a  função main() pode retornar valores.  O valor
retornado  pela  função  main()  se  escoa para o sistema operacional
ou    para    o    programa    que    chamou    o    seu    programa.

É costume retornar um número inteiro, geralmente 1,  para dizer que o
processo foi executado com sucesso e,  em caso contrário,  zero ou -1
para problemas.

23.2 int main (int nCmd, char **sCmd) - Argumentos de entrada



Ao iniciar o programa,  o sistema operacional chama a função main() e
lhe  passa  como  argumento  a  linha  de  comando  dividida  em dois
argumentos,  que são o número de elementos da linha de comando e  os
elementos   dessa   linha   organizados   numa  array  de  ponteiros.

Para  capturar  estes  argumentos, basta você declarar nos argumentos
de entrada da função main() um variável inteira e o segundo argumento
como   uma   array   de   ponteiros   na  forma  *sCmd[]  ou  **sCmd.

Exemplos:
int main (int argc,char *argv[]);

int main (int nCmd, char **sCmd);

nCmd  vai  indicar  o  tamanho  da array *sCmd[]. O primeiro elemento
da  array  sCmd[0]  traz  o  caminho e o nome do programa e os outros
argumentos   trazem   o   restante  da  linha  de  comando.  Exemplo:



 




/* Programa: argmain.c - track 23       */
/* Argumentos da função main()          */
#include "stdio.h"
#include "stdlib.h"
#include "string.h"

/* Protótipos */
void initSys( char *title);

int main (int argc,char *argv[])

{

int ncx;
initSys("title argmain.c");

for (ncx = 0; ncx < argc; ncx++)
{
printf(" argv[%d]\n %s\n\n",ncx, argv[ncx]);
if (!strcmp(argv[ncx], "\\h")) printf("\t\t Hello!\n");
}

getchar();
return 1;
}

/* ------------------ Função initSys()  -------------------- */
void initSys( char *title)
{
/* Configura a janela */
system(title);
system("color F1");
printf("\n\n");
}
