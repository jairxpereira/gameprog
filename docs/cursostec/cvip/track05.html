05. Execução condicional e controle de fluxo 1/2

É extremamente rotineiro em programação a  ocasião  em  que  temos  a
necessidade de avaliar uma situação, e conforme for o resultado desta,
tomarmos rumos diferentes para cada resultado obtido.

A execução de um bloco de código pode ficar condicionada ao resultado
da avaliação de uma situação, que geralmente é a verificação do valor
de uma variável para se determinar se a mesma é maior, menor ou igual
a certo valor estabelecido como limite. Como exemplo concreto podemos
citar a avaliação de uma variável inteira que representa a  idade  de
uma pessoa e conforme for o valor da mesma habilitar  ou  desabilitar
funções dependendo se a maioridade do indivíduo foi atingida  ou  não.

Na linguagem C fazemos estas  avaliações  usando  as  instruções  if,
switch e o operador ternário "?:" para alterarmos o fluxo de execução
do programa.

 5.1 if 
Observe atentamente o código fonte do nosso programa:


/* Programa: iftst01.c - track05 */
/* Este programa testa a instrução if */
#include "stdio.h"
#include "stdlib.h"

int main(void)  {

int nIdade;
char sMsg[] = " Digite sua idade: ";
char sMenor[] = " Vc e' menor de idade!";
char sIgual[] = " Vc tem 18 anos!";
char sMaior[] = " Vc e' maior de idade!";
char sCarta[] = " Vc ja' pode tirar sua carta de motorista!";

/* Configura a janela */
system("title iftst01.c");
system("color F1");

printf("\n%s ",sMsg);
scanf("%i",&nIdade);

if (nIdade == 18)  printf("%s \n",sIgual);
if (nIdade <= 18)  printf(" idade <= 18 \n");
if (nIdade >= 18)  printf(" idade >= 18 \n");
if (!(nIdade == 18)) printf (" !(18) Vc nao tem 18 anos!\n");

if (nIdade < 18)
printf("%s \n",sMenor);

/* {} Um bloco de código if */
if (nIdade >= 18)
{
printf("%s \n",sMaior);
printf("%s \n",sCarta);
}

printf("\n\n");
system("pause");

return 1;
}









A instrução if é rica em sintaxes,  em  nosso  programa  introduzimos
três  modos  de  usá-la  que  de  maneira  esquemática  são:
	

 if (situação) código;
if (nIdade == 18) printf("Vc tem 18 anos! \n",sIgual);

Nesta primeira sintaxe, o if é seguido  pela  situação  ou  expressão
que   vai  ser  avaliada  entre  parêntesis,   por  sua  vez  seguida
imediamente pelo código  que  será  executado  caso  o  resultado  da 
avaliação seja verdadeiro.




 if (situação)
código;
if (nIdade < 18)
printf("Vc é menor de idade! \n",sMenor);

Nesta segunda sintaxe, o if é  seguido  pela  situação  ou  expressão
que vai ser avaliada entre parêntesis, por sua vez seguida  na  linha
seguinte pelo código que será executado caso o resultado da avaliação
seja verdadeiro.





 if (situação)
{
bloco de código;
...;
}
if (nIdade > 18)
{
printf("Vc é maior de idade! \n",sMaior);
printf("Vc já  pode tirar sua carta de motorista! \n",sCarta);
}

Nesta terceira sintaxe,  depois da avaliação da situação,  você  pode
executar  um  bloco  de  código  entre  as  chaves.



Em nossos exemplos, o que nós chamamos  de  situação  é  tecnicamente
denominada  expressão lógica  que será avaliada;  o  resultado  desta
avaliação se define em falso ou verdadeiro. Na linguagem C,  o  valor
falso é representado por zero e  o  verdadeiro  é  representado  por
qualquer número positivo maior que zero.

Alguns  compiladores   e  algumas  funções  usam  valores  negativos,
principalmente  o  -1  para  representar  o  falso.

Nos processos internos, a avaliação de uma  expressão  lógica  se  dá
através de uma  subtração  que  através  de  seu  resultado  pode  se
estabelecer que um valor é maior, menor ou igual ao valor  comparado.
A subtração de números iguais resulta em zero revelando a  igualdade,
e para diferenças vamos ter valores  maiores  ou  menores  que  zero.

Numa expressão lógica simples vai constar um operador  relacional  de
igualdade ou diferença entre os  dois  valores  comparados.  Veja  na
tabela abaixo os operadores relacionais que você pode usar nos if's e
outras instruções da linguagem C:




Operador RelacionalSignificado 


 == igualdade 


 > maior que   


 < menor que  


 >= igual ou maior que  


 <= igual ou menor que   


 !=  diferente    


 !  negação. Inverte resultado da avaliação.    




 5.2 if...else  

Como sempre, observe bem atentamente o programa exemplo:


/* Programa: iftst02.c - track05 */
/* Teste da instrução else       */
#include "stdio.h"
#include "stdlib.h"

int main(void)  {

int nIdade;
char sMsg[] = " Digite sua idade: ";

/* Configura a janela */
system("title iftst02.c");
system("color F1");
printf("\n\n");

/* Ler valor do teclado */
printf("%s",sMsg);
scanf("%i",&nIdade);

if (nIdade == 18)
printf(" Vc tem 18 anos!\n\n");
else
printf(" idade <> 18! \n\n");


/* Bloco de código if\else */
if (nIdade == 18)
{
printf(" Vc pode tirar sua carta de motorista!\n");
printf(" Vc tem 18 anos!");
}

else
{
printf(" idade <> 18 \n");
printf(" Vc nao tem 18 anos! \n");
}

printf("\n\n");
system("pause");

return 1;
}












Em nosso programa exemplo apresentamos mais duas sintaxes de  uso  da
instrução if.

A instrução else é usada em  conjunto  com  if  e  seu  significado e
efeito quer dizer "caso contrário". O código ou o bloco  de código da
instrução else é executado quando a avaliação da expressão lógica  no
if() resulta em falso.

Veja abaixo o esquema de uso do conjunto if...else que você pode usar
nos seus programas:




if (expressão)
		código;
else
		código;

if (nIdade == 18)
		printf("Vc tem 18 anos!\n\n");
else
		printf("Talvez maior ou menor...\n\n");

Nesta sintaxe, o código deve vir imediatamente  abaixo  da  instrução
if, em seguida vem a instrução else e na linha imediatamente  abaixo,
o código da instrução else que será executado se a expressão contida
no if resultar em falso.





if (expressão)
 {
		bloco de código;
		...;
 }

else
 {
		bloco de código;
		...;
 }

if (nIdade == 18) {
		printf("Vc pode tirar sua carta de motorista!\n");
		printf("Vc tem 18 anos!");}

else    {
		printf("Talvez maior ou menor...\n");
		printf("Vc não tem 18 anos!\n");}

Nesta sintaxe o código de bloco abaixo do if e  do  else  deve  estar
devidamente delimitados pelas chaves.



 5.3 if...else if()

Agora vamos apresentar a última sintaxe do if, através dela você pode
encadear uma série de avaliações. Veja o programa  exemplo  primeiro:


/* Programa: iftst03.c - track05 */
/* Teste da instrução else if    */
#include "stdio.h"
#include "stdlib.h"

int main(void)  {

int nIdade;
char sMsg[] = " Digite sua idade: ";

/* Configura a janela */
system("title iftst03.c");
system("color F1");
printf("\n\n");

/* Lê dado do teclado */
printf("%s",sMsg);
scanf("%i",&nIdade);

if (nIdade == 18) printf(" Vc tem 18 anos!");
else if (nIdade < 18) printf (" Vc e' menor de idade!\n");
else if (nIdade > 18)
{
printf(" Vc e' maior de idade!\n");
printf(" Vc pode tirar sua carta de motorista!\n");
}
else if (nIdade > 65) printf (" Vc e' da terceira idade!");

printf("\n\n");
system("pause");
return 1;
}













if (nIdade == 18) printf("Vc tem 18 anos!");
else if (nIdade < 18) printf ("Vc é menor de idade!\n");
else if (nIdade > 65) printf ("Vc é da terceira idade!");

else if (nIdade > 18) {
printf("Vc é maior de idade!\n");
printf("Vc pode tirar sua carta de motorista!\n");
}

Perceba que depois de if(expressão),  else  if(expressão)  você  pode
digitar uma linha de código ou um  bloco  de  código  delimitado  por
chaves  como  visto  anteriormente  nas  outras  sintaxes  do  if.




Esse aninhamento de else if's pode ser substituído por uma construção
usando  a  instrução  switch  que  vamos  ver  no  próximo  capítulo.
Com  switch  o  código  fica  com  melhor  estética,  mais legível, e
logicamente melhor organizado.
