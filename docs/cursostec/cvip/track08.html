08. Funções de usuário e escopo de variável

8.1 Funções de usuário
Funções de usuário são as funções  que  você  mesmo  escreve  no  seu
programa.

Protótipo
Para inserir uma função  no  seu  programa  primeiramente  você  deve
declarar  o protótipo da função no topo do programa antes  da  função
main() que consiste no nome da função acompanhado  das  determinações
dos tipos de dados de entrada e saída.
Ex.: int Multiplica (int x, int y);

Implementação
Depois, o código  da  função  pode  aparecer  em  qualquer  lugar  do
programa; este segundo passo é  chamado  de  implementação enquanto o
primeiro chama-se declaração. Lembramos  que o código da função  deve
estar delimitado pelas chaves {}  que indicam respectivamente  o  seu
início e fim.

No primeiro passo,  na declaração do protótipo,  a  sintaxe  geral  é
esta:




TipoReTorno Função(ArgTipo Argumento, ArgTipo Argumento...);
Ex.: int Multiplica (int x, int y);

TipoRetorno
representa o tipo de dado que a função retorna,  que  geralmente  vai
ser  int,  long  e  outros  tipos  que  podem  ser  definidos.

Função(...)
representa  um  nome  qualquer  que  você  pode  dar  a  sua  função.
A nomeação de funções segue os critérios de  nomeação  de  variáveis.

ArgTipo
representa o tipo de dado dos argumentos de  entrada  da  função  que
geralmente vai ser colocado dessa forma: int x, long y e outros tipos
que podem ser definidos.

O tipo de retorno é o que chamamos de valor de saída  da  função;  os
argumentos, também conhecidos como  parâmetros,  são  os  valores  de
entrada.



Posteriormente em qualquer lugar do programa,  você  deve  colocar  o
código de implementação da função desta maneira:



 TipoReTorno Função(ArgTipo Argumento, ArgTipo Argumento...)
 {
 bloco de código;
 return TipoRetorno;
 }

 Ex.:
 int Multiplica (int x, int y) {
 int nValor;
 nValor = x * y;
 return nValor;
 }



void: função sem tipo de retorno ou sem argumentos de entrada
Quando  a  função  não  retornar  valores,  ou  quando  não  requerer
argumentos, use a instrução void para assinalar  esta  condição.  Ex:
void clrscr(void);
int ShowInfo(void);

Lembramos que a instrução return  indica o valor a ser retornado pela
função.

O protótipo da função agiliza o trabalho do  compilador  ao  informar
previamente os valores e entrada e saída da função com a qual ele vai
se deparar posteriormente. O protótipo pode ser omitido desde  que  a
declaração e a implementação da função seja  colocada  no  ínicio  do
programa. Recomendamos o uso do protótipo em qualquer caso, pois isto
facilita a leitura e o re-uso do código.

8.2 Escopo da variável

O escopo se refere à acessibilidade da variável dentro  do  programa,
é um conceito que divide  as  variáveis  em  dois  grupos,  a  saber,
variáveis públicas ou locais.

As variáveis públicas são aquelas cujos  valores  são  acessíveis  em
qualquer  função  do  programa,  de  qualquer  parte  do  programa. A
variável  pública é criada quando você realiza sua declaração no topo
do programa, antes da função main().

A variável é local quando você a declara dentro da função. Neste caso
o valor da variável somente poderá ser visto dentro da  função  aonde
foi declarada.

As variáveis locais são criadas no ínicio da execução de  uma  função
e são destruídas no final do seu processo.  Isto ocasiona economia de
memória. Por outro lado, as variáveis públicas são criadas no  início
do programa e  são  mantidas  até  o  final  de  execução  do  mesmo,
ocasionando menos memória ao sistema.

Veja o programa exemplo dessa seção:


 





/* Programa: loop02v2.c - track08 */
/* Funções de usuário e escopo de variável */
#include "stdio.h"
#include "stdlib.h"

/* Variável global g_limite */
int g_limite;

/* Protótipos */
int teste(void);
void initSys(void);
void espera(void);

/* Função main() */
int main(void)  {

int ncx = 0;

initSys();

printf(" ncx - Var. de controle \n ----------------------- \n");

for (ncx=0; teste(); ncx++)
 {
   printf (" ncx:%i  g_limite: %i\n", ncx, g_limite);
 }

espera();
return 1;
}

/**************** Função teste()  */
int teste(void)
{
// Esta Variável ncx é local
int ncx = g_limite;

g_limite = g_limite - 1;

return ncx;

}

/*************** Função initSys()  */
void initSys(void)
{
/* Configura a janela */
system("title loop02.c - Ver 2.0");
system("color F1");
printf("\n\n");
g_limite = 6;
}

/*************** Função espera()  */
void espera(void)
{
printf("\n\n");
system("pause");
}
