02. Tipos de dados e declaração de variáveis

Introdução
Inicialmente podemos visualizar dois tipos básicos de dados, que são:

* numéricos
que são usados em cálculos.

* alfanuméricos
São dados que podem ser compostos por qualquer combinação  de  letras
do alfabeto, números e outros símbolos. São usados, por exemplo, para
a manipulação de textos.

O tipo de dado numérico se divide em subgrupos de acordo com  a faixa
de valores  que estes números atingem,  conforme  eles  são negativos
ou positivos e conforme eles são números inteiros ou decimais.

Variáveis
O que é conhecido como variável é um local de memória usado  para  se
guardar temporariamente  um tipo de dado para submetê-lo a cálculos e
processos posteriores.

O nome da variável pode ter  até  32  letras  e  incluir  o  carácter
underline '_' como parte de seu nome. O nome da variável deve começar
por uma letra ou underline e não  pode  ter  o  mesmo  nome  que  uma 
instrução ou uma palavra chave da linguagem C.

A linguagem C é case sensitive, isto quer dizer que letras minúsculas
e suas contrapartes maiúsculas são tratadas  como  diferentes.  Assim
a variável com  o  nome  de  sNome  é  diferente  da  variável  snome.

Os nomes das funções padrões e palavras-chaves  da  linguagem  C  são
todas em letras minúsculas. É muito comum ocorrer erros  no  programa
devido a erros de soletração no nome  das  funções ou  das variáveis,
fique atento quanto a este aspecto de  case sensitive  da  linguagem.

2.1 Declarando números inteiros
Observe o exemplo de como declarar números inteiros:

int nIdade = 33;
int salario = 5000;

A instrução int é usada para declarar números inteiros.
A faixa de valores desse tipo é de -2.147.483.648  até  2.147.483.647

As variáveis do tipo de dado numérico  inteiro  são  precedidas  pela
instrução unsigned quando elas  não  vão  assumir  valores  negativos
durante o programa. Quando for este o caso, você  deve  declarar  sua
variável desta maneira:

unsigned int nFilhos = 6;
unsigned int nVendas = 5000;

O unsigned int  assume  os  valores  na  faixa  de  0 a 4.294.967.295

Quando a variável inteira vai assumir  valores  negativos  você  deve
declarar sua variável assim:

signed int nSaldo = -5000;
signed int nValor = 8000;

As variáveis com valores exclusivamente  positivos  são  chamadas  de
variável inteira não-sinalizada e sua declaração deve  ser  precedida
obrigatoriamente pela instrução unsigned.

As variáveis que assumem valores tanto positivos como  negativos  são
chamadas variável inteira sinalizada, e sua  declaração  é  precedida
pela instrução signed.

Alguns compiladores, como o que acompanha o Turbo C 3.3  da  Borland,
quando  a condição de unsigned ou signed da variável não é declarada,
o compilador assume como default  que  a  variável  é signed, isto é,
sinalizada para comportar valores negativos.
Desta forma int nVendas = 50 é equivalente a signed int nVendas = 50.
A fim de manter a clareza  e  a  acurácia  do  programa  é  prudente
declarar  o  tipo  int  como  signed  ou  unsigned  conforme  for  a 
necessidade da situação.

2.2 Declarando números inteiros longos

unsigned long int nVendas = 66000;
unsigned long nSalario = 88000;

A instrução unsigned long int é usada para declarar números inteiros
longos, cuja faixa de valores vai de 0 a 4.294.967.295
Você pode omitir int quando estiver declarando um long.

2.3 Declarando números longos sinalizados


signed long int nVendas = 66000;
signed long int Saldo = -88000;
long nKilometros = 575000

A instrução signed long int é usada para declarar números longos
sinalizados que tem essa faixa de valores:
				de -2.147.483.648 à 2.147.483.647


2.4 Declarando números decimais
Os números decimais ou fracionários são denominados números reais  ou
números de ponto flutuante.  Eles são declarados usando as instruções
float e double da linguagem C.

float  PI       = 3.141516f;
float  NegPi    = -3.141516f
double Saldo    = 88000.589;
double NegSaldo = -88000.589;

Tanto o float como o double assumem valores positivos ou negativos.
A faixa de valores do float vai de -3.2 * 10±38 a 3.2 * 10±38.
A faixa de valores do double vai de -1.7 * 10±308 a 1.7 * 10±308.



float PI = 3.141516f;
Alguns compiladores exigem que o tipo float seja declarado com um f
para diferenciá-lo do double. 


2.5 Declarando caracteres
O tipo char é usado para declarar letras, símbolos,  nomes e palavras
em geral.  Os  nomes  e  palavras  em  geral são denominados strings.

Na verdade o char pode ser usado para representar  valores  de  faixa
pequena, dentro da faixa de 0 a 255, esta faixa de valores corresponde
aos valores atribuídos às letras do alfabeto e  sinais  matemáticos e 
outros símbolos usados em comunicação escrita.

O tipo signed char recebe valores de  -127 a 128,  a faixa de valores
do unsigned char vai de 0 a 255.

Exemplo:

char cLetra = 'x';
unsigned char nFilhos = 6;
signed char nSaldo = -100;

2.5 Declarando strings

Como dissemos acima o tipo char é usado para  declarar  strings. Você
pode usar essas duas maneiras de declarar  strings  na  linguagem  C:

char sNome[] = "Fernando Henrique";
char *sNome = "Fernando Henrique";

No capítulo sobre arrays e ponteiros você vai  se  deparar  com  mais
detalhes esclarecedores sobre estas duas maneiras de declarar strings
no programa.

2.6 O tipo inteiro curto - short int

Há tipo unsigned short int cuja variação é de 0 a 65535, e o
signed short int cuja faixa de valor é -32767 a 32768.

2.7 Declaração múltipla e inicialização de variáveis

Você pode declarar várias variáveis do  mesmo  tipo  usando  a  mesma
instrução de tipo. Por exemplo:

int x, y, z;
Na linha acima nós declaramos trêz variáveis inteiras.

Inicializar a variável significa prover o primeiro valor da variável,
a inicialização pode  ser  feita  depois  ou  durante  a  declaração:

int x = 100, y = 50, z = 100;


2.8 Quando e porque usar o tipo certo

Quanto maior a faixa de valores alcançados pela tipo  de  dado,  mais
espaço na memória ele ocupa e mais tempo de processamento  ele  gasta
durante os cálculos.   As operações com números decimais são as  mais
exigentes  em  termos  de  tempo  e  uso  de  espaço  na  memória. Em
face disso, é importante aplicar o tipo  de  dado  certo  conforme  a 
situação para economia de tempo e memória.

Por exemplo, vamos supor que você vai precisar de  uma  variável para
guardar o número de filhos de um casal, como não é comum um casal com
mais de 255 filhos,  você deve usar  unsigned char  para este caso, e
também porque não existe um casal com um número de  filhos negativos!
