06. Execução condicional e controle de fluxo 2/2

Vamos ver aqui neste capítulo a instrução  switch  e  o  operador  ?:
que é uma sintaxe curta e prática para  se  executar  uma  avaliação.

A instrução switch é  usada  para  avaliar  uma  variável  inteira  e
conforme o valor desta executa os blocos de códigos correspondentes a
cada valor.

A instrução  switch  equivale ao mesmo efeito que se pode obter com a
última  sintaxe  mostrada  do if  ( if... else if ),  que  permite  o
encadeamento de avaliações.  Porém com switch,  o  código  fica  mais
curto e mais legível.

Veja o programa exemplo:


/* Programa: switch.c - track06 */
/* Este programa testa a instrução switch */
#include "stdio.h"
#include "stdlib.h"

int main(void)  {

int nEscolha;
int nDinheiro = 500;
int espada_custo = 50;
int bota_custo = 60;
int capa_custo = 80;

int capa, bota, espada;
capa = bota = espada = 0;

/* Configura a janela */
system("title switch.c");
system("color F1");
printf("\n\n");

printf(" espada:%i bota:%i capa:%i \n", espada, bota, capa);
printf(" Seu saldo: %i \n\n", nDinheiro);
printf(" 1- espada  - 50 pts \n");
printf(" 2- bota    - 60 pts \n");
printf(" 3- capa    - 80 pts \n");
printf("\n\n Que item vc quer comprar? ");
scanf("%i",&nEscolha);

  switch(nEscolha) {
  case 1:
  printf(" Saindo uma espada...\n");
  nDinheiro = nDinheiro - espada_custo;
  espada = espada + 1;
  break;

  case 2:
  printf(" Saindo uma bota...\n");
  nDinheiro = nDinheiro - bota_custo;
  bota = bota + 1;
  break;

  case 3:
  printf(" Saindo uma capa...\n");
  nDinheiro = nDinheiro - capa_custo;
  capa = capa + 1;
  break;

  default:
  printf(" ...Hoje nao temos este item... \n");
  }

printf("\n\n Seu saldo: %i \n", nDinheiro);
printf(" espada:%i bota: %d capa: %i \n", espada, bota, capa);

printf("\n\n");
system("pause");

return 1;
}




 



Veja a sintaxe de uso da instrução switch:



switch (nVariável)
{
case nValor:
bloco de código;
break;

case nValor:
bloco de código;
break;

default:
bloco de código;
}

nVariável   representa   uma  variável  do   tipo  inteiro  declarada
anteriormente.   O bloco de código do  switch deve ser delimitado por
chaves.  nValor representa um valor constante  que  nVariável  assume.




A instrução break  quebra o switch, seguindo para a próxima instrução
depois da chave "}" de fechamento do switch.  A omissão de break  faz
com que o bloco de código do  próximo  case  seja executado, o quê em
algumas situações pode ser útil.

A instrução  default  é  opcional,  o  código  abaixo  de  default  é
executado caso  nenhuma  condição  seja  encontrada,  isto é,  quando
nVariável  assume  um  valor  que  não  é  tratado  por  case nValor.

6.1 O operador ?:

O  operador   ?:   serve  para  abreviar  uma  avaliação  cobrindo  a
funcionalidade de um curto bloco if...else.

Vamos supor  o caso de três números inteiros,  sendo que  o  primeiro
receberá  o  valor  do  que  for  maior  entre  os  outros  dois.

Com if...else você procede desta maneira:

if (a > b) nMaior = a;
else
nMaior = b;

Através do uso do operador ?: podemos abreviar este bloco para apenas
uma linha:

nMaior = (a > b) ? a:b

nRetorno = (expressão) ? nVerdadeiro:nFalso

A expressão lógica é avaliada,  se for  verdadeira  retorna  o  valor
nVerdadeiro,  caso  contrário,  retorna  nFalso.

Programa exemplo:


/* Programa: getmax.c - track06 */
/* Este programa testa a instrução ?: */
#include "stdio.h"
#include "stdlib.h"

int main(void)  {

int a = 10;
int b = 20;
int nMaior = 0;

/* Configura a janela */
system("title getmax.c");
system("color F1");
printf("\n\n");

printf (" Digite dois numeros: ");
scanf("%i",&a);
scanf("%i",&b);

nMaior = (a>b) ? a:b;

printf (" O maior e' %i \n",nMaior);

printf("\n\n");
system("pause");

return 1;
}
