8.0 Código ascii  
                                                                                      
                                                                                                                        
        a) Código ascii e a função ASC()                                                                         
        Cada carácter que aparece no teclado tem um código ascii, que é                                                 
        um código internacional para o intercâmbio de informaçòes.                                                      
                                                                                                                        
        Por exemplo, código ascii da letra A é 65. Para saber o código                                                  
        ascii de um carácter você usa a função ASC() do VBA.                                                         
                                                                                                                        
        x% = ASC(x$)                                                               
                                                                                                                        
        A função asc() retorna um número inteiro que representa o código                                                
        ascii da carácter informado.                                                             
                                                                                                                        
		MsgBox Asc("A")                                                                                                  
		letra$ = "B"
		codeascii% = ASC(letra$)
		MsgBox codeascii%,,letra$                                          
                                                                                                                        
                                                                                                                        
        b) A função CHR$()                                                                                       
                                                                                                                        
        x$ = CHR$(x%)                                                              
                                                                                                                        
        Esta função retorna o carácter que corresponde ao                                                               
        número fornecido. Ex.:                                                                   
                                                                                                                        
        MsgBox CHR$(65)                                                                                                  
        letra$ = CHR$(66): MsgBox letra$                                                                          
                                                                                                                        
        c) Mais detalhes sobre a tabela ascii                                                                    
                                                                                                                        
        O código ascii tem exatamente um byte de tamanho. É por isso                                                    
        que os valores variam de 0 a 255. Cada cáracter ocupa por isso                                                  
        um byte de memória. Um nome de 10 letras ocupa 10 bytes. Esta                                                   
        relação entre um byte para cada letra nos dá oportunidade de                                                    
        ter uma noção concreta da ocupação do espaço em informática.                                                    
                                                                                                 
                                                                                                                        
        Os caracteres de 0 a 32 são caracteres de controle usado na                                                     
        comunicação e controle de periféricos. Os valores de até 128                                                    
        são conhecidos como valores ascii padrão.                                                                       
                                                                                                                        
        Os valores de 129 até 255 são conhecidos como código ascii estendido,                                           
        e sobre estes valores não há rígidez de controle nos caracteres que                                             
        eles representam. A prova disso é que o mesmo código ascii estendido                                            
        impressos no MsDos e Windows apresentam caracteres diferentes.                                                 
                                                                                                                        
        Execute este programa para ver a tabela ascii completa:                                  
        FOR x% = 0 TO 255                                                                                               
        MsgBox x%,,CHR$(x%)                                                                                              
        NEXT x%                                                                                                         
        END
