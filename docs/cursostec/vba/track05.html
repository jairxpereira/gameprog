 5.A instrução IF e os desvios condicionais

        A instrução IF é a mais importante do mundo das linguagens de
        programação. Ela avalia expressões e toma atitudes baseado no
        resultado destas expressões caso elas sejam verdeiras ou falsas.

        Antes de nos aprofundarmos no IF vamos apresentar primeiro os
        operadores relacionais, que são aqueles sinais a seguir:

        >            Maior que
        <            Menor que
        <>         Diferente
        =               Igual
        >=           Maior ou igual
        <=           Menor ou igual

        IF expressão THEN ação
        Esta é a sintaxe mais simples da instrução IF.

        A expressão geralmente é uma sentença matemática que pode resultar
        em dois estados, falso ou verdeiro. A parte ação são os comandos
        e as instruções do VBA que vão ser executadas caso a expressão
        seja verdadeira.

        Exemplo 01:
		Dim nome As String
		Dim idade As Integer
		Dim txt As String   

		txt = InputBox("Qual a sua idade?", "Coleta de informações")

		' A função Val() converte um texto em dado numérico
		idade = Val(txt)


		IF idade < 18 THEN MsgBox "Você é de menor."
		IF idade > 18 THEN MsgBox "Você é de maior."
		IF idade = 18 THEN MsgBox "Você alcançou a maioridade."

        A parte ação somente é executada caso a expressão
        avaliada resulte numa verdade.

        a) Desvios condicionais e instruindo fim de programa

        Você pode combinar rótulos, IF's e GOTO's para criar desvios
        condicionais. Exemplo 02:

inicio:
                Dim nome As String
                Dim idade As Integer
                Dim txt As String

                nome = InputBox("Qual o seu nome?", "Coleta de informações")
                txt = InputBox("Qual a sua idade?", "Coleta de informações")

                ' A função Val() converte um texto em dado numérico
                idade = Val(txt)

                If idade < 18 Then GoTo MenorDeIdade
                If idade >= 18 Then GoTo MaiorDeIdade
                End

MenorDeIdade:
                MsgBox " não tem 18 anos.", , nome: End
MaiorDeIdade:
                MsgBox " é maior de 18 anos.", , nome: End       

        b) Terminando a execução do programa com comandos

        Geralmente o programa encerra-se naturalmente ao atingir a
        última linha executável. Quando houver necessidade de encerrar
        o programa antes de seu final linear, use estes comandos a
        seguir:

        O comando END
        O comando END faz o programa ser encerrado normalmente.

        O comando STOP
        O comand STOP pára a execução do programa de maneira abrupta.
