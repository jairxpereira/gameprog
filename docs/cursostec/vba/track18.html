 18. Final do Curso Básico de VBA
                                                                    
        Você está preparado para fazer muita coisa com o VBA. Como final deste                                       
        curso básico nós vamos passar como programar um jogo da forca.                                                  
                                                                                                                        
        Não vá direto ao computador desenvolver o seu programa.                                                         
        Pense um pouco sobre ele. Defina qual que é objetivo do                                                         
        programa e divida a execução desta tarefa em várias etapas                                                      
        ou subrotinas.                                                                                                  
                                                                                                                        
        Pense antecipadamente em todas as subrotinas que este programa                                                  
        deve ter.                                                                                                       
                                                                                                                        
        Depois de ter feito o rascunho do que você deve fazer no papel                                                  
        vá então desenvolver o seu programa.                                                                            
                                                                                                                        
        Vamos aplicar este método ao nosso Jogo da Forca.                                                               
                                                                                                                        
        O que é o Jogo da Forca?                                                                                        
        O jogo da Forca é um jogo no qual o computador sorteia uma                                                      
        palavra que nós devemos descobrir qual é.                                                                       
                                                                                                                        
        1- Devemos escolher as palavras para o jogo, colocando-as                                                       
        no comando DATA.                                                                                                
        Deve haver uma variável para vidas e pontos                                                                     
                                                                                                                        
                                                                                                                        
        2- Criamos uma matriz de strings para estocar estas palavras                                                    
                                                                                                                        
        3- Desenvolvemos a subrotina que sorteia um número aleatório                                                    
        que corresponde a uma palavra                                                                                   
                                                                                                                        
        3.5 O computador mostra a primeira letra da palavra sorteada.                                                   
        Na tela, para cada letra da palavra dever ser exibido um tracinho                                               
        na tela.                                                                                                        
                                                                                                                        
                                                                                                                        
        4- Escrevemos a subrotina que trata o teclado. Para cada letra                                                  
        pressionada, o programa deve verificar se ela existe na palavra.                                                
        Caso, a letra não exista, as vidas devem ser diminuidas em 1.                                                   
                                                                                                                        
        4.5 Caso a letra exista, devemos trocar o tracinho correspondente                                               
        na tela por ela                                                                                                 
                                                                                                                        
        5- Caso o a palavra seja descoberta, uma boa mensagem deve                                                      
        ser exibida e o jogador ganha 10 pontos.                                                                        
                                                                                                                        
        6- A cada 40 pontos o jogador ganha uma vida extra                                                              
                                                                                                                        
        7- Caso o jogador falhe em acertar a palavra, o computador                                                      
        mostra a palavra secreta e exibe "Game Over" e pergunta se                                                      
        o jogador quer jogar novamente.
