 13. O gerador de números aleatórios
                                                                    
        Há um interessante recurso do Qbasic que se chama gerador de                                                 
        números aleatórios ou randômicos. Este gerador sorteia um                                                   
        número dentro da grandeza fornecida como se fosse um sorteio                                                    
        da loto.                                                                                                        
                                                                                                                        
        Este recurso é usado em simulações ou em jogos. Nas simulações                                                  
        ele é usado para gerar números para algum processo para evitar                                                  
        que esses números sejam digitados manualmente.                                                                  
                                                                                                                        
        Em jogos, ele é usado para criar o elemento surpresa, para                                                      
        espalhar os inimigos pelo cenário construído sem nenhuma ordem                                                  
        conhecida.                                                                                                      
                                                                                                                        
        Para usar este gerador você deve usar estas linhas                                                              
                                                                                                                        
        RANDOMIZE TIMER                                                            
        Este comando deve estar no início de seu programa para                                                          
        ligar o gerador.                                                                                                
                                                                                                                        
        x! = RND * valor%                                                          
        Esta sintaxe sorteia um número entre 0 e valor%. Também é gerado                                                
        uma parte decimal aleatória.                                                             
        Ex.: sorteio! = RND * 100               'Pode resultar em 76.10                                                 
                                                                                                                        
        x% = INT(RND * valor%)                                                                            
        Use esta sintaxe para obter o sorteio de um número inteiro.                              
        Ex.: x% = INT(RND*100): MsgBox x%        'Pode resultar em 23                                             
																													
	Para excluir o zero, some mais um ao final da linha:                                     
	linha% = INT(RND*25)+1: 
	MsgBox "Surpresa!",,str$(linha%)                                              
                                                                                                                        
        Sem o comando RANDOMIZE TIMER os mesmos números sorteados se                                             
        repetem toda vez que o programa é executado.
