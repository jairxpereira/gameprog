 6. O bloco IF...THEN...ELSE

        A instrução IF permite a construção de programas sofisticados.
        Podemos usá-la em forma de blocos que você vai ver a seguir.

        IF  expressão THEN ação ELSE AçãoAlternativa
        Nesta sintaxe, a ação normal é executada caso o resultado da
        da expressão seja verdadeiro; caso o resultado seja falso a
        Ação Alternativa é executada.

        Nosso programa que verifica maioridade pode ser escrito desta
        forma:

        IF idade >= 18 THEN MsgBox "Você é de maior" ELSE MsgBox "Você é de menor"

        Esta linha é interpretada desta maneira: SE idade for maior ou igual
        a 18 ENTÃO vou exibir "Você é de maior." CASO CONTRÁRIO vou exibir
        "Você é de menor."

        a) O bloco IF...THEN...ELSE

        A formatação em bloco do IF acontece com esta sintaxe:

        IF expressão THEN
                código
        ELSE
                código
        END IF

        Com código queremos informar que nesta parte você deve
        colocar os comandos,instruções do seu programa que ali serão
        executados conforme a expressão for verdadeira ou não.

        O nosso programa que verifica maioridade ficaria assim organizado:
        
        IF idade >= 18 THEN
                MsgBox "Você é de maior"
        ELSE
                MsgBox "Você é de menor"
        END IF

        O bloco permite construções maiores, melhores e mais limpas.

        b) Operadores lógicos

        Os operadores lógicos permitem a avaliaçao de uma combinação de
        expressões numa instrução IF e em outras instruções do Qbasic.

        Abaixo estão os operadores lógicos:

        AND..............e
        OR...............ou
        NOT..............não

        Por exemplo:
        Dim idade As Integer
        Dim nSaude As Integer
        Dim txt As String        

        txt = InputBox("Qual a sua idade?", "Coleta de informações")

        ' A função Val() converte um texto em dado numérico
        idade = Val(txt)
        
        txt = InputBox("Sua saúde está boa? 1=Sim 0=Não", "Coleta de informações")
        nSaude = Val(txt)
        
        If idade < 18 Or nSaude = 0 Then
        MsgBox "Você não pode tirar sua carta de habilitação."
        Else: MsgBox "Você pode tirar sua carta de habilitação."
        End If
        End
        
		
	O operador OR retorna verdadeiro se uma ou outra expressão for
    verdadeira. 

     O operador AND retorna verdadeiro caso as duas expressões sejam verdadeiras.        

        O operador NOT faz uma negação.
        Ex.:
        IF NOT (idade = 18) THEN MsgBox "Você não tem 18 anos"

        Nesta linha, se idade não for igual a 18 então a mensagem
        será impressa. NOT inverte o resultado da avaliação entre parêntesis.
