02. Variáveis e tipos de dados


1.Variável
A variável assemelha-se a uma caixa na qual guardamos um objeto
para recuperá-lo e usá-lo mais tarde.  Em  um  programa  VBA  a 
variável  é  um  nome que representa uma informação ou um número 
que a qualquer momento pode ser submetido a cálculo ou a outros
processos.

Tecnicamente a variável é um endereço numerado de memória na qual
as informações são gravadas para sofrerem processos posteriores.

1.1 Tipo de dados
O tipo de dado refere-se a natureza dessa informação, por exemplo,
se a informação é um número dizemos que  a  variável  é  do  tipo 
numérica; se a informação é um nome ou  um  endereço  residencial 
dizemos que ela é do tipo alfanumérica.

As variáveis numéricas se dividem em subtipos conforme a grandeza
de seus valores.

2. Declarando variáveis alfanuméricas (strings)

O tipo mais comum de variável é a variável alfanumérica que
pode ter como conteúdo letras, números e caracteres especiais.
A variável alfanumérica é conhecida como string. Para
declarar uma variável string proceda desta forma:

DIM nome AS STRING
Aqui criamos uma variável string que será usada para guardar
um nome.

Inicialização é o processo de prover o conteúdo da variável.
Depois que declaramos a variável usamos o sinal de igual para
estabelecer seu conteúdo:

nome = "Jair"
Assim o conteúdo da variável do tipo string nome é a string "Jair".
As strings devem sempre estar entre aspas.

Há uma outra forma de se criar variáveis strings, basta apenas
colocar o sinal de dólar ($) após o seu nome e prover em seguida
o conteúdo. O mesmo exemplo neste estilo fica assim: nome$ = "Jair".
Contudo, a declaração da string usando DIM é mais profissional.


Uma string vazia é produzida com aspas vazias: nome = "".
Esta sintaxe esvazia o conteúdo de nome caso haja conteúdo anterior.


b) Declarando e inicializando variáveis numéricas

As variáveis numéricas se dividem nos seguintes grupos:

Identificador do Tipo de Dado            Grandeza de valor
% ou INTEGER                    -32767 a 32767
& ou LONG                       -2.147.483.648 a 2.147.483.648

! ou SINGLE                     -3.2*10^38  a 3.2 * 10^+38
# ou DOUBLE                     -1.7*10^308 a 1.7*10^+308

(*) O chapéuzinho (^) indica uma operação de potência. Então 2^3 quer
dizer dois (2) elevado(^) ao cubo.

Então para declarar variáveis numéricas usamos a seguinte sintaxe:
DIM idade AS INTEGER
DIM salario AS LONG
DIM pi AS SINGLE
DIM astronumber AS DOUBLE

Depois inicialize as variáveis normalmente:
idade = 18
salario = 40000
pi = 3.141516
astronumber = 2^33 


Declaração e inicialização do jeito simples é assim:
idade%=18
salario&=40000
pi!=3.141516
astronumber#=2^-33

c)Considerações importantes sobre o tipo de dado.

Quanto maior é o número, maior é o consumo de bytes que ele requer para
ser estocado e maior tempo ele gasta para ser processado, então procure
usar o tipo de dado certo para suas variáveis.

Por exemplo, se seu  programa pede a idade de uma pessoa, não use um
tipo LONG para esta variável, isto porque uma pessoa vive no máximo
120 anos e LONG pode atingir até 2 bilhões. Um INTEGER reserva 2 bytes
de memória para ser estocado, o LONG reserva quatro bytes.

É importante declarar o tipo da variável, ou colocar o identificador
junto com seu nome. Esta medida torna o processamento do programa
mais rápido.

Os nomes das variáveis podem ter até 40 letras, podem incluir números
mas devem começar por uma letra.

A sintaxe geral do comando DIM é esta:

DIM NomeDaVariável AS TipoDeDado
