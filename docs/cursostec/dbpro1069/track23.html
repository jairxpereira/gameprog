23. Video
                                                                     
 O DarkBasic apresenta comandos para verificar e configurar o modo de video. Um                                         
 dos aspectos do modo de vídeo é a resolução de tela que inicialmente é 640x480.                                 
 Outras resoluções de vídeo que são comuns é a 800x600 e a 1024x768.                                                    
                                                                                                                        
 Outro aspecto com relação ao video é a profundidade que determina o número de                                   
 cores que pode ser gerado, quanto maior a profundidade, maior o número de                                              
 cores e maior o gasto em memória e processamento de imagens. As profundidades                                          
 mais comuns são de 16,24 e 32 bits, sendo a de 24 bits a mais fartamente encontrada                                    
 nas imagens que circulam.                                                                                              
                                                                                                                        
 O terceiro aspecto com relação ao vídeo é a aceleração 3D que depende das                                       
 características do hardware. Atualmente é muito comum o computador doméstico                                           
 vir com um sistema de aceleração 3D embutido mas com certeza a melhor opção                                            
 é a aquisição de uma placa aceleradora 3D. Os recursos 3D destas placas diferem                                        
 de um modelo para outro. O DarkBasic oferece funções para verificar a existência                                       
 dessas placas e dos recursos que possivelmente elas podem trazer.                                                      
                                                                                                                        
 PERFORM CHECKLIST FOR DISPLAY MODES                                                                      
 Este longo comando realiza uma pesquisa de todas as resoluções de video disponíveis.                                   
 Você vai precisar utilizar alguns comandos para verificar a saída desse comando:                                       
                                                                                                                        
 nQtd = CHECKLIST QUANTITY() ..... Quantidade de modos disponíveis                                                      
 xcol = CHECKLIST VALUE A(ndx) ... Retorna resolução horizontal (640,800,1024...)                                       
 ylin = CHECKLIST VALUE B(ndx) ... Retorna resolução vertical (480,600,768...)                                          
 zdep = CHECKLIST VALUE C(ndx) ... Retorna a profundidade de cor                                                        
                                                                                                                        
 Veja nosso programa exemplo:                                                                                           

                                                       
cls:hide mouse                                                                                                          
x$ = space$(3)                                                                                                          
                                                                                                                        
perform checklist for display modes                                                                                     
nqtd = checklist quantity()                                                                                             
print nqtd                                                                                                              
                                                                                                                        
for ncx = 1 to nqtd                                                                                                     
                                                                                                                        
xcol = checklist value a(ncx)                                                                                           
ylin = checklist value b(ncx)                                                                                           
zdep = checklist value c(ncx)                                                                                           
                                                                                                                        
print xcol,x$, ylin,x$,zdep                                                                                             
                                                                                                                        
if ncx = 20                                                                                                             
suspend for key                                                                                                         
cls                                                                                                                     
endif                                                                                                                   
                                                                                                                        
next ncx                                                                                                                
suspend for key
                                                                                            
                                                                                                                        
 nBool = CHECK DISPLAY MODE(nLargura, nAltura, nProfundidade)                                             
 Este comando verifica a existência de um modo de vídeo particular; caso o                                              
 o modo exista, o valor retornado é um.                                                                                 
                                                                                                                        
 Ex.:                                                                                           
 cls: hide mouse                                                                                                        
 xcol = 800: ylin = 600: zdep = 32                                                                                      
 nbool = check display mode(xcol,ylin,zdep)                                                                             
 if nbool = 1 then print "(800,600,32) OK"                                                                       
                                                                                                                        
 SET DISPLAY MODE nLargura, nAltura, nProfundidade                                                        
 Este comando configura o modo de vídeo com a nova resolução dada por                                                   
 nLargura, nAltura e nProfundidade.                                                                                     
                                                                                                                        
 Ex.: set display mode 800,600,32                                                        
                                                                                                                        
 Verificando o modo de vídeo corrente                                                                            
 xcol = SCREEN WIDTH() .... Retorna a largura atual                                                              
                                                                                                                        
 ylin = SCREEN HEIGHT() ... Retorna a altura corrente                                                            
                                                                                                                        
 zdep = SCREEN DEPTH() .... Retorna a profundidade                                                               
                                                                                                                        
 nfps = SCREEN FPS() ...... Retorna a capacidade de frames por segundo                                           
                                                                                                                        
 n3d = SCREEN TYPE() ...... Retorna um caso tenha placa de aceleração 3D, ou zero caso             
                            não tenha.                                                                                  
                                                                                                                        
 Veja nosso programa exemplo:                                                                                           

                                                       
cls:hide mouse                                                                                                          
xcol = SCREEN WIDTH()                                                                                                   
ylin = SCREEN HEIGHT()                                                                                                  
zdep = SCREEN DEPTH()                                                                                                   
nfps = SCREEN FPS()                                                                                                     
n3d =  SCREEN TYPE()                                                                                                    
                                                                                                                        
PRINT "xcol: ",xcol                                                                                                     
print "ylin: ",ylin                                                                                                     
print "zdep: ", zdep                                                                                                    
print "fps: ", nfps                                                                                                     
print "n3d: ", n3d                                                                                                      
suspend for key
                                                                                            
                                                                                                                        
 VideoDrv$ = CURRENT GRAPHICS CARD$()                                                                     
 Esta função retorna o nome do dispositivo de vídeo corrente. Um exemplo de retorno                                     
 desta função é "Driver de vídeo primário".                                                                             
                                                                                                                        
 PERFORM CHECKLIST FOR GRAPHICS CARDS                                                                     
 Os computadores podem ter mais de uma placa gráfica instalada. Este comando pesquisa                                   
 o nome dos dispositivos de controle de vídeo que estiverem instalados.                                                 
 Exemplo:                                                                                       
 perform checklist for graphics cards                                                                                   
 ntam = checklist quantity()                                                                                            
 for ncx = 1 to ntam                                                                                                    
 print checklist string$(ncx)                                                                                           
 next ncx                                                                                                        
                                                                                                                        
 Controlando a aceleração gráfica por hardware                                                            
 SET EMULATION OFF                                                                                               
 Este comando desliga a emulação de aceleração gráfica por software, passando                                           
 a utilizar os recursos de hardware do sistema para a renderização. Utilize este                                        
 comando caso você tenha uma placa gráfica instalada em seu sistema.                                                    
                                                                                                                        
 SET EMULATION ON                                                                                                
 Este comando liga a emulação de aceleração gráfica por software. Utilize este                                          
 comando caso você não tenha uma placa gráfica instalada em seu sistema.                                                
                                                                                                                        
 nBool = EMULATION MODE()                                                                                        
 Esta função verifica o modo de aceleração gráfica. O valor zero indica que a emulação                                  
 por software está desligada, com o valor um indicando que está ligada.                                                 
                                                                                                                        
 nBool = TNL AVAILABLE()                                                                                         
 Este comando informa se o controlador de vídeo está usando capacidade de hardware para                                 
 processar transformação e iluminação. TNL é uma abreviação para Transformação (T) e(N)                                 
 Iluminação (Lightening). O valor um indica que este recurso está em uso.                                        
                                                                                                                        
 ENABLE TNL/DISABLE TNL                                                                                          
 O primeiro comando habilita a renderização de transformação e iluminação via hardware, caso                            
 a sua placa gráfica disponha desse recurso. DISABLE TNL desliga o recurso de TNL.
