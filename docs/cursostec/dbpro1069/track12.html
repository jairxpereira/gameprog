12. Objetos 3D básicos 1/2
                                                    
                                                                                                                        
 Vamos introduzir neste capítulo a criação e o posicionamento de objetos                                                
 tridimensionais básicos. Essa introdução é para fornecer a base para                                                   
 abordarmos a criação de câmera e terreno nos próximos capítulos.                                                       
                                                                                                                        
 O DarkBasic oferece comandos para criar objetos 3D primitivos como pisos,                                              
 esferas, cones, caixas e cilindros.                                                                                    
                                                                                                                        
 Esses objetos apresentam propriedades básicas como raio, largura, altura,                                              
 profunidade, textura e posição dada em termos de (x,y,z). Sendo cada um                                                
 destes aspectos passível de ser manipulado por outros comandos.                                                        
                                                                                                                        
 Tal como os bitmaps e imagens, esses objetos recebem um número de identificação                                        
 pelo qual eles sofrem as manipulações posteriores.                                                                     
                                                                                                                        
 MAKE OBJECT SPHERE nObj, nRaio                                                                                  
 Este comando cria uma esfera com o raio especificado por nRaio e sendo identificado                                    
 pelo número inteiro nObj.                                                                                              
                                                                                                                        
 Ex.:   MAKE OBJECT SPHERE 1,2                                                           
                                                                                                                        
 POSITION OBJECT nObj, x, y, z                                                                                   
 Este comando reposiciona o objeto 3D identificado por nObj para a posição                                              
 dada por (x,y,z).                                                                                                      
                                                                                                                        
 Ex.:                                                                                           
 MAKE OBJECT SPHERE 1,2                                                                                                 
 POSITION OBJECT 1,0,0,10                                                                                               
 SUSPEND FOR KEY                                                                                                 
                                                                                                                        
 DELETE OBJECT nObjeto                                                                                           
 Quando um objeto não tiver mais uso, apague-o para preservar recursos                                                  
 de processamento como memória e velocidade. Você deleta um objeto                                                      
 usando DELETE OBJECT.                                                                                           
                                                                                                                        
 TEXTURE OBJECT nObjeto, nImagem                                                                                 
 Este comando texturiza o objeto nObjeto com a imagem selecionada.                                                      
 Veja que é muito simples texturizar um objeto:                                                                         
                                                                                                                        
 Ex.:                                                                                           
 LOAD IMAGE "c:\media\textura.bmp",1                                                                                    
 MAKE OBJECT SPHERE 1,2                                                                                                 
 TEXTURE OBJECT 1,1                                                                                                     
 POSITION OBJECT 1,0,0,10                                                                                               
 SUSPEND FOR KEY                                                                                                 
                                                                                                                        
 COLOR OBJECT nObjeto, nCor                                                                                      
 Este comando colore o objeto com a cor dada. Você pode gerar a cor usando a                                            
 função RGB() que tem a seguinte sintaxe, nCor = RGB(nRed, nGreen, nBlue).                                              
                                                                                                                        
 Ex.:                                                                                           
 MAKE OBJECT SPHERE 1,10                                                                                                
 COLOR OBJECT 1,RGB(255,10,10)                                                                                   
                                                                                                                        
                                                                                                                        
 MAKE OBJECT BOX nObj, nLargura, nAltura, nProfundidade                                                          
 Este comando cria uma caixa, com largura, altura e profundidade especificados                                          
 por nLargura, nAltura e nProfundidade.                                                                                 
                                                                                                                        
 Ex.:  make object box 1,2,20,100                                                        
                                                                                                                        
                                                                                                                        
 Rodando os objetos                                                                                              
 O DarkBasic apresenta vários comandos para rodar um objeto em cada eixo                                                
 individualmente e um comando para aplicar a rotação em todos os eixos                                                  
 simultâneamente.                                                                                                       
                                                                                                                        
 XROTATE OBJECT nObj, rx ........ Roda o objeto no eixo X                                                               
 YROTATE OBJECT nObj, ry ........ Roda o objeto em torno de si mesmo no eixo Y.                                         
 ZROTATE OBJECT nObj, rz ........ Roda o objeto no eixo Z.                                                              
 ROTATE OBJECT nObj, rx,ry,rz ... Roda o objeto nos três eixos simultâneamente.                                         
                                                                                                                        
 Os argumentos rx,ry,rz especificam o ângulo de rotação, sua faixa de valores                                           
 vai de 0 a 360.                                                                                                        
                                                                                                                        
 SCALE OBJECT nObj, xs, ys, zs                                                                                   
 Este comando aplica escala no objeto, ocasionado, dependendo do valor aplicado,                                        
 redução ou ampliação do objeto.                                                                                        
                                                                                                                        
 xs é a escala horizontal no eixo x.                                                                                    
 ys é a escala vertical no eixo y.                                                                                      
 zs é a escala de profundidade no eixo z.                                                                               
                                                                                                                        
 Uma escala em um eixo individualmente provoca uma escala não-uniforme no objeto                                        
 dando o efeito de um esticamento e ocasionando a perda de simetria geométrica no                                       
 objeto. Essa escala não-uniforme é interessante para criar variações de objetos                                        
 com o mesmo objeto base.                                                                                               
                                                                                                                        
 Para aplicar um efeito uniforme e assim preservar a simetria e forma original do                                       
 objeto, configure com o mesmo valor os argumentos de escala xs, ys e zs.
