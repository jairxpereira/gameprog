09.2 Obtendo a lista de modos de vídeo disponíveis

O usuário pode rodar sua aplicação em um computador com configuração inferior 
a sua ou ainda em condições limitadas como um netbook ou um tablet etc.	

Então é importante que sua aplicação verifique as condições do hardware aonde
está sendo rodada e que se adapte aos limites dessas condições quando isto 
não comprometer a finalidade da mesma. Se não der pra rodar mesmo, pelo menos
sua aplicação deve sair do ar com graciosidade enquanto informa o usuário das
condições mínimas de hardware para executá-la. 

Nesta seção vamos aprender como	obter uma lista de modos de vídeo disponíveis
para serem usadas na função set display mode.

perform checklist for display modes
Este comando produz uma lista com todos os modos de vídeo disponíveis.

nqtd = checklist quantity()
Essa função retorna a quantidade total de modos de vídeo disponíveis.

nLargura = checklist value a(ncx)
Essa função retorna a largura do modo de vídeo referente ao índice da
lista especificado (ncx). 

nAltura = checklist value b(ncx);
Essa função retorna a altura do modo de vídeo referente ao índice da
lista especificado (ncx). 

nProfundidade = checklist value c(ncx);
Essa função retorna a profundidade de cor do modo de vídeo referente ao
índice da lista especificado (ncx). 

Nas funções acima o argumento ncx que representa um índice de acesso a 
um item numa lista e o valor de retorno são do tipo integer.









Rem Project: videolist
Rem Created: 04/08/2013 17:32:47

inicio:
global branco as integer = 0xffffff
global azul   as integer = 0x0000ff

' Propriedades de vídeo
nLargura       as integer = 0
nAltura        as integer = 0
nProfundidade  as integer = 0

txt            as string = ""
ncx            as integer = 0
nqtd           as integer = 0

initsys("videolist.dba")

Perform Checklist For Display Modes
nqtd = Checklist Quantity()

' Cabeçalho de apresentação da lista
Print
Print " #ncx: (largura x altura x profundidade)"
Print "------------------------------------------"

 for ncx = 1 to nqtd
 ' Obtém largura, altura e profunidade do modo de vídeo #ncx
 nLargura         = Checklist Value A(ncx)
 nAltura          = Checklist Value B(ncx)
 nProfundidade    = Checklist Value C(ncx)

 if (ncx = 4 or ncx = 10 or ncx = 13) then print

 ' Mostra capacidade de vídeo
 print " #", ncx, space$(4);
 print nLargura," x ", nAltura," x ", nProfundidade

 next ncx

final:
print
print " Pressione qualquer tecla para encerrar"
suspend for key
end

' ----- initsys() - Configuração básica da aplicação -----
function initsys(txt as string)
' configurando a janela
set window on
set window title txt
ink azul, branco
cls branco
endfunction
