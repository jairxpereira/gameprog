16. Desenho 2D
                                                                
                                                                                                                        
 Neste capítulo nós vamos apresentar comandos básicos de desenho para produzir                                          
 formas básicas tais como quadrados, círculos e outras coisas relacionadas.                                             
                                                                                                                        
 É interessante lembrar que podemos manipular uma imagem bitmap na memória                                              
 antes de mostrá-la; essa manipulação também pode ser feita com qualquer                                                
 comando de desenho ou texto, basta selecionar o bitmap alvo com o comando                                              
 SET CURRENT BITMAP, e assim o bitmap passa a receber todos os efeitos de                                               
 desenho.                                                                                                               
                                                                                                                        
 nCor = RGB (nRed, nGreen, nBlue)                                                                                
 Esta função retorna uma cor combinando os três componentes primários de cor                                            
 que são o vermelho (nRed), verde (nGreen) e azul (nBlue).                                                              
                                                                                                                        
 Ex.:  nCor = rgb(255,10,10)                                                             
                                                                                                                        
 CLS nCor                                                                                                        
 O comando CLS, além de limpar a tela pode ser usado para determinar a cor de                                           
 fundo. Você também pode usá-lo para limpar o conteúdo de um bitmap e deixando-o                                        
 com uma cor de fundo especificada.                                                                                     
                                                                                                                        
 Ex.: CLS RGB(0,0,255)                                                                   
                                                                                                                        
                                                                                                                        
 INK nFrente, nFundo                                                                                             
 Este comando é usado para estabelecer a cor de frente e a cor de fundo. A cor                                          
 de frente determina a cor da linha dos desenhos produzidos pelos comandos de                                           
 desenho. A cor de fundo também é chamada de cor de background e é usada para                                    
 colorir a tela no efeito de um comando CLS.                                                                            
                                                                                                                        
 Ex.:                                                                                           
 nVermelho = RGB(155,10,10)                                                                                             
 nVerde = RGB(0,255,0)                                                                                                  
 INK nVermelho, nVerde                                                                                                  
 CLS                                                                                                             
                                                                                                                        
                                                                                                                        
 SYNC ON...SYNC                                                                                                  
 Esse par de comandos é utilizado para controlar manualmente o refrescamento ou                                         
 o redesenhamento da tela. O uso deles corrige dois problemas que podem ocorrer                                         
 com a exibição de textos e desenhos; o primeiro problema que ocorre é o visual                                         
 flamejante ou cintilante que tira a firmeza dos desenhos, o segundo problema                                           
 é a perda de velocidade na medida em que aumenta o volume de textos e desenhos                                         
 exibidos na tela.                                                                                                      
 Use SYNC ON no topo do seu programa, e use SYNC depois de todos os comandos de                           
 desenho e exibição de texto.                                                                                           
                                                                                                                        
 CIRCLE xcol, ylin, nRaio                                                                                        
 Este comando desenha um círculo na posição especificada, com o tamanho                                                 
 dado por nRaio.                                                                                                        
                                                                                                                        
 Ex.:  CIRCLE 320,240,50                                                                 
                                                                                                                        
 DOT xcol, ylin                                                                                                  
 Este comando desenha um ponto na tela na coordenada dada por (xcol,ylin).                                              
                                                                                                                        
 Ex.:                                                                                                                   
 CIRCLE 320,240,50                                                                              
 DOT 320,240                                                                                                     
                                                                                                                        
 LINE xcol, ylin, dx,dy                                                                                          
 Este comando desenha uma linha através da ligação dos dois pontos dados por                                            
 (xcol,ylin) e (dx,dy).                                                                                                 
                                                                                                                        
 Ex.: LINE 320,240,370,290                                                               
                                                                                                                        
 BOX xcol, ylin, dx,dy                                                                                           
 Este comando desenha uma caixa através da ligação dos dois pontos diagonais dados                                      
 por (xcol,ylin) e (dx,dy).                                                                                             
                                                                                                                        
 Ex.: BOX 320,240,370,290                                                                
                                                                                                                        
 ELLIPSE xcol, ylin, dx,dy                                                                                       
 Este comando desenha uma elipse na coordenada dada por (xcol,ylin), raio horizontal                                    
 dado por dx e raio vertical dado por dy.                                                                               
                                                                                                                        
 Ex.: ELLIPSE 320,240,120,60                                                             
                                                                                                                        
 nCor = POINT(xcol,ylin)                                                                                         
 Este comando é usado para obter a cor do ponto ou pixel na coordenada dada por                                         
 (xcol,ylin). Este comando é útil para detectar a presença de um objeto animado                                         
 através da sua cor para verificar colisão ou estabelecer alguma reação inteligente                                     
 em seu personagem baseado na proximidade de algum objeto.  Este comando também é                                       
 útil para ler um bitmap ponto por ponto.                                                                               
                                                                                                                        
 Você pode obter separadamente o valor de cada cor primária do valor nCor através                                       
 das funções nVermelho = RGBR(nCor), nVerde = RGBG(nCor), nAzul = RGBB(nCor).
