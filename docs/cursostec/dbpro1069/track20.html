20. Terrenos 2/2
                                                              
 Rolando a textura do terreno                                                                                    
 O Darkbasic possui comandos para rolar a textura da matrix para cima, baixo, esquerda                                  
 e direita criando a ilusão de movimento e deslocamento. Por exemplo, em um jogo de                                     
 corrida pode-se utilizá-los para rolar o asfalto para dar a impressão de que o                                         
 veículo está correndo.                                                                                                 
                                                                                                                        
 SHIFT MATRIX UP nMatrix  ....... Desloca a textura da matrix para cima                                          
 SHIFT MATRIX DOWN nMatrix ...... Desloca a textura da matrix para baixo                                         
 SHIFT MATRIX LEFT nMatrix ...... Desloca a textura da matrix para esquerda                                      
 SHIFT MATRIX RIGHT nMatrix ..... Desloca a textura da matrix para direita                                       
                                                                                                                        
 POSITION MATRIX nMatrix, xpos, ypos, zpos                                                                       
 Este comando reposiciona a matrix no mundo 3D; nMatrix é o terreno que você quer                                       
 reposicionar e (xpos,ypos,zpos) representa a nova posição.                                                             
                                                                                                                        
 Renderização wirefrane da matrix                                                                                
 SET MATRIX WIREFRAME ON nMatrix ..... Aciona visualização wireframe                                             
 SET MATRIX WIREFRAME OFF nMatrix .... Desliga visualização wireframe                                            
                                                                                                                        
 Efeito fantasma da matrix                                                                                       
 GHOST MATRIX ON nMatrix .... Aciona efeito fantasma                                                             
 GHOST MATRIX OFF nMatrix ... Desliga efeito fantasma
