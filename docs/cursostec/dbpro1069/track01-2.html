01.2 Estrutura de dados


Você pode criar um novo tipo de dado composto agregando vários tipos básicos
em um só conjunto semelhante a um registro de dados. Por exemplo, podemos
criar um dado tipo Monstro que agrega atributos como energia, capacidade
de estrago e tipo de monstro.
 
Proceda desta maneira para criar um novo tipo de dado:
 

 
 type Monstro	
	   nome	 as string
	   tipo	 as byte 
	energia	 as integer
	estrago	 as byte	  
 endtype
 
 
Depois para utilizar este novo tipo de dado proceda dessa maneira:
 

' Criando uma variável tipo Monstro
boss as Monstro
boss.nome = "Godzila"
boss.tipo = 1
boss.energia = 100
boss.estrago = 5

' Monstrando os dados na tela
print " Monstro.nome:    ", boss.nome
print " Monstro.tipo:    ", boss.tipo
print " Monstro.energia: ", boss.energia
print " Monstro.estrago: ", boss.estrago
