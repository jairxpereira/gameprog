06.1 Funções

1. Declarando funções
 function...endfunction
 Funções são bloco de códigos que realizam uma tarefa, e geralmente recebem
 valores de entrada e emitem valores de saída.
 Você monta as suas funções usando as instruções function...endfunction:
         
 function Nome_da_função(argumento, argumento)
 (...) código 
 endfunction Valor_de_retorno 
 
 Os argumentos de uma função são os valores de entrada que são declaradas
 como variáveis locais após o nome da função. 
 Ex:  quadrado(nval as integer). 
 
 Dica: Após o nome da função não coloque um espaço antes do parentesis
 de abertura, senão o compilador acusa erro de sintaxe. 

 Ex.: 


 function quadrado(nval as integer)  ' recebe um valor de entrada (nval)
 temp = nro * nro               ' calcula o quadrado do argumento
 endfunction temp               ' retorna o quadrado do número dado

 valor = quadrado(12)           ' Exemplo de uso básico da função 
 


 Perceba que o valor retornado é dado em endfunction. Os argumentos de
 entrada e o valor de retorno são opcionais.

 Você pode sair do bloco de código a qualquer momento usando a instrução
 exitfunction.

 As funções devem ser definidas no final do programa.

 Veja o programa exemplo:
 

  
 

 


Rem Project: functions
Rem Created: 03/08/2013 08:41:52

inicio:
' Configurando a janela
branco as integer = 0xFFFFFF
azul   as integer = 0x0000FF
set window on
set window title "functions.dba"
ink azul, branco
cls branco


valor as integer
quad as double integer

repeat
print
input " Digite um numero ou zero para encerrar: ", valor
quad = quadrado(valor)
print " Quadrado = ", quad

print
print " Pressione qualquer tecla para continuar"
suspend for key
cls branco
until (valor = 0)
end

function quadrado( nval as integer)
temp as double integer
temp = nval * nval
endfunction temp



2. Variáveis globais

 As variáveis declaradas dentro de uma função são locais, isto é,
 são vistas apenas internamente pela própria função.
 
 Em muitas ocasiões você vai querer que algumas variáveis sejam
 vista por muitas funções. Você consegue isso usando variáveis
 globais que são declaradas simplesmente dessa forma:
 
 global nval as integer
 
 A partir desse momento nval é uma variável inteira que é vista em 
 qualquer ponto da sua aplicação.
 
 Outro exemplo:
 
 global g_energia as integer = 100
