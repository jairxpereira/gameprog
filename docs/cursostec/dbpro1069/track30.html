30. Sub-objetos 3D
                                                            
 É muito comum os objetos 3D serem compostos de peças ou sub-objetos que o DarkBasic                                    
 denomina LIMBS ou membros em bom português. Essa denominação de membro do sub-objeto                            
 tem uma referência bem apropriada aos membros que compõem o corpo humano porque                                        
 tal como nossos membros, geralmente a ligação entre esses sub-objetos é hierárquica.                                   
                                                                                                                        
 O DarkBasic apresenta comandos e funções para manipular os membros de um objeto 3D.                                    
 Esses comandos não oferecem dificuldade em seu uso pois são semelhantes em formas                                      
 e efeitos aos que você usa para manipular os objetos inteiros. Você vai encontrar                                      
 comandos para texturizar, movimentar, rotacionar e escalar individualmente cada                                        
 membro componente de um objeto. Essa possibilidade é útil para imprimir movimentos                                     
 bem calculados aos nossos objetos.                                                                                     
                                                                                                                        
 A consequência da ligação hierárquica entre os membros consiste no fato de que                                         
 a transformação que você aplica em um membro reflete-se nos outros membros que                                  
 estão coligados numa linha de subordinação com esse membro. Para ilustrar isso                                         
 um bom exemplo é o que ocorre quando movimentamos nosso braço: o movimento do                                          
 braço reflete-se na mão que está coligada ao braço.                                                                    
 (*) Transformação em computação gráfica ou geometria refere-se a aplicação de                                          
 rotação, movimentação, posicionamento e escala nos objetos 3D.                                                         
                                                                                                                        
 Na sintaxe dos comandos que manipulam membros você vai encontrar a palavra limb                                 
 associada ao nome do comando em conjunto com o nHandle do objeto e o nHandle                                           
 que se refere ao membro dentro desse objeto. Vamos lembrar que o nHandle é um                                          
 número inteiro que identifica um objeto e lhe serve como "alça" de manipulação.                                        
                                                                                                                        
 PERFORM CHECKLIST FOR OBJECT LIMBS nObjeto                                                               
 Este comando realiza uma pesquisa de membros no objeto especificado. A quantidade                                      
 de membros você pega na função nqtd = CHECKLIST QUANTITY(). Cada membro do objeto                               
 costuma ter um nome que você pega na função nome$ = CHECKLIST STRING$(ndx).                                     
                                                                                                                        
 Veja nosso programa exemplo:                                                                                           

                                                       
rem pesquisando os membros de um objeto 3D                                                                              
cls: hide mouse                                                                                                         
cd "c:\dbCurso"                                                                                                         
                                                                                                                        
load object "walk.x",1                                                                                                  
set camera view 150,0,640,480                                                                                           
                                                                                                                        
perform checklist for object limbs 1                                                                                    
nqtd = checklist quantity()                                                                                             
                                                                                                                        
print nqtd                                                                                                              
for ncx = 1 to nqtd                                                                                                     
print checklist string$(ncx),"_",ncx                                                                                    
next ncx                                                                                                                
suspend for key
                                                                                            
                                                                                                                        
 Exibição de limbs                                                                                        
 HIDE LIMB nObjeto, nMembro ..... Esconde o membro do objeto especificado                                        
 SHOW LIMB nObjeto, nMembro ..... Re-exibe o membro do objeto especificado                                       
                                                                                                                        
 Comandos e funções de transformação de limbs                                                             
 OFFSET LIMB nObjeto, nMembro, dx, dy, dz                                                                        
 Este comando desloca o membro com relação ao posicionamento do objeto e ainda com                                      
 relação ao sub-objeto que lhe é superior. Os sub-objetos que estão subordinados                                        
 ao membro também serão reposicionados.                                                                                 
                                                                                                                        
 ROTATE LIMB nObjeto, nMembro, rx, ry, rx                                                                        
 Este comando rotaciona um membro do objeto especificado nos eixos x,y,z.                                               
                                                                                                                        
 SCALE LIMB nObjeto, nMembro, sx,sy,sz                                                                           
 Este comando escala um membro do objeto especificado nos eixos x,y,z.                                                  
                                                                                                                        
 texturização de limbs                                                                                    
 TEXTURE LIMB nObjeto, nMembro, nImagem                                                                          
 Este comando texturiza um membro com a imagem especificada.                                                            
                                                                                                                        
 COLOR LIMB nObjeto, nMembro, nCor                                                                               
 Este comando colore um membro com a cor especificada.                                                                  
                                                                                                                        
 SCROLL LIMB TEXTURE nObjeto, nMembro, dx, dy                                                                    
 Este comando rola a textura aplicada ao membro objeto. Os argumentos dx,dy devem                                       
 ser dados em números reais (decimais).                                                                                 
                                                                                                                        
 SCALE LIMB TEXTURE nObjeto, nMembro, sx, sy                                                                     
 Este comando escala a textura do membro do objeto especificados, determinando o                                        
 espalhamento, repetição ou tiling da textura sobre o membro. Os argumentos sx,sy                                       
 devem ser dados em números reais (decimais).                                                                           
                                                                                                                        
 Adição e ligação de limbs                                                                                
 ADD LIMB will create a new limb from a mesh and add it to an existing 3D object                                        
 LINK LIMB will link a newly added limb to the hierarchy of the main 3D object
