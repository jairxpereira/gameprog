27. Leitura e gravação de arquivos
                                            
 O DarkBasic oferece um conjunto de comandos e funções para ler e gravar                                                
 arquivos em disco. Isso permite a gravação da configuração ou savegame                                                 
 do seu jogo, assim como a possibilidade de criar os mais diversos                                                      
 arquivos de dados.                                                                                                     
                                                                                                                        
 MAKE FILE Arquivo$                                                                                       
 Este comando cria um arquivo no diretório corrente.                                                                    
                                                                                                                        
 OPEN TO WRITE nHandle, Arquivo$                                                                          
 Este comando abre um arquivo para gravação, você precisa identificar o arquivo                                         
 com o número de handle. Você pode abrir até 32 arquivos simultâneamente.                                               
                                                                                                                        
 WRITE STRING nFile, x$                                                                                   
 Este comando grava uma string no arquivo nFile.                                                                        
                                                                                                                        
 CLOSE FILE nFile                                                                                         
 Este comando fecha um arquivo aberto.                                                                                  
                                                                                                                        
 OPEN TO READ nHandle, Arquivo$                                                                           
 Este comando abre um arquivo para leitura, você precisa identificar o arquivo                                          
 com o número de handle. Você pode abrir até 32 arquivos simultâneamente.                                               
                                                                                                                        
 READ STRING nFile, x$                                                                                    
 Este comando lê uma string no arquivo nFile e grava o conteúdo na variável x$.                                         
                                                                                                                        
  Veja na sequência dois exemplos que ilustram gravação e leitura de arquivos textos:                                   

                                                       
cls: hide mouse                                                                                                         
rem gravado arquivo                                                                                                     
                                                                                                                        
set dir "c:\dbCurso"                                                                                                    
arquivo$ = "teste.txt"                                                                                                  
dados$ = "Programar é legal!"                                                                                           
                                                                                                                        
sts = file exist("teste.txt")                                                                                           
                                                                                                                        
if sts = 1 then delete file arquivo$                                                                                    
if sts = 0 then make file arquivo$                                                                                      
                                                                                                                        
open to write 1,arquivo$                                                                                                
                                                                                                                        
for cx = 1 to 10                                                                                                        
write string 1,dados$                                                                                                   
next cx                                                                                                                 
                                                                                                                        
close file 1
                                                                                               
                                                                                                                        
 Lendo um arquivo texto:                                                                                                

                                                       
cls: hide mouse                                                                                                         
                                                                                                                        
rem lendo arquivos textos                                                                                               
set dir "c:\dbCurso"                                                                                                    
arquivo$ = "teste.txt"                                                                                                  
linha$ = space$(80)                                                                                                     
                                                                                                                        
sts = file exist(arquivo$)                                                                                              
                                                                                                                        
if sts = 0                                                                                                              
print "Arquivo não existe!"                                                                                             
suspend for key: end                                                                                                    
endif                                                                                                                   
                                                                                                                        
if sts = 1 then open to read 1,arquivo$                                                                                 
                                                                                                                        
for cx = 1 to 10                                                                                                        
read string 1,linha$                                                                                                    
print cx,linha$                                                                                                         
next cx                                                                                                                 
                                                                                                                        
close file 1
                                                                                               
                                                                                                                        
 Gravando números                                                                                                
 WRITE BYTE nArquivo, nValor                                                                              
 WRITE WORD nArquivo, nValor                                                                              
 WRITE LONG nArquivo, nValor                                                                              
 WRITE FLOAT nArquivo, nValor                                                                             
                                                                                                                        
 Estes comandos gravam números no arquivo nArquivo especificado. Cada tipo de número tem                                
 uma faixa de valores, a faixa do byte é de 0 a 255; a faixa do WORD é de 0....65535.                                   
 O LONG tem quatro bytes de tamanho ou 32 bits; o FLOAT são os números decimais de                                      
 ponto flutuante.                                                                                                       
                                                                                                                        
 Lendo números                                                                                                   
 READ BYTE nArquivo, nVar                                                                                 
 READ WORD nArquivo, nVar                                                                                 
 READ LONG nArquivo, nVar                                                                                 
 READ FLOAT nArquivo, nVar                                                                                
                                                                                                                        
 Estes comandos leêm números do arquivo nArquivo especificado.                                                          
 Veja um exemplo abaixo:                                                                                                

                                                       
rem lendo arquivos numéricos                                                                                            
cls: hide mouse                                                                                                         
set dir "c:\dbCurso"                                                                                                    
nval = -50000                                                                                                           
arquivo$ = "teste.dat"                                                                                                  
                                                                                                                        
sts = file exist(arquivo$)                                                                                              
if sts = 1 then delete file arquivo$                                                                                    
if sts = 0 then make file arquivo$                                                                                      
                                                                                                                        
open to write 1,arquivo$                                                                                                
write long 1,nval                                                                                                       
nval = 0                                                                                                                
close file 1                                                                                                            
                                                                                                                        
open to read 1,arquivo$                                                                                                 
read long 1,nval                                                                                                        
print "lendo: ",nval                                                                                                    
close file 1
