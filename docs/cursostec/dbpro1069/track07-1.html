07.1 Processamento do teclado

 tecla$ = inkey$()
 A função  inkey$() verifica o teclado e retorna a tecla digitada. Essa
 função não pára a execução do programa  tal como  faz o comando input, 
 então é necessário  saltar para  um rótulo aonde começa a manipulação 
 do teclado para não encerrar o programa.
       
 var$ = inkey$()        

 Exemplo:



cls
hide mouse

teclado:

tecla$ = inkey$()
if tecla$ <> "" then cls
if tecla$ = "q" then end

set cursor 320,240
print tecla$

goto teclado



------------------------------------------------------------------------------
 funções de verificação de teclado
 nBool = upkey()         - verifica seta para cima
 nBool = downkey()       - verifica seta para baixo
 nBool = leftkey()       - verifica seta esquerda
 nBool = rightkey()      - verifica seta direita
 nBool = controlkey()    - verifica tecla CONTROL
 nBool = shiftkey()      - verifica tecla SHIFT
 nBool = spacekey()      - verifica tecla espaço
 nBool = returnkey()     - verifica tecla ENTER
 nBool = escapekey()     - verifica tecla ESC

 Estas funções retornam o valor um (1) quando a referida tecla está 
 pressionada. Um modo conveniente de usá-las consiste nas seguintes
 maneiras: 

 if upkey() = 1 then FaçaAlgumaCoisa()

      ou

 if upkey() = 1

 (...) código...

 endif

 ------------------------------------------------------------------------------
 sc = scancode()
 scancode() retorna o código de scan da tecla pressionada. Esta função
 é necessária  para a utilização das teclas de função tal como F1 ou F2
 e as outras teclas especiais como INSERT ou DELETE.
 
 nAscii = asc(tecla$)
 asc(tecla$) retorna  o código  ASCII do  cáracter digitado,  que foi
 um código criado para intercâmbio de informações. O código de scan é 
 um código rústico  relacionado com o teclado em termos  de  hardware 
 com relação a percepção elétrica da tecla digitada.

 nBool = keystate(nScancode)
 Esta  função  verifica  o pressionamento da tecla que tem o scancode 
 especificado. Caso esta tecla esteja pressionado, o valor de retorno
 é um (1).
 
 Ex.: 
 if keystate(59)=1 then print "F1 Ok" 

 ------------------------------------------------------------------------------
 disable escapekey
 enable escapekey
 Com o comando disable escapekey você desativa a tecla ESC como forma de
 encerrar a execução do  programa,  cabendo a você implementar uma outra
 forma de sair do seu programa. 
 enable escapekey religa a função original da tecla ESC.
 
 Veja o programa exemplo:
 

 
 

 

 
Rem Project: teclado
Rem Created: 03/08/2013 11:40:32

inicio:
' Configurando a janela
branco as integer = 0xFFFFFF
azul   as integer = 0x0000FF
set window on
set window title "teclado.dba"
ink azul, branco
cls branco

tecla_enter    as integer = 0
tecla_cima     as integer = 0
tecla_baixo    as integer = 0
tecla_esquerda as integer = 0
tecla_direita  as integer = 0
tecla_F1       as integer = 0

scancode_cdg   as integer = 0
ascii_cdg      as integer = 0

tecla          as string = ""

teclado:
tecla_enter    = returnkey()
tecla_cima     = upkey()
tecla_baixo    = downkey()
tecla_esquerda = leftkey()
tecla_direita  = rightkey()
scancode_cdg   = scancode()
tecla          = inkey$()

' Verifica o código asc da tecla digitada
if tecla <> "" then ascii_cdg = asc(tecla)

' Verifica se a tecla F1 foi pressionada
if keystate(59) = 1 then tecla_F1 = 1 else tecla_F1 = 0

cls branco
print
print " Pressione 'q' para sair"

print
print " Enter:          ", tecla_enter
print " tecla_cima:     ", tecla_cima
print " tecla_baixo:    ", tecla_baixo
print " tecla_esquerda: ", tecla_esquerda
print " tecla_direita:  ", tecla_direita

print
print " scancode_cdg:   ", scancode_cdg
print " tecla:          ", tecla
print " ascii_cdg:      ", ascii_cdg
print " tecla_F1:       ", tecla_F1

if tecla = "q" then goto final
goto teclado

final:
print
print " Pressione qualquer tecla para encerrar"
suspend for key
end
