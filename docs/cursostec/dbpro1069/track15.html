15. Sprites
                                                                   
                                                                                      
                                                                                                                        
 Sprite é uma célula de animação. O conceito fundamental da animação consiste                                           
 no fato de que qualquer animação é criada a partir da exibição de uma                                                  
 sequência fotos, quadros ou imagens. Por exemplo, a animação de um bule                                                
 se movendo consiste numa sequência de imagens ou células de animação                                                   
 apresentando o bule numa posição diferente em cada foto, célula ou quadro.                                             
                                                                                                                        
 Para criar a sua animação você deve criar o seu personagem ou objeto em várias                                         
 posições diferentes e depois carregar essas imagens para dentro do seu programa                                        
 darkbasic. Uma maneira prática de se fazer isso é utilizar os editores de                                              
 imagem para concatenar as várias células de animação. Por exemplo, nossas                                              
 imagens foram criadas em um editor 3D e na sequência foram concatenadas pelo                                           
 programa visualizador de imagens Irfanview, fácil de ser localizado na internet                                 
 devido a sua popularidade e gratuidade.                                                                                
                                                                                                                        
 A célula de animação em nossa foto de exemplo mede 64x64 e estão separadas                                             
 uniformemente para facilitar o recorte, captura e identificação individual de                                          
 cada quadro.                                                                                                           
                                                                                                                        
 SPRITE nSprite, xcol, ylin, nImagem                                                                             
 Este comando gera uma sprite a partir de uma imagem previamente carregada,                                             
 adicionalmente ele mostra a sprite na coordenada xcol,ylin que representam                                             
 coluna e linha da tela respectivamente. Veja que a sprite, como todo objeto                                            
 do Darkbasic é identificado por um número nSprite que pode variar de um a                                              
 65535.                                                                                                                 
                                                                                                                        
 PASTE SPRITE nSprite, xcol, ylin                                                                                
 Este comando exibe a sprite nSprite na coordenada especificada por                                                     
 xcol,ylin.                                                                                                             
                                                                                                                        
 Exemplo:                                                                                       
                                                                                                                        
 cls: hide mouse                                                                                                        
 set dir "c:\dbCurso"                                                                                                   
                                                                                                                        
 load image "unibule.bmp",1                                                                                             
                                                                                                                        
 sprite 1, 20, 40, 1                                                                                                    
 paste sprite 1,80,40                                                                                                   
                                                                                                                        
 suspend for key                                                                                                 
                                                                                                                        
 Manipulação das sprites                                                                                         
 Tal como para os bitmaps, existem vários comandos para manipular as sprites,                                           
 aplicando efeitos como virar, espelhar, esconder, mostrar, esticar e outros;                                           
 e existem funções para checar se estes efeitos estão ativos na sprite                                                  
 selecionada.                                                                                                           
                                                                                                                        
 HIDE SPRITE nSprite ...................Esconde a sprite nSprite                                                        
 SHOW SPRITE nSprite ...................Mostra a sprite nSprite                                                         
 FLIP SPRITE nSprite ...................Inverte verticalmente a sprite                                                  
 MIRROR SPRITE nSprite .................Inverte horizontalmente a sprite                                                
                                                                                                                        
 SCALE SPRITE nSprite, nEscala .........Este comando re-escala o tamanho da sprite                                      
                                        reduzindo ou apliando a sprite conforme o                                       
                                        valor do argumento nEscala                                                      
                                                                                                                        
 OFFSET SPRITE nSprite, dx, dy .........Este comando desloca a sprite em relação ao                                     
                                        seu posicionamento inicial. dx é o valor para                                   
                                        o deslocamento horizontal e dy é o valor para                                   
                                        o deslocamento vertical.                                                        
                                                                                                                        
 SIZE SPRITE nSprite, xtam, ytam .......Este comando redefine o tamanho da sprite.                                      
                                        xtam representa a largura e ytam a altura                                       
                                                                                                                        
 STRETCH SPRITE nSprite, xs, ys ........Este comando aplica escala não uniforme na                                      
                                        sprite, xs é a escala horizontal, ys é o valor                                  
                                        da escala vertical.                                                             
                                                                                                                        
                                                                                                                        
 Verificando o estado das sprites                                                                                
 A qualquer momento você pode verificar o estado de alguma sprite para determinar                                       
 se ela está virada, espelhada, verificar posição, escala e deslocamento com                                            
 relação ao posicionamento original. O valor de retorno nBool é um quando o                                             
 efeito está aplicado e zero quando o efeito está desligado na sprite checada.                                          
                                                                                                                        
 nBool = SPRITE EXIST(nSprite) ......Verifica se a sprite existe                                                        
 nBool = SPRITE FLIPPED(nSprite) ....Verifica se a sprite está virada verticalmente                                     
 nBool = SPRITE MIRRORED(nSprite) ...Verifica se a sprite está espelhada horizontalmente                                
                                                                                                                        
 xcol = SPRITE X(nSprite) ...........xcol recebe a posição horizontal da sprite (coluna)                                
 ylin = SPRITE Y(nSprite) ...........ylin recebe a posição vertical da sprite (linha)                                   
                                                                                                                        
 dx = SPRITE OFFSET X(nSprite) .........dx recebe deslocamento horizontal relativo ao                                   
                                        posicionamento inicial da sprite.                                               
                                                                                                                        
 dy = SPRITE OFFSET Y(nSprite) .........dy recebe deslocamento vertical relativo ao                                     
                                        posicionamento inicial da sprite.                                               
                                                                                                                        
                                                                                                                        
 nAltura = SPRITE HEIGHT(nSprite) .......Verifica altura da sprite                                                      
 nLargura = SPRITE WIDTH(nSprite) .......Verifica largura da sprite                                                     
 nImg = SPRITE IMAGE(nSprite) ...........Verifica a imagem de origem da sprite                                          
                                                                                                                        
 xs = SPRITE SCALE X(nSprite) ...........Verifica a escala horizontal da sprite                                         
 ys = SPRITE SCALE Y(nSprite) ...........Verifica a escala vertical da sprite                                           
                                                                                                                        
 Colisão e sobreposicionamento de sprites                                                                        
 Colisão é um aspecto fundamental na criação de jogos e certamente pode ser                                             
 útil na criação de animações, dando a nossos personagens e objetos reações                                             
 inteligentes diante do que lhes acontece.                                                                              
                                                                                                                        
                                                                                                                        
 Há uma função para verificar se uma sprite colidiu com outra e ha uma outra                                            
 função para verificar se uma sprite está sobrepondo-se sobre  outra.                                                   
                                                                                                                        
 nBool = SPRITE HIT(nSprite, nSpriteAlvo)                                                                        
 Esta função verifica se a sprite nSprite atingiu a sprite alvo nSpriteAlvo,                                            
 caso verdadeiro nBool é igual a um.                                                                                    
                                                                                                                        
 nBool = SPRITE COLLISION( nSprite, nSpriteAlvo)                                                                 
 Esta função verifica se a sprite nSprite está sobrepondo-se sobre a sprite alvo                                        
 nSpriteAlvo, caso verdadeiro nBool é igual a um.                                                                       
                                                                                                                        
 Se você zerar o argumento nSpriteAlvo desses comandos, a verificação é feita                                           
 sobre qualquer sprite que atingir a sprite principal.                                                                  
                                                                                                                        
 O monitoramento do impacto e sobreposição é feito com base no posicionamento                                           
 das sprites aplicado com os comandos OFFSET SPRITE e SPRITE.
