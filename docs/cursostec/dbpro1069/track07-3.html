07.3 Joystick

 O DarkBasic tem comandos e funções para processar a entrada do joystick, 
 que torna o controle do jogo mais confortável. Esses comandos e funções
 abre um precioso espaço  para  a  produção  de  programas  de  diversas 
 finalidades tendo o joystick como interface de controle.

 O controle do programa pelo joystick não apresenta nenhuma dificuldade 
 se você tiver um joystick plugado no seu computador porque o  processo 
 é semelhante ao controle do programa pelo teclado.  De forma geral são
 funções que retornam o valor quando  certo  botão  ou  certo  eixo  do 
 joystick está pressionado.

 Verificação dos eixos
 nBool = joystick up()  ..... verifica a direção cima
 nBool = joystick down() .... verifica a direção baixo
 nBool = joystick left() .... verifica a direção esquerda
 nBool = joystick right() ... verifica a direção direita

 Verificação dos botões
 nBool = joystick fire a() .... verifica botão #1
 nBool = joystick fire b() .... verifica botão #2
 nBool = joystick fire c() .... verifica botão #3
 nBool = joystick fire d() .... verifica botão #4

 Verificação dos botões #5 até #32
 nBool = joystick x(nBotão) .... Verifica o botão #nBotão

 Veja um exemplo de código de joystick:


rem controlando via joystick
if joystick left()	= 1 then xpos = xpos - 5
if joystick right()	= 1 then xpos = xpos + 5
if joystick up()	= 1 then zpos = zpos + 5
if joystick down()	= 1 then zpos = zpos - 5

if joystick fire a()= 1 then tecla$ = "5"
if joystick fire b()= 1 then tecla$ = "5"
if joystick fire c()= 1 then tecla$ = "+"
if joystick fire d()= 1 then tecla$ = "-"

if joystick fire x(5)= 1 then tecla$ = "7"
if joystick fire x(6)= 1 then tecla$ = "9
