21. Colisão de objetos 3D
                                                     
 A detecção da colisão de objetos é um processo fundamental na programação de jogos,                                    
 pois é através desse processo que você determina uma reação inteligente ao seu                                         
 personagem conforme ele se depara com um outro objeto. O cálculo para se detectar                                      
 uma colisão em um mundo 3D é complexo, porém é importante saber que ele é feito                                        
 baseado na coordenada do objeto e no raio que o objeto ocupa ao redor dessa                                            
 coordenada, logo se houver uma intersecção de coordenadas ou de raios com um outro                                     
 objeto houve então uma colisão.                                                                                        
                                                                                                                        
 1)Colisão de um objeto com outro objeto particular                                                              
                                                                                                
                                                                                                                        
 SET OBJECT COLLISION ON nObjeto                                                                                 
 Este comando aciona a detecção de colisão do objeto especificado por nObjeto                                           
 com qualquer outro objeto.                                                                                             
                                                                                                                        
 nBool = OBJECT COLLISION(nObjeto,nObjetoAlvo)                                                                   
 Esta função retorna o valor um caso o objeto nObjeto colidiu com o objeto alvo.                                 
 Caso você queira detectar a colisão do objeto com qualquer outro objeto, configure                                     
 nObjetoAlvo para zero.                                                                                          
                                                                                                                        
 Exemplo:                                                                                       
 set object collision on 1                                                                                              
 if object collision(1,2) > 0                                                                                           
 play sound 1                                                                                                           
   for cx = 1 to 20                                                                                                     
   inc zbox: position object 2,xbox,ybox,zbox                                                                           
   next cx                                                                                                              
 endif                                                                                                           
                                                                                                                        
                                                                                                                        

                                                       
cls: hide mouse                                                                                                         
                                                                                                                        
load sound "c:\dbCurso\arrastar.wav",1                                                                                  
                                                                                                                        
rem //preparando o terreno                                                                                              
load image "c:\dbCurso\textura2x2.bmp",1                                                                                
make matrix 1,1000,1000,5,5                                                                                             
prepare matrix texture 1,1,2,2                                                                                          
fill matrix 1,0,2                                                                                                       
                                                                                                                        
rem //coordenada dos objetos                                                                                            
xpos = 400: ypos = 10: zpos = 300                                                                                       
xbox = 400: ybox = 10: zbox = 350                                                                                       
                                                                                                                        
rem //preparando os objetos                                                                                             
make object sphere 1,10                                                                                                 
position object 1,xpos,ypos,zpos                                                                                        
make object cube 2,30                                                                                                   
position object 2,xbox,ybox,zbox                                                                                        
texture object 2,1                                                                                                      
                                                                                                                        
rem //ligando detecção de colisão                                                                                    
set object collision on 1                                                                                           
                                                                                                                        
position camera 400,50,250                                                                                              
                                                                                                                        
teclado:                                                                                                                
tecla$ = inkey$()                                                                                                       
                                                                                                                        
rem // Processando teclado                                                                                              
if tecla$ = "q" then end                                                                                                
if leftkey()=1 then dec xpos,5                                                                                          
if rightkey()=1 then inc xpos,5                                                                                         
if upkey()=1 then inc zpos,5                                                                                            
if downkey()=1 then dec zpos,5                                                                                          
                                                                                                                        
rem //movendo os objetos                                                                                                
position object 1,xpos,ypos,zpos                                                                                        
position object 2,xbox,ybox,zbox                                                                                        
                                                                                                                        
rem reagindo à colisão: empurrando a caixa                                                                              
if object collision(1,2) > 0                                                                                     
play sound 1                                                                                                            
 for cx = 1 to 20                                                                                                       
   inc zbox: position object 2,xbox,ybox,zbox                                                                           
 next cx                                                                                                                
endif                                                                                                                   
goto teclado
