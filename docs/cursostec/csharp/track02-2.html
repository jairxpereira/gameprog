2.2 Estrutura de dados
1.1 Primeira forma de declarar uma struct 


A estrutura é um tipo de dado composto, definido pelo usuário, geralmente 
agrupando vários tipos de dados formando uma só entidade. Veja abaixo uma
forma simples de definir uma struct em C Sharp:



struct Monstro
{
 public int energia;
 public int tipo;       
} // fim da estrutura Monstro


Mais adiante no seu código, você declara assim a sua variável tipo Monstro:

Monstro vampiro;
vampiro.energia = 100;
vampiro.tipo = 2;


1.2 Segunda forma de declarar uma struct 
Tal como em c++, a declaração e uso da struct é muito semelhante ao que se
faz numa classe, naturalmente com a struct trazendo algumas limitações em
comparação com a grande gama de recursos que tem a construção de classes.

Na declaração da struct Ponto no código abaixo colocamos um construtor que
posteriormente torna mais confortável a declaração da variável tipo Ponto:
	


struct Ponto
{
  public int x;
  public int y;

  public Ponto (int a, int b)
 {
  x = a;
  y = b;

 } // fim do construtor
} // fim da estrutura Ponto 


Mais adiante no seu código, você pode declarar e inicializar na mesma linha 
os atributos da variável tipo Ponto:

Ponto pos_vampiro = new Ponto(320, 240);

Veja agora nosso programa exemplo completo:
	


 // Projeto testStruct
 // Este programa testa struct
 using System;

 namespace testStruct
 {    
    
    struct Monstro
    {
        public int energia;
        public int tipo;       
        
    } // fim da estrutura Monstro

    struct Ponto
    {
        public int x;
        public int y;

        public Ponto (int a, int b)
        {
            x = a;
            y = b;

        } // fim do construtor
    } // fim da estrutura Ponto 
    
    
    class Program
    {
        static void Main(string[] args)
        {

            Monstro vampiro;
            vampiro.energia = 100;
            vampiro.tipo = 2;
            Ponto pos_vampiro = new Ponto(320, 240);

            // Vamos configurar a janela de console
            Console.BackgroundColor = ConsoleColor.White;
            Console.ForegroundColor = ConsoleColor.Black;
            Console.Title = " Estrutura de dados";
            Console.Clear();

            // Vamos mostrar os dados...
            Console.WriteLine("\n Monstro.tipo: " + vampiro.tipo);
            Console.WriteLine(" Monstro.energia: " + vampiro.energia);
            Console.WriteLine(" pos_vampiro.x: " + pos_vampiro.x);
            Console.WriteLine(" pos_vampiro.y: " + pos_vampiro.y);
            Console.Read();

        } // main() fim
    } // fim da classe Program
 }// fim do nameSpace testStruct
