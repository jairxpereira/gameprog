2.1 Tipos de dados e declação de variáveis
1.1 Tipos primitivos de dados e declaração de variáveis


Preste atenção no trecho de declaração de variáveis de nosso programa exemplo:

// Definição de variáveis
String fase_nome = "Boss Final";
int nPontos = 100;
float xpos = 320.2f;
double pi = 3.1415;

// Existem duas formas de declarar as variáveis
bool lGameOver = false;
System.Boolean lFinalLocalizado = true;

O exemplo acima demonstra claramente como declarar as variáveis em C Sharp que 
segue as linhas gerais de como se faz em linguagem c/c++. Veja que a linguagem
apresenta os mesmos tipos básicos como o int, o float e o double.

O tipo string é implementado pela classe String conforme você vê acima.
Veja que há o tipo bool, nativo da linguagem C Sharp, para manipulação de valores
lógicos.

1.1.1 Duas formas de declaração de variáveis
// Existem duas formas de declarar as variáveis
bool lGameOver = false;
System.Boolean lFinalLocalizado = true;

Como demonstra o exemplo acima, existem duas formas de declaração de variáveis
na linguagem C Sharp. Existe a forma tradicional que é similar ao que se faz em
c/c++. E existe a segunda forma de declarar variáveis aonde o tipo de dado é
precedido pelo qualificador System. com uma ligeira diferença na nomenclatura
do tipo de dado.

Veja agora nosso programa exemplo completo:


	

 using System;

 namespace prjVariaveis
 {
    class Program
    {

        static void Main(string[] args)
        {
	
	// Definição de variáveis
	String fase_nome = "Boss Final";
	int nPontos = 100;
	float xpos = 320.2f;
	double pi = 3.1415;
	
	// Existem duas formas de declarar as variáveis
	bool lGameOver = false;
	System.Boolean lFinalLocalizado = true;

	// Vamos configurar a janela de console
	Console.BackgroundColor = ConsoleColor.White;
	Console.ForegroundColor = ConsoleColor.Black;
	Console.Title = "Variáveis";
	Console.Clear();

	
	// Vamos exibir as variáveis
	Console.WriteLine("\n");
	Console.WriteLine(" Nome da fase: " + fase_nome);
	Console.WriteLine(" Posição do jogador: " + xpos);
	Console.WriteLine(" PI: " + pi);
	Console.WriteLine(" O jogo acabou?: " + lGameOver);
	Console.WriteLine(" O final da fase foi localizado?: " 
							+ lFinalLocalizado);           
	
	Console.WriteLine(" Pontos: " + nPontos);
	
	// Pressione ''enter'' para continuar
	Console.Read();

        }// main() fim        

    } // fim da classe Program

 } // fim do namespace

3. Constantes

const int grau_fervura = 100;
const float pi = 3.1415f;

Através  da  palavra-chave  const  ao lado do tipo de dado são 
criadas  as  constantes.  São  valores  que  não  são  mudados 
durante a execução do aplicativo. O seu uso facilita a leitura
e manutenção do código fonte.
