2.3 Enumeração
1.1 Enum 

A enumeração criada pela palavra-chave enum consiste numa lista	de valores 
constantes claramente identificados que facilita a memorização de valores
e a leitura do código do programa.

1.1.1 Primeira forma de declarar a enumeração	

	
public enum NivelDificuldade
{
 Facil,
 Medio,
 Dificil,
 Expert
} // fim da enumeração NivelDificuldade 


Depois, você utiliza desta forma a enumeração: 

int ndificuldade = (int) NivelDificuldade.Expert ;

A linha acima é mais elegante e mais clara do que o seguinte código que seria
equivalente:  int ndificuldade = 3; 

Nesta primeira forma de declarar a enumeração a primeira constante nomeada
recebe valor zero e as seguintes crescem sequencialmente em +1.

1.1.2 Segunda forma de declarar a enumeração
Na segunda forma de declarar a enumeração você pode assinalar valores para
as constantes nomeadas ou você pode assinalar o valor do primeiro elemento,
lembrando que os demais crescem sequencialmente a partir do primeiro valor
declarado:
	


public enum Personagem
{
 Arqueiro = 1,
 Guerreiro,
 Feiticeiro,
 Cavaleiro,
 Mago
} // fim da enumeração Personagem




1.2 Acessando as strings e os valores inteiros das enumerações
1.2.1 Acessando as strings


String[] listaPersonagem = Personagem.GetNames(typeof(Personagem));

Seguindo exemplo da linha acima, você estoca todos os nomes das constantes na
forma de uma array de strings. Posteriormente você pode acessá-los desta
forma:
// Mostra a lista de personagens
for (int ncx = 0; ncx < listaPersonagem.Length; ncx++)
Console.WriteLine(" {0}- {1} ", ncx+1, listaPersonagem[ncx]);

1.2.2 Acessando os valores inteiros das enumerações
// Obtendo o valor inteiro pela string
int nMago =  (int)Enum.Parse(typeof(Personagem), "Mago");
Console.WriteLine("\n Código numérico do Mago: " + nMago);
	  


Veja agora nosso programa exemplo completo:
	



// Projeto testEnum
// Este programa testa enum 
 using System;


 namespace testEnum
 {

  public enum NivelDificuldade
  {
   Facil,
   Medio,
   Dificil,
   Expert

  } // fim da enumeração NivelDificuldade

 public enum Personagem
 {
  Arqueiro = 1,
  Guerreiro,
  Feiticeiro,
  Cavaleiro,
  Mago

 } // fim da enumeração Personagem



  class Program
  {
   static void Main()
  {

 String sNivel = " Escolha o nível de dificuldade do jogo:";
 String sPersonagem = " Escolha seu personagem:";

 String[] listaPersonagem = Personagem.GetNames(typeof(Personagem));
 String[] listaDificuldade = NivelDificuldade.GetNames(typeof(NivelDificuldade));
 int ndificuldade = (int) NivelDificuldade.Expert ;
 int npersonagem = (int) Personagem.Mago;

 config_janela();

 // Mostra a lista de níveis de dificuldade
 Console.WriteLine("\n" + sNivel);            

 for (int ncx = 0; ncx < listaDificuldade.Length; ncx++)
	  Console.WriteLine(" {0}- {1} ", ncx, listaDificuldade[ncx]);

 Console.WriteLine(" >> " + ndificuldade);            


 // Mostra a lista de personagens
 Console.WriteLine("\n" + sPersonagem);

 for (int ncx = 0; ncx < listaPersonagem.Length; ncx++)
	  Console.WriteLine(" {0}- {1} ", ncx+1,listaPersonagem[ncx]);
 Console.WriteLine(" >> " + npersonagem);            


 // Obtendo o valor inteiro pela string
 int nMago =  (int)Enum.Parse(typeof(Personagem), "Mago");
 Console.WriteLine("\n Código numérico do Mago: " + nMago);

 // Vamos esperar a leitura da tela
 Console.Read(); 

 } // main() fim


	 // Método para configurar a janela
	 private static void config_janela()
	 {
	 // Vamos configurar a janela
	 Console.ForegroundColor = ConsoleColor.Blue;
	 Console.BackgroundColor = ConsoleColor.White;
	 Console.Title = " Testando enum";           
	 Console.SetBufferSize(80, 30);
	 Console.SetWindowSize(80, 30);
	 Console.Clear();
	 } // config_janela() fim       

  } // fim da classe Program
 } // fim do namespace testEnum
