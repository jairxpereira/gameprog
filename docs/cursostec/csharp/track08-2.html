08-2 Utilizando a classe StringBuilder
1.1 Visão geral
StringBuilder produz um objeto que trabalha internamente com uma string
mutável. Este objeto é indicado para construção dinâmica de textos, por
exemplo, montar uma página html na memória.
A classe gerencia automaticamente a memória necessária na ocorrência do
crescimento da string utilizada.

Para melhor performance da sua aplicação,  quando for concatenar muitas
strings em tempo de execução utilize StringBuilder().  Na  concatenação
de um número fixo de strings utilize a classe String.

1.2 Contexto da aplicação de exemplo





Nossa aplicação de exemplo simula a construção de uma pequena página html
na memória utilizando várias strings e a classe StringBuilder presente no
namespace System.Text.

2. Declarando um objeto StringBuilder


string topo = "<html>";

// Define um objeto StringBuilder
StringBuilder sb_html;

// Inicializa o objeto com a string topo e capacidade máxima
sb_html = new StringBuilder(topo, 30000);


No exemplo acima construímos um objeto StringBuilder a partir da string
topo e definimos sua capacidade inicial para 30.000 caracteres.

3. Propriedades principais



  Console.WriteLine("Tamanho:{0} Capacidade:{1}", 
                sb_html.Length, sb_html.Capacity);



Lenght retorna o tamanho atual da string.

Capacity retorna ou configura a capacidade da string interna. O processo
de  realocação  de  memória  ocorre  quando  a  string  ultrapassa  essa 
capacidade. 

4. Métodos principais


// Adiciona texto no cabeçalho
sb_html.Append("<head>\n");

// Adiciona título da página
sb_html.AppendFormat("<title>{0}{1}-{2}</title>\n", 
    titulo, capitulo, topico);

// Adiciona conteúdo do corpo
sb_html.AppendLine(" Produzido na escola Gameprog");
            
// Fecha o rodapé da página
sb_html.AppendLine(rodape);

// Modifica objeto
sb_html.Replace("#", "Sharp");



Append adiciona uma string no final da string do objeto.

AppendLine adiciona uma string no final  da  string  do  objeto com o '\n'
adicional de nova linha.

AppendFormat adiciona uma string formatada, tal como se usa em WriteLine(),
no final da string do objeto.

Replace substitui todas as ocorrências de uma string fonte pela string de
destino.

5. Código fonte do programa exemplo: prj_StringBuilder




// Projeto prj_StringBuilder
// Este programa mostrar como utilizar a classe StringBuilder
using System;
using System.Text;

namespace prj_StringBuilder
{
    class Program
    {


        static void Main(string[] args)
        {

            config_janela("prj_StringBuilder");

            // Variáveis para construir uma página de Internet
            string titulo = "Curso_C#_fase_";
            int capitulo = 14;
            int topico = 2;

            string topo = "<html>";
            string rodape = "</body></html>\n\n";
            
            // Define um objeto StringBuilder
            StringBuilder sb_html;
            
            // Inicializa o objeto com a string topo e capacidade máxima
            sb_html = new StringBuilder(topo, 30000);
            
            // Adiciona texto no cabeçalho
            sb_html.Append("<head>\n");
            
            // Adiciona título da página
            sb_html.AppendFormat("<title>{0}{1}-{2}</title>\n", 
                titulo, capitulo, topico);
              
            // Fecha cabeçalho e abre corpo
            sb_html.AppendLine("</head><body>");
            
            // Adiciona conteúdo do corpo
            sb_html.AppendLine(" Produzido na escola Gameprog");
                        
            // Fecha o rodapé da página
            sb_html.AppendLine(rodape);


            // Exibe objeto
            Console.WriteLine(sb_html);

            // Modifica objeto
            sb_html.Replace("#", "Sharp");           
           
            // Exibe objeto
            Console.WriteLine(sb_html);
            Console.WriteLine("Tamanho:{0} Capacidade:{1}", 
                sb_html.Length, sb_html.Capacity);

            Console.Read();

        } // main() fim

        // Método para configurar a janela
        private static void config_janela(string titulo)
        {
            // Vamos configurar a janela
            Console.ForegroundColor = ConsoleColor.Blue;
            Console.BackgroundColor = ConsoleColor.White;
            Console.Title = titulo;
            Console.SetBufferSize(80, 25);
            Console.SetWindowSize(80, 25);
            Console.Clear();
            Console.Write("\n");
        } // config_janela() fim    

    } // fim da classe Program
} // fim do namespace prj_StringBuilder
