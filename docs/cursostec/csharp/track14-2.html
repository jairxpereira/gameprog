14-2 Gravação e leitura de arquivos binários
1.1 Visão geral
No processo de  gravação e leitura de dados utiliza-se o conceito de
stream que pode ser traduzido como  um fluxo de água corrente.  Essa
analogia é usada para visualizar um fluxo corrente de bytes ou dados
digitais  para  uma  fonte  ou provindo de uma fonte que pode ser um 
arquivo em disco, um bloco de memória ou uma conexão de rede. 

1.2 Contexto da aplicação de exemplo


Na aplicação de exemplo vamos criar um arquivo e nele gravaremos 10
bytes  no  teste  de gravação. No teste de leitura, vamos ler esses
dados que foram gravados.  Os dados serão  gravados e lidos em modo
binário  que  geralmente não resulta em um arquivo legível tal como 
os arquivos gravados em modo texto.

2. Criando um arquivo e gravando dados


// gravar_dados() - Grava dados no arquivo delme.txt
private static void gravar_dados()
{
  Byte[] dados = { 71, 97, 109, 101, 112, 114, 111, 103, 10, 13 };

  Stream arquivo = File.OpenWrite("delme.txt");
  arquivo.Write(dados, 0, dados.Length);
  arquivo.Close();     

  Console.WriteLine(" \t Gravação de dados Ok");
} // gravar_dados().fim


Stream arquivo = File.OpenWrite("delme.txt");
Para leitura e gravação de dados deve ser criado uma objeto da classe
Stream.  O método  File.OpenWrite()  abre um arquivo para gravação de 
dados; esta função cria o arquivo caso ele não existir.

arquivo.Write(dados, 0, dados.Length);
O objeto  Stream  possui métodos  para leitura e gravação  de  dados. 
O método Write() pede uma array de bytes (dados),  posição  a  partir
da qual os dados serão gravados (0) e a quantidade  de  dados a serem
gravados (dados.Length).

arquivo.Close();
Feche o arquivo no final do processo de gravação ou leitura de dados.  

3. Abrindo um arquivo para leitura de dados


// ler_dados() - Lê dados do arquivo delme.txt
private static void ler_dados()
{
  // Buffer de memória para receber os dados
  Byte[] dados = new Byte[10];

  // Abre o arquivo para leitura
  Stream arquivo = File.OpenRead("delme.txt");
  
  // Faz leitura dos dados
  arquivo.Read(dados, 0, 10);   

  // Mostra os bytes na Tela
  Console.Write("\n\n\t Leitura de dados Ok: \n\t");
  foreach (Byte dado in dados)
  {
    Console.Write(" {0}, ", dado);
  }
  Console.Write("\n");

  // Fecha o arquivo
  arquivo.Close();
 
} // ler_dados().fim



Stream arquivo = File.OpenRead("delme.txt");
Aqui o arquivo delme.txt é aberto para leitura.  Apesar da extensão, o
arquivo foi criado e lido na forma binária. Se o arquivo pode ser lido
a 'olho nú' é porque os byte correspodem aos caracteres legíveis,  mas
geralmente esta não é a realidade dos arquivos binários.

Geralmente os arquivos binários são mais rápidos para serem gravados e
lidos. Outra vantagem que eles oferecem é a possibilidade de gravar ou
ler em qualquer ponto do arquivo.

arquivo.Read(dados, 0, 10);
O método  Read()  faz a leitura dos dados  de acordo com a posição (0)
e  a  quantidade  (10) especificada em seus argumentos e transfere-os 
para o buffer (dados) fornecido.  

4. Código fonte do programa exemplo: prj_Streams




// Projeto prj_Streams -  Arquivo: Programa.cs
// Este programa ilustra gravação e leitura de arquivos binários
using System;
using System.IO;

namespace prj_Streams
{
  class Program
  {

    const int arquivo_gravar = 1;
    const int arquivo_ler = 2;
    const int nulo = 0;
      
    static void Main(string[] args)
    {

      config_janela("prj_Streams");
      
      int escolha = 0;

      escolha = menu();

      Console.Write("\n");

      if (escolha == arquivo_gravar) gravar_dados();

      if (escolha == arquivo_ler) ler_dados();

      Console.WriteLine("\n\t Pressione ENTER para encerrar!");

      Console.Read();

    } // main() fim

    // gravar_dados() - Grava dados no arquivo delme.txt
    private static void gravar_dados()
    {
      // Dados para serem gravados
      Byte[] dados = { 71, 97, 109, 101, 112, 114, 111, 103, 10, 13 };

      // Abre o arquivo para gravação
      Stream arquivo = File.OpenWrite("delme.txt");

      // Gravando os dados...
      arquivo.Write(dados, 0, dados.Length);

     // Fechando o arquivo
      arquivo.Close();     

      Console.WriteLine(" \t Gravação de dados Ok");
    } // gravar_dados().fim


    // ler_dados() - Lê dados do arquivo delme.txt
    private static void ler_dados()
    {
      // Buffer de memória para receber os dados
      Byte[] dados = new Byte[10];

      // Abre o arquivo para leitura
      Stream arquivo = File.OpenRead("delme.txt");
      
      // Faz leitura dos dados
      arquivo.Read(dados, 0, 10);   

      // Mostra os bytes na Tela
      Console.Write("\n\n\t Leitura de dados Ok: \n\t");
      foreach (Byte dado in dados)
      {
        Console.Write(" {0}, ", dado);
      }
      Console.Write("\n");

      // Fecha o arquivo
      arquivo.Close();
     
    } // ler_dados().fim


    // menu() - Apresenta um menu com duas opções
    private static int menu()
    {
      Console.WriteLine(" \t \t Menu \n \t -------------------------\n");
      Console.WriteLine(" \t G- Gravar dados no arquivo delme.txt \n\n");
      Console.WriteLine(" \t L- Ler dados do arquivo delme.txt \n\n" );
      Console.Write("\t Escolha sua opção (G, L): ");

      ConsoleKeyInfo tecla;
      tecla = (ConsoleKeyInfo)Console.ReadKey();

      if (tecla.Key == ConsoleKey.G)
      {
        // A linha Main() trava e aguarda o final do processamento das linhas.
        return arquivo_gravar;
      } // endif

      if (tecla.Key == ConsoleKey.L)
      {
        return arquivo_ler;
      } // endif

      return nulo;

    } // menu().fim


    // Método para configurar a janela
    private static void config_janela(string titulo)
    {
      // Vamos configurar a janela
      Console.ForegroundColor = ConsoleColor.Blue;
      Console.BackgroundColor = ConsoleColor.White;
      Console.Title = titulo;
      Console.SetBufferSize(80, 30);
      Console.SetWindowSize(80, 30);
      Console.Clear();
      Console.Write("\n");
    } // config_janela() fim    

  } // fim da classe Program
} // fim do namespace prj_Streams
