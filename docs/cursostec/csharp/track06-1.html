06-1 Examinando a construção de classes
1. Visão geral




 class Monstro
{
    
public int m_energia;
public int m_estrago;

public Monstro ()
{
 m_energia = 100;
 m_estrago = 10;
} // Fim do Construtor
 
    public void atacar()
 {
 Console.WriteLine ( "\t Vc foi atacado e perdeu " +  
   m_estrago.ToString() + " pontos de energia. \n\n");
 } // atacar().gom
} // fim da classe Monstro


Na contrução de uma classe todos os métodos devem ser implementados
dentro do corpo da classe. Não é possível tal como se faz  em  C++,
declarar os métodos no corpo  da  classe  e  depois  implementá-los
em outro lugar no arquivo da classe. Para fixar isso não é possível:


// No corpo da classe Monstro
public void atacar();

public void Monstro::atacar()
{
(...)
}



2. Construtores


public Monstro ()
{
 m_energia = 100;
 m_estrago = 10;
} // Fim do Construtor


A declaração de construtores é tal como se faz em C++, porém não
há o atalho alternativo de inicialização dos tributos tal como
existe em C++. Isso não existe em C#:



Monstro::Monstro(int energia, int estrago): m_energia(energia), m_estrago(estrago),
