14-1 Acessando o sistema de arquivos
1.1 Visão geral





O namespace System.IO contém classes fundamentais para o acesso ao sistema
de  arquivos  bem  como realizar operações com pastas e arquivos incluindo
leitura e gravação de dados.

A  classe  estática  Directory  oferece  operações  diretas   para   obter  
informações sobre pastas e arquivos e realizar operações  com  pastas  tal
como criar, deletar ou mover diretórios.

A classe estática File oferece operações diretas  para  obter  informações
sobre  arquivos  e realizar operações básicas  com os mesmos  como  copiar
e  deletar  além de  permitir operações brutas de leitura  e  gravação  de
dados.

Existem  duas  classes equivalentes às classes citadas acima que podem ter
objetos instanciados,  respectivamente as classes DirectoryInfo e FileInfo
que vamos mostrar neste tópico.

* Diretório é  um termo antigo para pasta; subspasta é uma pasta dentro de
outra pasta.

1.2 Contexto da aplicação de exemplo

O programa deste tópico acessa a pasta do sistema,  geralmente c:\windows,
e  exibe  as  pastas  e arquivos que estão dentro dela. O programa mostra
também os arquivos que estão dentro da subpasta ..\Media.

2.1 Acessando a lista de pastas e arquivos

DirectoryInfo pasta_Sistema = new DirectoryInfo("C:\\WINDOWS");
Aqui neste código criamos um objeto DirectoryInfo, pasta_Sistema,  que foi
configurado com o caminho  c:\windows  para coletar informações sobre esta 
pasta. Este objeto possui o método GetDirectories()  que retorna novamente
uma coleção de pastas. A propriedade Name retorna o nome da pasta.

Outro método importante deste objeto é GetFiles()  que retorna uma coleção
de arquivos codificados no objeto dado pela classe FileInfo:

3. Código fonte do programa exemplo: prj_Pastas




// Projeto prj_Pastas
// Este programa mostra como andar pelo sistema de arquivos
using System;
using System.IO;

namespace prj_Pastas
{
  class Program
  {
    static void Main(string[] args)
    {
      config_janela("prj_Pastas");     

      // Pega o caminho da pasta do sistema. Geralmente, C:\WINDOWS
      string windir = Environment.GetEnvironmentVariable("SystemRoot");

      // Monta um objeto de informações sobre a pasta do sistema
      DirectoryInfo pasta_Sistema = new DirectoryInfo(windir);

      int npasta = 0;

      // Mostra todas subpastas presentes na pasta do Sistema
      foreach (DirectoryInfo subpasta in pasta_Sistema.GetDirectories())
      {
        Console.WriteLine(" {0}.[{1}]", npasta, subpasta.Name);
        npasta++;
        if (subpasta.Name.Contains("Media")) mostrar_arquivos(subpasta);

      } // endforeach

      // Mostra os arquivos da pasta de sistema
      mostrar_arquivos(pasta_Sistema);

      // Pausa para a leitura do texto final
      Console.Read();

    } // Main().fim

    // Exibe os arquivos da pasta  
    private static void mostrar_arquivos(DirectoryInfo pasta)
    {
      int narquivo = 0;

      foreach (FileInfo arquivo in pasta.GetFiles())
      {
        Console.WriteLine(" \t{0}. {1}", narquivo, arquivo.Name);
        narquivo++;
        
      } // endfor each
    } // mostrar_arquivos().fim

    // Método para configurar a janela
    private static void config_janela(string titulo)
    {
      // Vamos configurar a janela
      Console.ForegroundColor = ConsoleColor.Blue;
      Console.BackgroundColor = ConsoleColor.White;
      Console.Title = titulo;
      Console.SetBufferSize(80, 250);
      Console.SetWindowSize(80, 25);
      Console.Clear();
      Console.Write("\n");
    } // config_janela() fim    

  } // fim da classe Program
} // fim do namespace prj_Pastas
